    What I'd like here is a simple plugin where admins can designate items in the game that can be traded in for an automatic output of a different item.

    For example :
    Admin sets ash as a trade-able item and sets it to give back det cap
    Player types "/tradein ash 30" or "/ti ash 30"to trade in 30 ash
    Plugin gives player 30 det caps
    *Preferably you'd be able to set also set it like 30 ash gives 1 det cap, instead of 1:1

    (of course I'd never do this with these particular items XD)

    Additionally I'd like to be able to set more than 1 trade-able item.... as well as functions for players to see the items available for trade in, what they will yield, and their item ID's if that will be needed for the plugin to function as desired.

    This would be extremely helpful to servers who do things such as use Ruby to trade in for reward. It could also be the beginnings of a true bartering plugin.... which personally I'd prefer over the money based hurtstore
