    for those of you that are looking for the same help.
    stop your server - open your ftp - go in the /save folder - open the /procedural map_(seed #) - there will be several map saves, save them all to your computer - delete the ones in the server - load one map save in to your server from your computer at a time - rename the one you just loaded to "0.sav" (0 is a zero) - restart your server and see if its the save youre looking for, if not, stop your server and repeat.
