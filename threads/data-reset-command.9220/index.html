    It needs to be done while the server is stopped. Editing data files while the server is running will not affect them, as the server saves the already loaded data once the server is restarted/shutdown. Editing data files manually generally should not be needed while the server is running. The actual configuration files for plugins can still be edited while the server is running, and then reloaded by reloading the plugin.
