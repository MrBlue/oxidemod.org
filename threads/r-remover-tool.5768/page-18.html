  3. Mughisi

    Mughisi updated R-Remover Tool with a new update entry:

    Refund change

    Added an extra variable to the configuration file that allows you to change what is refunded, you can enable or disable refunding all grades. When this is disabled the refund will only give the player back the materials used for the current grade and the initial twig grade, example: Stone -> Refunds Stone and Twig.
    When it's enabled it will refund all the materials used for all the grades the building block has passed, example: Stone -> Refunds Stone, Wood and Twig.
