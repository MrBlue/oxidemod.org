    You don't need to use that; it's available only to CovalencePlugin, not normal plugins.

    CovalencePlugin:
    Code:
    {% raw %}
    [Command("test")]
    {% endraw %}
    Code:
    {% raw %}
    [Command("test", "test2")]
    {% endraw %}
    RustPlugin:
    Code:
    {% raw %}
    [ChatCommand("test")]
    {% endraw %}
