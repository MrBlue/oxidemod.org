  2. Wulf Community Admin

    Vector3 is from UnityEngine, so you'd need to add using UnityEngine; to the top of the plugin. You also have a good bit of unneeded code.
    Code:
    {% raw %}
    using UnityEngine;namespace Oxide.Plugins
{
    [Info("FireMan", "Unknown", 0.1)]
    [Description("Put fire on Fireman!")]    class FireSpawn : RustPlugin
    {
        void OnPlayerInit(BasePlayer player) {
           var fireMan = BasePlayer.Find("Fire Man");
           if (fireMan != null)
               Effect.server.Run("assets/prefabs/npc/patrol helicopter/effects/rocket_fire.prefab", fireMan.transform.position, Vector3.zero, null, false);
        }
    }
}
    {% endraw %}
    Pretty sure you can also remove null and false at the end too, as they are assumed/default if I recall correctly.

  6. Caffeine

    I got you homie! The prefab you want is "assets/bundled/prefabs/fx/player/onfire.prefab", Ideally you want a callback that can detect player movement to call this, someone mentioned "OnClientMove" in another thread but I don't see it documented anywhere yet. Anyway a quick test you can do where you're the only player

    Code:
    {% raw %}
    using UnityEngine;
using Oxide.Core;timer.Repeat(0f, 0, () =>
            {
                var Online = BasePlayer.activePlayerList as List<BasePlayer>;
                foreach( BasePlayer player in Online )
                {
                    Effect.server.Run("assets/bundled/prefabs/fx/player/onfire.prefab", player.transform.position);
                }
            });
    {% endraw %}
    The fire follows you around if you run, and engulfs you if you stand still, check it out!


    Edit: Here's a cleaner way to do the above, that's better on performance

    Code:
    {% raw %}
    void OnPlayerInput(BasePlayer player, InputState input)
        {
                Effect.server.Run("assets/bundled/prefabs/fx/player/onfire.prefab", player.transform.position);
        }
    {% endraw %}
