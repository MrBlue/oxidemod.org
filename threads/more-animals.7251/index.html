  9. PrettyFlowerGirl

    Put in console:
    • spawn.min_rate # (default .1)
    • spawn.max_rate # (default 1)
    • spawn.min_density # (default .1)
    • spawn.max_density # (default 1)

    This changes spawn rates and density for ALL gather-able entities trees/nodes/animals
