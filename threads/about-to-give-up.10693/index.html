    Do you have a plugin installed that provides the /help command? Oxide doesn't provide that by itself.

    Which version of Oxide do you have installed? You can check by using the "version" console/RCON command.

    The "owner" system is part of Rust, not Oxide. You can see how to set that up here: http://oxidemod.org/threads/admin-and-moderators-in-rust-experimental.6520/. Owners and moderators are stored under server/identity/cfg/users.cfg.
