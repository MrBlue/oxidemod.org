  2. Wulf Community Admin

    Auth levels are a BasePlayer thing, so you'd get it from the game's object.
    Code:
    {% raw %}
    var basePlayer = player.Object as BasePlayer
var authLevel = basePlayer.net.connection.authLevel;
    {% endraw %}
