    LaserHydra submitted a new resource:

    Promocodes - gives you the possibillity to set up promocodes

    Promocodes gives you the possibillity to set up promocodes.


    The plugin will allow you to set up promocodes which grant a certain permission when redeemed.

    You can either use custom codes or generate some using the console command promocode.generate.

    Chat Commands:
    • /promo <code> - to try to redeem a code

    Console Commands:
    • promocode.generate <amount> - generate a specific amount of codes...
