  2. Ankawi

    To wipe a map, go inside your saves folder and delete that file. It should be named such as Procedural Map_<size>_<seed>. Storage.db is where signs are stored and UserPersistence.db is where blueprints are stored. When you delete your map, make sure to edit your .bat where you startup the server to the seed that you want, or just keep it the same.
     
    Ankawi, Feb 3, 2016
    #2

  3. k1lly0u

    Also make sure to remove any data files that will become obsolete, ZoneManager and teleportation data are a couple of examples of what will need to be removed. If you don't remove tp data players will be able to tp to the position of their old homes etc. There will be more depending on what plugins you are running
