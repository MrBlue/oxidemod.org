    If you can update your startup parameters try adding -autoupdate to your starting command line.. this should get it to update. Do note though that this setting will trigger an update automatically each time a new version comes out and this update will throw out Oxide, so you server will be Mod-less until you can update your Oxide manually (or through the host if they do that).

    Sometimes having a vanilla server up rather than an outdated modded server is good, but depending on the mods it might not be.
