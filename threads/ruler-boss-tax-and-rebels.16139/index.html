  1. Ventrex

    I think this is a big project but the plugin I really would like to have it like Devils Island in Rust: Devil's Island for Rust | Oxide

    The idea below is not my idea but the maker of Devil's Island / Nick Holmes
    The plugin is like Reign of Kings.
    You have a Boss of the Server who can claim to be the boss.

    He can choose what the Tax on the server will be and all items will be deposit in a chest. (or locker)
    Every wood a player harvest he will get a % of it.
    For example when the tax is 10% every 10 wood he will get 1.

    He can loose this position when he got killed. If he got killed by a player that player will be the boss.
    If he dies by for example fall damage everyone on the server can be the boss by claiming it.

    If the plugin works with HWClans it would be nice that all clan members will be his henchmen.
    Also if someone is sick of paying tax he can be a rebel, the boss and henchmen can try to kill that player so he has to pay taxes again.

    The server will say where the boss is located. Also when there is a rebel the boss will see his position.
    A long story and don't know if it is possible at all
