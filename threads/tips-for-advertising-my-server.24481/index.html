  2. Slut

    You don't really need to advertise, you've just gotta make your server clean and unique.

    If you were to join a server what makes you wanna stay? Clearly it's going to be something unique to other servers, or how well the server performs.
    Lag could be one thing that is letting your server down.

    One thing that I dislike about playing modded servers is the amount of clutter, I dislike UI's everywhere, I dislike stupid features that take the fun out of rust.
    It's much easier to build a community under minimal ruling, strict rules will take longer for you to build a community.

    I own a server that gets around ~225 (max slots) on wipe day. That's a great achievement for me as it's a 20x server which can be considered a "lag fest" to other people.
