  2. Resistance

    You just need to create a batch file with something like this:
    Code:
    {% raw %}
    C:\SteamCMD\steamcmd.exe +login anonymous +force_install_dir C:\rustserver\ +app_update 258550 validate +quit
    {% endraw %}
    Obviously you'll need to set the steamcmd and rust server directories to wherever you have those installed

    Edit updater text file only add both files where your first rust files are it will be same place your steamcmd file is

    This will not install oxide for you but its not hard to download the latest and install it OxideMod/Oxide.Rust
