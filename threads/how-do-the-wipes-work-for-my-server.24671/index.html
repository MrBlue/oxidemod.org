  2. 4lex

    Rust automatically wipes on the first Thursday of the month as it is force. You don't need to schedule your own wipes, but if you wanted a weekly wipes you can either change the seed in your startup script and/or delete the .sav file to the corresponding map each week (I usually delete them all after the forced wipe, so it doesn't become a mess). The longest duration you can have without is usually 4 weeks but there are some odd months where this will be 5 weeks.
