    To add a new item to the default list (in the .cs file), you can add this line to the big long list at the bottom of the file:
    defaultTradeList.Add(new string[3] { "MY NEW ITEM", "<Price (x 1000)", "<Max Stack Size>" });

    Just make sure that you spell the item correctly with appropriate Capitals. This is importantz!

    When you have added the new item to the default list, you should then be able to use the /addstoreitem command in game.
    I know that is currently rather clunky, so at some point (when I have a chance) I'll look into upgrading it to a config file.
