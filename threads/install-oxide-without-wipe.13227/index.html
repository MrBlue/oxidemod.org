  3. Um-Acht

    okay. i try it again. may be i did something wrong. thank god i have a backup

    I used this manual:
    1. Shutdown your server, as it can't be modded while running.
    2. Install Oxide by downloading and extracting the latest version, overwriting when prompted. [Oxide for Hurtworld]
    3. Launch the server then close it after it has started. This creates the necessary folders for Oxide.
    4. Install your plugins (.cs, .lua, .js, .coffee, and .py files) in the "oxide/plugins" directory.
    5. Launch the server and let it fully load.
    6. Configure plugins to taste by editing the .json config files in the "oxide/config" directory.
    7. Reload any or all plugins to read the config file changes.
    8. Done!
    So i have to overwrite some existing files and i thought, i overwrite the saved world data.
