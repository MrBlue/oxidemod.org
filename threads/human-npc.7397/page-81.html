  12. Stewenfree

    Tecthose: I think thre is still some bugs in this plugin or some other plugin not compatibile.
    Try to make an offline server with only Human NPC and patf./waypoint in it.

    aSPENBACK: Thay must have kit setted up. Need Kit plugin.
    You can give kit with command ingame or writing cfg.
    Search for line:
    "spawnkit": "yourkitname",
    In kit cfg you must set the items to wear(on the npc), belt(weapon), main(other loot item).
    Weapon must be on belt and settings: "weapon": true,
    When NPC killed thay drop all tree category.
    Npc need ammo if cfg settings: "needsAmmo": true,
    If you turn false thay dont need.

    aphextwinaka: NPC default return to spawn point but sometime not do it. maybe bug, dunno why.
