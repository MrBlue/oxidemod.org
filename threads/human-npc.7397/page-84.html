  15. 12345homrun

    there is no permissions exist at this time, :-(

    but you can add a ID check. (Your Steam ID)

    - open the plugin with notepad
    - find "bool hasAccess(BasePlayer player)"
    - delete the follow line "if (player.net.connection.authLevel......"
    - add this two lines:
    if (player.net.connection.authLevel == 0)
    if (player.userID < 12345678901234567)

    for "12345678901234567" give your Steam ID

    Save Plugin, and now you have acces to use HumanNPC
