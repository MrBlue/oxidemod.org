    I got it to work. Here is what I realized: I hadn't installed both dependencies. So, make sure you have both plugin dependencies installed.
    • I had only installed the waypoints plugin.
    • I had forgotten to install the pathfinder plugin.
    1. I installed the pathfinder plugin.
    2. I reinstalled the Human NPC plugin.
    3. I exited out of the game (and Steam for good measure).
    4. When I returned to the game, the "/npc_add" command worked.
