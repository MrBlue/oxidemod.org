  4. the7thguest

    Well, the automatic ones are pretty comfortable to just build directly the tier you want.
    For "just playing around" like you said, I use Build for Rust: http://oxidemod.org/plugins/build.715/
    The "/buildgrade X all" command has the same effect like yours, except it doesn't upgrade stepwise but you have to enter a tier (1 - wood, 2 - stone, 3 - metal, 4 - armored).

    If you use only "/buildgrade 4" it will only upgrade the part you are looking on (by right click).
    When you use "/buildgrade 4 all" it will upgrade the entire building you are looking at to Tier 4
