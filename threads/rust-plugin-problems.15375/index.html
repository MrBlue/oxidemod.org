  3. Wulf Community Admin

    Plugins are loaded automatically once they are placed int he correct folder, you do not need to load them manually. If you ever do need to reload/unload/load manually, make sure you are using the exact filename minus the extension, ie. 'reload ZLevelsRemastered', otherwise it will error like shown in your log. The filenames should also not have (#) in them, those are named that because of duplicate copies being downloaded.

    If a plugin doesn't accept the config, if is likely that the config is not valid or the plugin hasn't been reloaded yet.
     
    Wulf, Feb 3, 2016
