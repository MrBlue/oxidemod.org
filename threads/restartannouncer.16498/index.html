    RestartAnnouncer - Adds a timed restart and shutdown option with announcements

    I often had that I needed to restart my server because of some changes to config files or temporarily shutdown the server for an update. Instead of using the /announce command from the game I made a plugin.

    This very simple plugin gives you the option to put a timer on the original /restart and /shutdown commands and announces at certain times how long it is till the restart/shuts down.

  9. D-Kay

    The plugin is only as extra help for then you suddenly need to restart the server. If you want to restart the server like each 4 hours it's best you do this in the actual server settings (which has a build in feature for restarting each set amount of time). Changes like this in the server configuration files will require a restart of the server. You can easily do the command "/trestart 20 changed some server settings" so you get a notification by the plugin in the chat for everyone to see, which you won't have to put up yourself every now and then, and once the time has passed it restarts the server for you.
