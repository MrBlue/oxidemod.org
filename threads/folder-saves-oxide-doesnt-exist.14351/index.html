  2. Wulf Community Admin

    If your host doesn't provide a means to install via their control panel, then just download the latest Oxide build from the Downloads page, extract it, and upload it via your server's FTP. Make sure the server is shutdown beforehand, but other than that there isn't anything else to it.

    Once you've installed it, you can verify it using the '/oxide.version' chat command. If it comes back as invalid, then you do not have Oxide installed fully and I'd recommend contacting your host for assistance.
