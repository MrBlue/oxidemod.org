  2. PitMonk

    I've had similar steam authentication errors, mine were caused by the server not shutting down correctly. After closing, and the dos screen going, the RustDedicated.exe file was still running as a task. When I relaunched the server, and this task was still running I would get these errors.
    After closing down the server and before relaunching it, go to task manager, close the RustDedicated.exe task, then relaunch the server.
    Hope this helps others.
