    Idea
    So. Cheeky name I know, however, I need your honest opinion on if you think this would be a cool idea/server. So as you know games like, Battlefield one, which I thoroughly enjoy have the game mode called Rush.

    What Is Rush?
    Rush is a fast paced, insanely C-Q-B(Close quarters combat) based game mode. You have two bombs within, A : B, which can both be planted and destroyed via the attackers. The defenders job is to defend the object from the attackers until time is up. Every time they finish one zone(A AND B) not only does time get added, but, a new zone is deployed somewhere up the battlefront.

    What Makes This Unique?
    Well. First of all this is in rust. Making it a much more, survival intensive mode. Of course things like, food and water will be disabled. However, things like wounded, bleeding, etc will be disabled. This will include various maps made by me if I continue to develop this, as well as a few custom things for me to assist moderating. I plan to have multiple measures to ensure your best gaming experience.
    Such as:
    * Anti-hack(probably custom)
    * Air support(helicopters and planes)
    * Custom sounds(Now this is something I will look into using a website playing sound, however this requires a player to have their browser open)
    * Pre-made bombs, and donations.
    * And for me and other admins, a custom admin menu. This will be for assisting players, (teleportation, etc)

    Development Notes:
    So. Currently if I make this, I will most likely be the only developer. If I ever get this up, which I most likely will IF I get enough support. This will take me a long time to create, as I will be customizing it as much as I can. This plugin if or when it comes out will have multiple GUI's, class creation techniques, etc.
