  2. KillParadise

    You need to convert the arg (Im assuming its coming from the chatbox?) to an integer in C# case, something like
    Code:
    {% raw %}
    num = Int32.parse(arg);
    {% endraw %}
    but I know only the basics of C#, as I am mainly a JavaScript user.

    EDIT: Looking at the source you also need to specify the arg you want. so in this case args[0] will be the number entered by the user, so
    Code:
    {% raw %}
    [ChatCommand("tvalue")]
        void cmdChattvalue(BasePlayer player, string command, string[] args)
        {
            args = argsv; -- Here your trying to set an already defined variable with a number args is an array from the chat box
            if ((args == null) || (args != null && args.Length == 0))
                SendReply(player, "TreeGather:", string.Format("Type: /tvalue \"AmountHere\""));
            else
                SendReply(player, "TreeGather:", string.Format("Tree gather value changed too ") + args);
        }
    {% endraw %}
    Instead you should do something along the lines of
    Code:
    {% raw %}
    [ChatCommand("tvalue")]
        void cmdChattvalue(BasePlayer player, string command, string[] args)
        {
            if ((args == null) || (args != null && args.Length == 0))
                SendReply(player, "TreeGather:", string.Format("Type: /tvalue \"AmountHere\""));
            else
                multiplyerTree = Int32.parse(args[0]);
                SendReply(player, "TreeGather:", string.Format("Tree gather value changed too ") + multiplyerTree);
        }
    {% endraw %}
    This will set your value to the entered value by the player, but of course it wont matter because in your on gather you immediately set it back to 100.
