  3. Alexander Nevsky

    Hi,

    Shut down server (restart 90 in console)
    /server/my_server_identity/storage.db and userpersistence.db, delete them if you want a blueprint wipe.
    /server/my_server_identity/save/ delete the folder with your map seed and size to do a map wipe.
    Alternativelly, you can set up a different seed and/or size in the bat you use to start the server, this will create a fresh world in a new folder in /save.

    Update server if required, using steamcmd or a .bat.
    Download latest oxide build and unzip on main server folder, replace files.
    Start server,
