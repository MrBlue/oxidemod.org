    Private Messages - This plugin allows you to send private messages to players.

    This plugin allows you to send private messages to players.

    Commands    :
    - /pm <Name> <Message> => Sends a private message to player || Permission: "privatemessages.use"
    - /r <Message> => Reply to your latest private message || Permission "privatemessages.use"

    Configuration File(oxide/config/PrivateMessages.json):
    Code:
    {% raw %}
    {
  "Anti_Ads_PM_BlockAdverts": false,
  "ConsoleLogPM": false
}
