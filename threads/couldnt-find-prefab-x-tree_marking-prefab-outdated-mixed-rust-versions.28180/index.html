    All prefabs are from Rust. In this case, the latest Rust updated added falling trees and a tree mini game. When you see "couldn't find prefab" and such, it is often from using mixed Rust versions together (ie. not fully updated) or there just being an issue with the update.

    The RPC error is most likely a separate issue.
