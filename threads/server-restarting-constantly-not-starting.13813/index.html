  4. Wulf Community Admin

    |G4P| Taizun said: ↑
    Don't start your server with the _start_example.bat-file. If you haven't edited the .bat-file yourself, it will keep looping through restarts - that's how the .bat-file is set up by default.

    I would suggest you create your own .bat-file and autoexec.cfg-file in your Hurtworld-folder. The .bat-file will be used to start your server.
    Alternatively you can edit the example-file.

    Read more on how to host your own server here:
    Hosting A Server
