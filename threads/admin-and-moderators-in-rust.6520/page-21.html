  2. Wulf Community Admin

    Resistance said: ↑
    I was talking about when you first put the game up you need to run that command to add yourself to the admin group
    Click to expand...
    The command is only needed for adding a user to a group if they don't already have it, but the user in question did have auth level 2 already, so he'd just need to logout and back in and he'd be in the group. In general though, if you have owner or moderator (auth level 1 or 2), then you'll automatically be placed in the correct Oxide permission group as described previously, so there isn't any need to worry about the group unless you don't have those.

    It won't hurt to use the command to add yourself to the group manually, but it won't grant you any permissions unless the group has setup those already. Telling users to run it though is just an extra step that they don't really need, especially since they already have to logout/login once for their auth level to kick in fully after adding themselves as owner/moderator.

    Admin steps: Run console command 'ownerid steamid' > logout > login > auth level and Oxide group now set. No need for additional commands.
     
    Wulf, Nov 5, 2015
