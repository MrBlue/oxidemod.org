    Binding is done on your client and has nothing to do with your server really. Using "write.cfg" (not valid) or "write cfg" will not make a difference with it either nor will restarting the server as it's client-side. What will make a difference is if the the ownerid (auth level 2) permission is lacking which is required for no clip, ent kill, debug camera, and god to be used by your client.
