  2. Ryan

    These additions are for plugin developers.

    Bool means us plugin developers can decide whether a vending machine can accept an item, the OnLiftUse means we could potentially modify the lifts behavior and OnPlayerKicked is a void so we can just handle plugin-side stuff with that.

    As a few examples:
    • Could use CanVendingAcceptItem to prevent a door being accepted
    • Could use OnLiftUse to not activate the lift if the player is bleeding
    • Could log to an external source when a player gets kicked using OnPlayerKicked
