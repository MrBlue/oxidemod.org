    In your server console, run the command:

    spawn.report

    This will give you an overview of your server's spawn information. Look at the population for "collectable-resource-hemp (autospawn/collectable/hemp)" and "animals (autospawn/animals)"

    These two will tell you the populations for the two things you asked about. If they are 0/... then your server is indeed not spawning those types of entities.

    From what I've seen, this is usually due to the map size being too small, and there not being enough room for the additional entities (without making changes to spawning settings). You can verify this by running the command:

    spawn.fill_populations

    And then re-run:

    spawn.report

    and check the new population values. If they don't really change, what is your map size? Otherwise, players should be able to find new things because you just force ran the respawn cycle.

    More activity on the server that results in barrels getting looted, nodes harvested, and other collectibles picked up should make space for new entities, but it also might be possible you have a mod interfering (although unlikely, since I've only seen one mod to modify this stuff, and it was paid). It's not too hard to work it out yourself, but modifying spawn populations is really messy, and can result in a laggy server if done wrong.
