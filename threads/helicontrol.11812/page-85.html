    You probably granted yourself or your group the permission "helicontrol.nodrop". Revoke this permission like so:
    Code:
    {% raw %}
    oxide.revoke user UserName helicontrol.nodrop
    {% endraw %}
    and for groups:
    Code:
    {% raw %}
    oxide.revoke group GroupName helicontrol.nodrop
    {% endraw %}
