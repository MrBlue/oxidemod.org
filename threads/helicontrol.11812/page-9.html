  17. hakuri.yellowsaber

    violentj9805 said: ↑
    but was working fine before the required wipe, had already changed the config. what are the console commands to accomplish that?
    Click to expand...
    heli.bulletaccuracy (number, default is 2, increasing the number decreases accuracy) for example heli.bulletaccuracy 9 turns the spray wide with maybe 2 hits per spray.

    heli.bulletdamagescale (number, default is 1 which i think does 20 damage per hit, decrease the number to reduce bullet damage) for example heli.bulletdamagescale 0.1 will allow lower population servers the chance they need to not face an impenetrable wall of death.

    before the forced wipe there wasnt a change in prefab paths. after the wipe debug spam indicated rust experimental file structure changes, which could definitely be a root cause.

    There were calls for items in file structures that no longer existed, I hope i still have the log from when i also found out that this forced wipe also disabled the ability to delete the saved maps and have them regenerate. had to have bluefang re-install rust experimental as backup maps re-uploaded wouldnt start the server.
