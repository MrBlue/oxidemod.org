    VorEconomy - Cross-Platform Economy plugin using a custom user-defined currency.

    VorEconomy is an economy plugin that allows you to sync your players' currency data to a MySQL database. This lets your players have one currency amount that can be used across all of your servers even if they are on different Oxide-supported games.

    Note: In order to use this plugin, you will need to have a MySQL database and working knowledge of how to manipulate it.
