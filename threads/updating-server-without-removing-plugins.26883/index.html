  2. Wulf Community Admin

    Unless you are specifically deleting the oxide folders, updating will not remove any of them. Keep in mind that Oxide is a mod, so it does need to be installed after each server update you do.

    The steps are short and simple:
    1. Shutdown server
    2. Update Rust server
    3. Download compatible Oxide build
    4. Extract Oxide build .zip
    5. Overwrite all files with those from the .zip
    6. Start server
    7. Done
