  10. JimLess

    4lex said: ↑

    Click to expand...
    Fast&Crude...
    Code:
    {% raw %}
                int advert = 0;            if (Configuration.Messages.Count > 1)
            {
                for (int i = 0; i < 3 && advert == lastAdvert; i++) {
                    advert = random.Next(0, Configuration.Messages.Count);
                }
            }            lastAdvert = advert;
    {% endraw %}
