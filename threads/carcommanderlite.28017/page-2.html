    Type into F1 - console or preferably rust admin Rcon -
    oxide.grant yoursteamID carcommanderlite.admin
    A good plugin to use to easily control permissions on your server is found here Permissions Manager for Rust | Oxide also another way to check if you have permission is to stand in map in game and type /spawncar into chat if one appears in front of you you got permission.hope this helps.
