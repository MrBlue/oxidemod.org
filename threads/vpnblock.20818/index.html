  1. Calytic Community Admin Community Mod

    Calytic submitted a new resource:

    VPNBlock - Automatically kick players using VPN

    This plugin is a modern update of copper's Banvpn plugin for Rust Legacy. Players using a VPN will automatically be kicked when connecting. Admins or players with permission will not be kicked.

    Permissions
    • vpnblock.canvpn
      Allows player to connect over VPN
      Ex. grant user Calytic vpnblock.canvpn
