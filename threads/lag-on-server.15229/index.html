  2. Zleeepy

    Yes, owrongs can cause lag spikes.
    You can see how many owrongs there are on the server.
    Code:
    {% raw %}
    rcon count owrong
    {% endraw %}
    Then, download this plugin: DestroyAll for Hurtworld | Oxide
    Code:
    {% raw %}
    /destroyall owrong
    {% endraw %}
    Usually when there's around 3,7-4k owrongs it causes some lag on my server.
    (This destroys all owrongs that are placed, it ignores these in the chests/inventories etc)

    You could also try modify the autoexec.cfg and change loadbalancerframebudget and targetfps to:
    Code:
    {% raw %}
    loadbalancerframebudget 200
targetfps 999
    {% endraw %}
