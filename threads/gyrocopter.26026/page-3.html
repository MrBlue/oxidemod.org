  13. Colon Blow

    Colon Blow updated Gyrocopter with a new update entry:

    1.0.10

    ** if you have any issues. remove your config file and lang file and reload plugin ***
    Only had time to do a few things tonight.. more stuff coming

    - removed the /copter command. you can now start flight mode by going up to copter and "spinning" the spinner wheel next to chair.
    and of course landing is still /copterland.

    - Added cooldown to reusing copter after landing. Cooldown can be globally disabled/enabled via config file. enabled by default. The more battery charge that is used up, the longer the recharge/cooldown time is.

    - When copter runs out of fuel, it does not land and explode anymore, it just auto rotates to ground and has max cooldown timer (approx. 16 minutes ish).

    - when player activates Flight mode, player will be moved upwards 10 ft then copter will appear. to help prevent glitching under ground and below minimum altitude settings in config.
