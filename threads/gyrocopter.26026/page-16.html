    I think Admin already has all permissions, but if not, type;
    o.grant group Admin gyrocopter.build
    o.grant group Admin gyrocopter.use
    o.grant group Admin gyrocopter.unlimited
    [DOUBLEPOST=1521252884][/DOUBLEPOST]
    Psyonicar said: ↑
    hey is there a way to delete all the unused gyrocpters as a admin
    Click to expand...
    Nope, but I combat this by scheduling an hourly task that reloads the plugin "oxide.reload Gyrocopter" A reload wipes all Gyros.

    It's not the cleanest method, but all the players know they should park the Gyro on the hour, every hour. Also, helps remove Gyros that players cant destroy (because the copter is lost or too high in the sky), and they only have a limit of 1 active Gyro.
