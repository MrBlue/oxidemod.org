  2. Wulf Community Admin

    So the server is starting, but Oxide doesn't work? If so, that means Oxide is not installed. Oxide is a mod, so make sure you are installing it last and not updating after you install it. You can verify it is installed by using the oxide.version command, or viewing the console in the bottom-right corner.
