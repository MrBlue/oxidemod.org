  2. Freaky

    Hi! You can use TimedExecute for Rust Experimental | Oxide

    "TimerRepeat": {
    "event.run": 300, ----> Calling airdrop every 5 minutes (default) -> you can modify/delete
    "server.save": 300, ----> saving the server every 5 minutes (default) -> you can modify/delete
    "heli.call": 3600 ----> Calling helicopter every hour (3600s) -> you can modify the timer
