  2. VinylFresh

    Hey Trabbel,
    When the update comes out tonight you will need to update server files (usually click Steam update) in your gtx gaming control panel also an Oxide update is usually out about 10 mins after the game update so wait for that and also click oxide update button to update that also.
    You should be able to leave all your plugins and data files alone and they will work after the update (some plugins may need updated).
    If you want a new map choose a new seed (rust.io map website to choose new map seed(number)) in your CMD line and wipe the files in server/my_server_identity/ (there is your player learned blueprint file in there only delete that if you want to reset blueprints if not leave it) just delete the rest of the files in there to clear map.you dont need to re enter the server cfg files they will save. If you want to backup your server first just copy the whole oxide folder with an ftp program ideally.Hope that helps its kinda confusing when you start out

  3. goldie

    What I usually do is:

    1) Notice no more players/cant connect/ get server update error when trying to connect
    2) Go to your game server Control Panel
    3) Change the wipe date in the server name
    4) Change the server seed
    5) Installed Mod: None
    6) Server Update: Yes, Update/Validate
    7) Restart server
    8) Wait till restart is done
    9) Installed Mod: Oxide
    10) Restart server again


    I use survivalservers so the names/options might slightly differ from your host
