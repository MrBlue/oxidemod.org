    Basically, I like to do force_install_dir C:/rust
    which is where I place my file to update it sometimes, although it may be uncessary.
    You have to sometimes make sure your server file is in the right location otherwise the steamcmd may open up another server in a different location rather than updating yours, I'm not so sure about it, but I'm always the type of the guy that makes sure his server is updated properly without any errors.

    Go to documents, and go to OS C:
    then place your Rust server folder
    open up steamcmd
    do -login anonymous
    -force_install_dir C:/(your folder's name)
    -app_update 258550

    -Then install oxide and start your server up.
