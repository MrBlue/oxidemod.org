    We have been running a server for several months. We create custom buildings that act as custom monuments (there is loot placed around using the Custom Loot Spawns plugin). There are currently 4 in different sections of the map. We use zone manager with the following flags: nobuild, noremove, undestr, nostability, noupgrade for the custom buildings. Each wipe we copy/paste the buildings back. There is also a script that regenerates the nopickup flag so people can't steal the doors.

    The last month or so, at completely random times, parts of one or more of the custom buildings will just collapse (everything that isn't stable). It could be one, or more of them at any given time. The zones are still in tact, the server hasn't rebooted, the loot is still there.

    We have confirmed all plugins and software are current versions, and are at a loss to what is causing the buildings to randomly collapse. There doesn't seem to be a common factor as to what causes it. Does anyone have any ideas?
