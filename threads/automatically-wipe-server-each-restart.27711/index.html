  4. Zhahaman2001

    This is part of my wipe server bat file. Its next to the start-example.bat file that comes with oxide and is named wipe-example.bat. You could leave it there and use it when its time to wipe, or you could add the line wipe-example.bat to the top of start-example.bat to wipe on each start. That is if you really wanted it to wipe after EVERY little restart...
    Code:
    {% raw %}
    @echo off::Set Date
set year=%date:~-4%
set month=%date:~4,2%
if "%month:~0,1%" == " " set month=0%month:~1,1%
set day=%date:~7,2%
if "%day:~0,1%" == " " set day=0%day:~1,1%
set FreshDate=%month%-%day%-%year%::Set Time
set hour=%time:~0,2%
if "%hour:~0,1%" == " " set hour=0%hour:~1,1%
set min=%time:~3,2%
if "%min:~0,1%" == " " set min=0%min:~1,1%
set secs=%time:~6,2%
if "%secs:~0,1%" == " " set secs=0%secs:~1,1%
set FreshTime=%hour%-%min%-%secs%::Set Backup Folder
set Backup=server-backup-%FreshDate%-%FreshTime%::Backup
ren server %Backup%::Restore Bans/Configs From Backups
for /F %%a in ('dir /b %Backup%') do (
xcopy "%Backup%\%%~NXa\cfg" "server\%%~NXa\cfg\*" /E
)
    {% endraw %}
