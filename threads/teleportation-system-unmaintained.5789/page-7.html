  9. Mughisi

    The sleep was added to prevent bugs like falling through the ground, ceilings and foundations. Teleports work completely different now then they did on Legacy, Rust even had its own teleport function there whereas now in the new Rust there isn't a teleport function available. If auto wake-up would be added it would be the same as just removing the sleeping from it again as there is no way to check if the player is still loading or not. If it's really wanted to have the sleeping removed I will add an option that allows you to disable it or either automatically wakeup after a second or two. This however might result again into problems with falling through the ground, foundations and ceilings.[DOUBLEPOST=1414650372][/DOUBLEPOST]
