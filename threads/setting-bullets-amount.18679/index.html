  2. k1lly0u

    Grabbing the item from the itemList and then using .GetHeldEntity() works for me. Here is some code from my test plugin that is full of useless stuff
    Code:
    {% raw %}
    private void GetAmmo(StorageContainer container)
        {
            var item = container.inventory.itemList[0];
            if (item != null)
            {
                var heldent = item.GetHeldEntity() as BaseProjectile;
                var ammo = heldent.primaryMagazine.contents;               
            }
        }
    {% endraw %}
