  7. PryMary

    RESiSTANCE 2 said: ↑
    Thanks Resistance, what would I have to edit to get my death notes to look like this ?
    Click to expand...
    Change the following line(s) in oxide/config/DeathNotes.json
    Code:
    {% raw %}
        "AnimalDeath": [
      "{attacker} killed a {victim} with a {weapon}{attachments} from {distance}m."
    ],
    {% endraw %}
    To:
    Code:
    {% raw %}
        "AnimalDeath": [
      "{attacker} |  {weapon} | {distance} | {victim}"
    ],
    {% endraw %}
    - Amend each one of the above to suit your needs then at the bottom of the file you will find the following (Not complete list, illustration only):

    "Settings": {
    "Attacker Color": "#f01322",
    "Title": "Death",
    "Title Color": "#d06d28",
    "Victim Color": "#5bba47",
    "Weapon Color": "#fefefe",

    Hope this helps
     
    PryMary, Jun 27, 2017
