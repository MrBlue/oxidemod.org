  13. -={U.S.P}=- DOCTROS

    Death Notes is no longer a automatic display, You now need to set permissions for who gets to see them.

    LaserHydra said: ↑
    People need the permission "deathnotes.see"
    use following console commands to grant it to all people:
    grant group player deathnotes.see
    grant group moderator deathnotes.see
    grant group admin deathnotes.see
    Click to expand...
    Or via Rcon:

    oxide.grant group player deathnotes.see
    oxide.grant group moderator deathnotes.see
    oxide.grant group admin deathnotes.see
     
    -={U.S.P}=- DOCTROS, Jan 24, 2016
