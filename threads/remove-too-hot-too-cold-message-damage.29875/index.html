  6. hoppel

    All you need to get rid of the message & the damage.
    Code:
    {% raw %}
            private void OnPlayerInit(BasePlayer player)
        {
            player.metabolism.temperature.min = 32;
            player.metabolism.temperature.max = 32;
        }
    {% endraw %}
