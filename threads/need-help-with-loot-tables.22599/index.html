    So ItemListBarrels will make everything posted under it available to barrels, same with ItemListCrates, for the crates. The barrel types are defined in the BetterLoot.cs file in your /plugins folder.

    The numbers after the name tell the BetterLoot plugin how many items CAN spawn in that barrel/crate, and the plugin chooses at random how many items spawn, for example: "arrow.hv": 15, says 15 arrows at max will spawn from the barrels, so any amount from 1-15 will spawn.

    You can use the NamesList.JSON in the oxide/data/BetterLoot folder to figure out what else you want to add to the barrels/crates. You may not even need MagicLoot.

    Hopefully i made it easy to understand.
