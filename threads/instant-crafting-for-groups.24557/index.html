  2. Razor14150

    @Snowie MagicCraft does indeed do what you want. You have to edit the file in the data folder:

    "gunpowder": {
    "MaxBulkCraft": 9000,
    "MinBulkCraft": 500,
    "displayName": "Gun Powder",
    "shortName": "gunpowder",
    "description": "Made from Sulphur and Charcoal.",
    "Enabled": true - This also must be set to true for it to work!
    },
