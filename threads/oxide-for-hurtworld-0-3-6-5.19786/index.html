    Some simple instructions
    1) clean solution
    2) copy updated files into Oxide\Games\Unity\Oxide.Game.Hurtworld\Dependencies\
    3) copy oxide patched(with oxide patcher) Oxide\Games\Unity\Oxide.Game.Hurtworld\Patched\
    4) compile.
    5) install

    Oxide patcher is most important part... remove all old dlls, copy new into patcher dir. and when u load project, it's automatically apply all changes to hooks... u must revise it. profit.
