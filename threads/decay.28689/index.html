  3. Quantum

    setting decay.upkeep to false will just revert back to the old decay method.

    If you want to fully remove the decay do the following:

    In console type the following from top to bottom:
    Code:
    {% raw %}
    decay.scale 0
decay.upkeep False
server.writecfg
    {% endraw %}
    OR

    You can add those to your start.bat file.
