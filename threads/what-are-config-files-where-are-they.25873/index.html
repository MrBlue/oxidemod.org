  2. Wulf Community Admin

    Config files are generated automatically by plugins if they have one. You can find config files under the oxide/config folder. The config files can be edited with any editor such as Notepad or Notepad++. After you make valid changes, reload the plugin using the 'oxide.reload PluginName' command to reload the config changes. Make sure when editing that you follow the default format so that it is valid JSON.
