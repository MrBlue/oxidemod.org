    Nice plugin.

    You should check if the entity is null before using, not after

    May also want to verify the server is loaded before hooking OnEntitySpawned so the plugin doesn't throw exceptions while the server is loading.

    void Init() => Unsubscribe(nameof(OnEntitySpawned));
    void OnServerInitialized() => Subscribe(nameof(OnEntitySpawned));
