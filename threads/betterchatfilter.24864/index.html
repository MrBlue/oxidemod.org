  1. Kinesis

    Kinesis submitted a new resource:

    BetterChatFilter - Chat Filter based on keywords for BetterChat

    All Credit to @LaserHydra for originally creating the majority of this code in v4 of BetterChat.

    Filter out unwanted keywords and replace them with a Censor Character (ie. '*') or simply replace the entire word with a substitute. Currently works only with BetterChat, default Chat is coming soon

    Config File:
    Code:
    {% raw %}
    {
  "Word Filter": {
    "Custom Replacement": "Unicorn", // Word you would like to be swapped in instead of the phrase
    "Enabled": true, // To...
    {% endraw %}
