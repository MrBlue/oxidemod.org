  1. redBDGR

    redBDGR submitted a new resource:

    BetterGrenades - Enhanced grenade features

    This is a simple plugin that just adds a few things on to the grenades
    • Cookable grenades - Players will now be able to cook grenades like in CoD / other games
    • Configurable fuse time - You can now set the fuse times for both beancan grenades and f1 grenades
    Default Configuration File:
    Code:
    {% raw %}
    {
  "Settings": {
    "Beancan Grenade Fuse Time": 5.0,
    "F1 Grenade Fuse Time": 10.0
  }
}
    {% endraw %}
