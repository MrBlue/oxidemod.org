  7. Resistance

    Permissions have all changed
    Permissions:
    • signartist.sil_url -- To use /sil <URL>
    • signartist.sil_owner -- To ignore owner check
    • signartist.sil_cd -- To ignore cooldown check
    You can edit all other settings in config!

    You can grant a user permission by using:
    oxide.grant user <username> <permission>

    To create a group:
    oxide.group add <groupname>

    To assign permission to a group:
    oxide.grant group <groupname> <permission>

    To add users to a group:
    oxide.usergroup add <username> <groupname>


    If you want to grant permission to all users, then you can use this console command:
    • oxide.grant group player signartist.sil_url
    To all admins:
    • oxide.grant group admin signartist.sil_url
