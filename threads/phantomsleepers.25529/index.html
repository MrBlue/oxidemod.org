    nivex submitted a new resource:

    Phantom Sleepers - Create phantom sleepers to lure ESP hackers.

    If you've ever needed a player to hide in a location to lure someone you suspect of using ESP then this tool is for you.

    Create phantom sleepers by using the console command 'createphantom' which will spawn a phantom player at the position you are looking at. This phantom is automatically given clothing, a weapon, and put to sleep. All items spawned are given random skins to make the sleeper look as unique as possible.

    Config options:

    Invisbility - Hide Real Sleepers (default: disabled)...
