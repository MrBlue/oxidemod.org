    1. Do you know how to use RCON? If so, connect to it.
    2. Once connected to RCON, grant your Steam user owner by using the command: ownerid YOURSTEAMID
    3. Save the newly added owner by using the console command: server.writecfg
    4. Join (rejoin if you're already in it) to your server, and you should have owner/admin with Rust.
    5. Read the Overview of any plugins you are attempting to use to see if they require Oxide permissions to use the commands.
