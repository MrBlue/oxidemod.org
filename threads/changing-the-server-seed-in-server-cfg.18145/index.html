  2. Wulf Community Admin

    Assuming you mean Rust and not Rust Legacy? Rust Legacy doesn't have either of these.

    Open the file and add (replace # with a number):

    server.seed #
    server.worldsize #

    Save the file. If the server is running, use server.readcfg, else start the server.
