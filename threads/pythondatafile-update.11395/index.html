    I am not sure if I am mistaken but if you are trying to use a config file look at my example below from my last plug-in.

    Code:
    {% raw %}
        def LoadDefaultConfig(self):
        return True    def Init(self):
        if self.Config.has_key('items'):
            pass
        else:
            self.Config['Example'] = 'itemID as Int32, Amount as Int32, Container as String, SlotPosition as Int32'
            self.Config['items'] = ({'itemID':3506021, 'Amount':1, 'Container':'Belt', 'SlotPosition': 0},
                                    {'itemID':-853695669, 'Amount':1, 'Container':'Belt', 'SlotPosition': 1})
            self.SaveConfig()    def OnPlayerRespawned(self, BasePlayer):
        if self.Config.has_key('items'):
            inv = BasePlayer.inventory
            inv.Strip()
            for item in self.Config['items']:
                inv.GiveItem(item['itemID'], item['Amount'], False)
                invItemID = inv.FindItemID(item['itemID'])
                if item['Container'].lower() == 'default':
                    pass
                elif item['Container'].lower() == 'main':
                    invItemID.MoveToContainer(inv.containerMain, item['SlotPosition'], False)
                elif item['Container'].lower() == 'belt':
                    invItemID.MoveToContainer(inv.containerBelt, item['SlotPosition'], False)
                elif item['Container'].lower() == 'wear':
                    invItemID.MoveToContainer(inv.containerWear, item['SlotPosition'], False)
                else:
                    pass
        else:
            print 'There is a problem with the configuration file.'
    {% endraw %}
    To work with config files use the function LoadDefaultConfig() and return True because this was the only way I made it work. Then use the function Init() to create your config file and add values but in my example if the key exists then it skip this is so it does not overwrite the config file. Then just use them as Python Dictionaries the config files are on Jason format.
