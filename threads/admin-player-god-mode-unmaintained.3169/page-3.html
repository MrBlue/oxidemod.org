    Just an fyi, you have a bug with admin gear. If you want 6 items in your hotbar then you have to change this:

    Code:
    {% raw %}
    if(i > 5) then
    error("More than 5 items in the Bar, WTF!")
    break
end
    {% endraw %}
    To this:

    Code:
    {% raw %}
    if(i > 6) then
    error("More than 5 items in the Bar, WTF!")
    break
end
    {% endraw %}
