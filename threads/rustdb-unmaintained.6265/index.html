  1. Reneb

    Reneb submitted a new resource:

    RustDB - Get RustDB banlist / autokick / Add or Remove bans from your rustdb banlist

    This plugin only supports the RustDB part, it can't add on it's own bans on RustDB, not remove them.
    An Alternative plugin will have to do that.

    For the moment only calling rustDB when a player joins a server is possible.
    (with auto warning / kick features)
    Don't forget to write the correct serverPort and serverIP (join Port, not the queryIp)

    RustDB Config file:
    Code:
    {% raw %}
    {
  "serverPort": "",
  "showRustDBAnswers": true,
  "serverIP": "",
  "onJoin": {
    "broadcastBanned": true,...
    {% endraw %}
