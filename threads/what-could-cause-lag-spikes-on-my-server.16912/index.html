  2. Calytic Community Admin Community Mod

    If I were experiencing latency on your server I would probably start by unloading..
    1. ZLevels
    2. Vanish
    3. NoDecay
    4. InfoPanel
    5. Hitmarker GUI
    6. BetterLoot
    7. Quick Sort
    Obviously, you can reload them when the lag issues are fixed.

    The biggest issue you have is NoDecay. The plugin itself isn't bad, but disabling decay will ensure you reach the entity limit (even on a moderately populated server) very quickly. Players might complain about having to repair stuff, but they'll complain even more about lag and sudden unplanned wipes due to the entity limit.
