    So I see you got it fine, just the saving seems to be your problem. I guess you know how to save the data to the file? The method to use. To change the players level you can loop through the Dictionary you got, and check if the current uid is the actually wanted one. Then you can change the Level. Sure there is another also propably simplier way to do that, thats just the way I use. Its basicly just knowing some useful stuff about custom non-static classes and Dictionaries.
    Code:
    {% raw %}
    void SaveData()
{
    Interface.GetMod().DataFileSystem.WriteObject("LevelData", storedData);
}void SetPlayerLevel(ulong uid, int level)
{
    foreach(ulong curUid in storedData.LevelDatabase.Keys)
    {
        if(curUid == uid)
        {
            LevelInfo info = storedData.LevelDatabase[curUid];
            info.Level = level;
        }
        else continue;
    }
    SaveData();
}
    {% endraw %}
