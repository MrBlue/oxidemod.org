  4. ignignokt84

    You can use HitInfo.damageTypes to at least get generic damage types/amounts:
    Code:
    {% raw %}
    if(hitinfo.damageTypes.Get(DamageType.Decay) > 0)
    // decay damage
    {% endraw %}
    DamageType values are: Generic, Hunger, Thirst, Cold, Drowned, Heat, Bleeding, Poison, Suicide, Bullet, Slash, Blunt, Fall, Radiation, Bite, Stab, Explosion, RadiationExposure, ColdExposure, Decay, ElectricShock

    The procedure HitInfo.damageTypes.Get(<type>) returns the amount of damage of that particular type.

    You can likely combine some of these types with HitInfo.Weapon, HitInfo.WeaponPrefab, or HitInfo.Initiator to determine where the damage came from.
