    I keep getting these type of messages with my customized adverts, how do I make it work correctly? I've added the config that I used. Cheers
    Click to expand...
    It is simple enough for you to see, you just need to understand the logic of the coloring system.

    For every color tag (e.g <red>) there must be a <end> tag where you want the color the finish. Here's an example:
    Code:
    {% raw %}
    <red>Hello<end> <blue>World<end>
    {% endraw %}
    Output:
    Hello World
    Click to expand...
    Here's a more advance example:
    Code:
    {% raw %}
    <lime>This server features the following plugins: <cyan>Notifier<end>, <orange>Rank-ME<end>, <green>Anti-Wounded<end>.<end>
    {% endraw %}
    Output:
    This server features the following plugins: Notifier, Rank-ME, Anti-Wounded.
    Click to expand...
    Let me know if you need further help

    Note: You can use Hex code for a more precise color instead of color names ( <#FF0000>Notifier<end> )
