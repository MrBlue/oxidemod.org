  12. NobodyFTW

    quickick said: ↑
    Hi
    Can you help me, where i can add more than one trigger per command?
    I want to do commands list on chat.
    Click to expand...
    By default you have this:
    Code:
    {% raw %}
    ...
"Triggers": {
      "Admins List": [
        "/admins"
      ],
...
    {% endraw %}
    Add a second command like this:
    Code:
    {% raw %}
    ...
"Triggers": {
      "Admins List": [
        "/admins",
        "/mods"
      ],
...
    {% endraw %}
