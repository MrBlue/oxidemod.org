    Yeah it's not autosaving and on server restart it uses whatever is saved in the data file. I think if you reload the plugin before restarting it adds the data to the data file. I know looking at the file it has the new data. The same thing is happening with N Teleportation.

    This seems to be a good work around until updates are made. Tested using the following.
    1. Played game. Upgraded a level and added some tele points.
    2. Exited game. Restarted game. All data was there.
    3. Reloaded plugin and restarted server.
    4. Restarted game and all data was there.

    So basically reloading the plugin before a restart does work for now.
