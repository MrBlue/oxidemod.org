  1. redBDGR

    HighTower 2 submitted a new resource:

    Personal Beacon - Displays a beacon at a marked location for easier navigation

    Ever get lost and think "Gee, wish there was some way to indicate where my base is"? Well that's what this plugin does.

    It's very basic at the moment, but it should do the job by displaying a beacon at a waypoint you set in the game world, so that you can always find your way back.

    Commands:

    /setwp - This will set the location you will want to find later.
    /wp - This will toggle the display of the beacon.

    Currently only supports 1 beacon per user.

  13. HighTower 2

    HighTower 2 updated Personal Beacon with a new update entry:

    New admin commands

    * added the /wpcount admin command. Displays the count of beacons that have been set.
    * added the /wpshowall admin command. Will display every beacon players have set. This is a toggle just like /wp.
    * added the /wpadmin admin command. This command will set and display a beacon to every player on the server at the location when the command was ran. Entering again will clear the waypoint. This location is not saved. Useful for getting multiple players to a location and added per request...
