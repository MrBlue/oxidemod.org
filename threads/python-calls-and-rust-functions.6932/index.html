  2. Wulf Community Admin

    Calls such as that are located in Rust's Assembly-CSharp.dll. You'd need to decompile it using a program such as JustDecompile and dig in. Everything that is in the Docs is essentially all of what what Oxide provides right now, the rest is natives functions from Rust. There really isn't any language-specific API either, just slightly different ways to call it.
