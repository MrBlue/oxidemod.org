  11. Wulf Community Admin

    If you can connect with your LAN IP, then it isn’t a setup issue on the machine. If you had an invalid IP or an IP not assigned to that machine, then you wouldn’t be able to connect at all as it would fail to bind the port. The issue would lie with the router most likely. You most likely won’t be able to use your own external IP, this is because of the router. Use a local IP for yourself, other players outside your network would use the external IP.
