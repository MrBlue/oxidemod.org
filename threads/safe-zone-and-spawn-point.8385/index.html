  3. Lupox

    How to creating a Safezone you want using existing Plugins:

    At first you will need Zones Manager : http://oxidemod.org/plugins/zones-manager.739/
    1. Install the Plugin in your servers oxide plugin directory and oxide should load it now.
    2.You go to the position where you want your SafeSpot
    3. Type in chat /zone_add -> Return
    4. Type in Chat /zone pvpgod true pvegod true sleepgod true nobuild true nodeploy true enter_message "You are now in a Safezone"
    leave_message "Now you are not safe anymore!" name "Safespot 1"

    5. Install the Plugin Spawn Database http://oxidemod.org/plugins/spawns-database.720/
    6. Set up the Spawnpointy you want (Use the description of the Plugin)

    7. Advertise, that your players are allowed to setup a Homepoint in any SafeZone (im using notifier for similar things)
    8. Go into your m-Teleportation.json config file
    Code:
    {% raw %}
      "Homes": {
    "CheckFoundationForOwner": true,  //Warning, turning this to false might create unhappy situations for Players and Admins.
    "Cooldown": 600,   // Set the Cooldown you want
    "Countdown": 15,  //Set the time that passes by between command and actual teleport
    "DailyLimit": 20, //Set the Limit how often players can teleport around per Day
    "ForceOnTopOfFoundation": true,  // If false no foundation is needed
    "HomesLimit": 5,  // Sets the limit of maximal Homepoins your players can save
    "LocationRadius": 25,
    "UseFriendsAPI": true
    {% endraw %}
    9. Have fun creating your Safespots
