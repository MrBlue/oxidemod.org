  5. DataDawg

    In console type "plugins" to see which ones didn't load and go to each plug ins page and check the "Ask questions/get support" tab and see if there is a new version (Usually last page of discussion). Download new version or same if not a new one and ftp into the plugins folder of your server then reload them one by one with the reload command for oxide. Then type "plugins" again in console to see if all is good. Sometimes you'll need to wait a day or more for the new versions.

    P.S. plugins name is case sensitive when using oxide.reload caps etc.
