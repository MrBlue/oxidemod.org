  2. Wulf Community Admin

    Oxide plugins are loaded automatically when you put them under the oxide/plugins folder; you don't need to stop the server for them.

    To connect your server manually, you can type in the local IP if hosted on the same machine, or the network IP of the machine, in the F1 console.

    Examples:
    • connect 0.0.0.0:12871
    • connect 192.168.1.100:12871
    Make sure you are using a unique main port and queryport for each server.
