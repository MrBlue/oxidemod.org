  2. Wulf Community Admin

    The error means that you do not have the setup setup properly. Make sure you have ALL files from the latest Oxide build as well as libc / libgc+ installed as it appears your server may be Linux so it'd need that. Also make sure you have chmod the CSharpCompiler with +x (make it executable) under the user running the server.
