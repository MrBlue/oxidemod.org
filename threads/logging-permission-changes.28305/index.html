  2. HOUGAN_Y

    There are few hooks on Oxide:

    OnGroupPermissionGranted
    OnGroupPermissionRevoked
    OnUserPermissionGranted
    OnUserPermissionRevoked
    OnUserGroupAdded
    OnUserGroupRemoved

    You can use them to make a log. If you need ready-made solution, i PMed you.
