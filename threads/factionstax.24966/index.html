    FactionsTax - A Tax System for Factions Core

    This plugin is an addition to FactionsCore. It adds a taxation system to FactionsCore.

    While this plugin is running any player killed by a faction will have to pay taxes on resources gathered which will automatically be deposited in a Faction Tax Box. Tax boxes are set through the UI built into FactionsCore under the "MAP" Section by moderators and leaders/owners.

    CONFIG OPTIONS:
    TaxRate = 5, --> Taxed amount by percentage (default 5% gathered is taken as tax).
