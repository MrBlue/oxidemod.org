  3. Mughisi

    The BasePlayer object has a SendConsoleCommand method that allows you to run a console command from the client which is what I think you want. A little example:
    Code:
    {% raw %}
            [ChatCommand("callcar")]
        private void CallCarCommand(BasePlayer player, string command, string[] args)
        {
            // Do some stuff like permission checks, user checks, whatever you need/want
            player.SendConsoleCommand($"call.car {player.UserIDString}");
        }
    {% endraw %}
