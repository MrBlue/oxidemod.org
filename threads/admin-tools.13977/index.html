  1. Noviets

    Noviets submitted a new resource:

    AdminTools - Simply kick, ban, and mute with permissions

    Permissions:
    admintools.kick
    admintools.ban
    admintools.mute

    Using Permissions:
    As an example, to grant the permission to a user:
    /grant user PlayerName admintools.kick
    /grant user PlayerName admintools.ban
    /grant user PlayerName admintools.mute

    To grant the permission to all admins (if you have an admin group) you can use:
    /grant group admin admintools.kick
    /grant group admin admintools.ban
    /grant group admin admintools.mute

    Usage:
    /kick Player Reason here
    /ban Player...
