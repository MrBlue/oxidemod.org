  4. DylanSMR

    Its because OnPlayerInit is called when someone is joining. Try a foreach loop that detects every player online.
    [DOUBLEPOST=1458504313,1458493225][/DOUBLEPOST]So like,
    Code:
    {% raw %}
    void OnServerInitialized()
{
    startTimer();
}void startTimer()
{
     timer.Once(1000, () =>
    {
        foreach(BasePlayer player in BasePlayer.activePlayerList)
        {
            cmdCui(player);
            startTimer();
        }
    }
}
    {% endraw %}
