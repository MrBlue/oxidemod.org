  2. Mughisi

    OnEntityTakeDamage is only called on animals and such. To hook player damage you use the OnPlayerTakeDamage hook:
    Oxide API for Hurtworld
    Code:
    {% raw %}
    void OnPlayerTakeDamage(PlayerSession session, EntityEffectSourceData source)
{
    Puts("OnPlayerTakeDamage works!");
}
    {% endraw %}
