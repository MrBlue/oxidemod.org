  2. Shady757

    It's most likely because the list is being modifying while it's looping. (The item is being removed from the container and put in another) Try either:

    Code:
    {% raw %}
    foreach (Item item in loot.inventory.itemList.ToArray())
    {% endraw %}
    Or:

    Code:
    {% raw %}
    for (int i = 0; i < loot.inventory.itemList.Count; i++)
{
var item = loot.inventory.itemList[i];
}
    {% endraw %}
    I'd recommend the latter, since it's probably faster, but both should work.
