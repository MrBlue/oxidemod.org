  11. sqroot

    Again, m-Teleportation doesn't use the permission system.
    You can allow moderators to use AdminTP via the config by setting "UseableByModerators": true.
    If you want the UseableByModerators flag to allow AdminTP for everyone, you'll have to edit the plugin itself.
    Line 2473 contains the method IsAllowed.
    Line 2482 states requiredAuthLevel = 1. Change this to requiredAuthLevel = 0 if you want the behaviour to change in the way I mentioned (this is just a quick hack for the least effort, don't throw things at me!).
    If you want to do anything more powerful, like limiting access to certain users or certain groups, you'll have to wait for the rewrite I'm afraid.
