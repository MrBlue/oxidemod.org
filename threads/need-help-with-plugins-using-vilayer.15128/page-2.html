  5. [SWG*] Angrychild

    Pix3l said: ↑
    What I did to add myself as owner was this:
    1) Stop your server
    2) Open the File Manager
    3) Navigate to server/vserver/cfg
    4) Create a new file called "users.cfg"
    5) Edit the file and type "ownerid [steamid64] "admin" "admin"
    6) Save the file
    7) Start your server back up

    On Step 5, only edit your SteamID64, keep the "" around admin.
