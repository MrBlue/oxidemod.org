    You can use these two versions of those plugins (this version of racesystem.cs has an extra function to make it easier for the kits.cs).
    The Kits.cs contains support for both the levelsystem and the racesystem by using 'race' or 'level' as a parameter just like you would do with things like 'cooldown' and 'uses'.

    Keep in mind this version of the Kits plugin is an unofficial patch I made. For an official one you will have to ask @Mughisi (the original maker). I will not take any credit for the Kits plugin.
