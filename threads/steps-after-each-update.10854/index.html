  2. Domestos

    - Shutdown your server
    - Update the server files through SteamCMD or your server providers control panel
    - Download the newest Oxide version
    - Copy all Oxide files into your server folder overwriting any existing file
    - Start your server
    - Check your Oxide log file for plugin errors. If any errors occur post them in their respective forum thread along with as much infos as possible like game/oxide version
