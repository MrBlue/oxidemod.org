  4. Reynostrum

    I use this personal code with a 60s repeating timer. Work's awesome for me.
    Code:
    {% raw %}
    void Reinicio()
        {
            var fecha = DateTime.Now.ToShortTimeString().ToString();
            if (fecha == "04:00" || fecha == "13:00" || fecha == "19:00")
            {
                var Timero = plugins.Find("Timero"); // ignore this
                Timero?.Call("ChequearTimer", "300", "Reinicio automático"); // ignore this
                timer.Once(300, () =>
                {
                    BasePlayer[] array = BasePlayer.activePlayerList.ToArray();
                    for (int i = 0; i < (int)array.Length; i++)
                    {
                        BasePlayer basePlayer = array[i];
                        basePlayer.Kick("Reinicio automático");
                    }
                    timer.Once(3, () =>
                    {
                        ConsoleSystem.Run.Server.Normal("quit");
                    });
                });
            }
        }
    {% endraw %}
    Time format must be the same on the code and on the machine.
