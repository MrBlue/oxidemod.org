  3. cowtrix

    There is in fact currently a mechanism in place that can do this, but it's a bit hacky. Should work though. Place a file called "runtimeexec.cfg" in the same way that you would place your autoexec. Every 10 seconds, the server will read this file, execute the commands within, and then delete the file.
