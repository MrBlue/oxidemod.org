  2. Wulf Community Admin

    You need to set the CSharpCompiler as executable, this is standard for Linux.

    Via SSH:
    Code:
    {% raw %}
    chmod +x CSharpCompiler
    {% endraw %}
    or via FTP set the permissions to 755 or 777.
