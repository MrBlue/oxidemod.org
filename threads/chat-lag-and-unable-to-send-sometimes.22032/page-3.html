    The thing which lags the chat are the colors, if your chat looks like a rainbow it will freeze every 3 minutes. You can run BetterChat smoothly without many colorful tags. Also for another plugins which send many messages to the chat. So the rule is:

    Less colors = Less chatlag

    That's why chat on officials don't freeze. Few community devs made an alternative chat systems using HWClans etc.. I have it running on server, its lag less but still lag sometimes when chat is full of colors. Tho this is the issue of Hurtworld Legacy, at Itemv2 you can have all the colors you want it will run perfectly.
