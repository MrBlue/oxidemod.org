  2. Wulf Community Admin

    No, you'd need some sort of framework to make plugins, otherwise you'd have to do a bunch of manual patches that would have to be applied each time the itemv2 branch updates (which is often). We do have itemv2 support though, just no pre-compiled builds. We generally only support the public, released versions of games, which is why itemv2 hasn't been pushed out yet, and because the itemv2 branch of Hurtworld is still changing frequently.

    See GitHub - OxideMod/Oxide at itemv2.
