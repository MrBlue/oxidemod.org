  2. Wulf Community Admin

    Plugins are loaded automatically when installed. If you see "no source found", it means that you're trying to load a plugin with an invalid name. Plugin names are case-sensitive, so if your plugin's file is named MyPlugin.cs, then you'd use "oxide.reload MyPlugin" if you want to reload it.
