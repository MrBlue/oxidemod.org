  8. OfficerJAKE

    do this with kits...

    Hold all the items you want in kit.
    Reload all guns. Place on belt etc..

    /kit add autokit - add a new kit 'autokit' is default for spawn kit.
    /kit edit autokit - start editing the new kit.
    /kit items - add the items from yourself to the kit.

    Now all players that die will spawn in this kit, on bag or beach it doesn't matter.
