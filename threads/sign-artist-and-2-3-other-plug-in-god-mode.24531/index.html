  5. Freaky

    F1 (console)/rcon console:
    grant user "name or steamid" permission

    ex:
    SignArtist:
    grant user Franchesco signartist.url
    grant user *********47346 signartist.url (steamid)
    GodMode:
    grant user Franchesco godmode.admin (then /god)

    or you can add permission to group
    grant group group_name permission
    ex: grant group admin signartist.url
    the players group is called: default

    Read the overview of all the plugins with permission (they said there how to grant permissions).
