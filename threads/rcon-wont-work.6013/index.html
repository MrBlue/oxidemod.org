    The experimental servers don't have a rcon.login command anymore, you'll need to mark yourself as owner with the command "ownerid <your steamid>" in the server console or a RCON client in order to use admin commands.

    To use RCON, you'd have to set your RCON port via the startup command-line argument +rcon.port, and then use a special version (maybe latest) of Rusty to connect. If you need help with Rusty, please use its discussion thread.
