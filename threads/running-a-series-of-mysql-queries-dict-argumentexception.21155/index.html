  2. Tolland

    Ah. It seems that this happens if you do a query on multiple tables, which each have the same fieldname.

    So changing the second query to this, fixed the problem;
    Code:
    {% raw %}
                string SelectTeams = "SELECT g.guid, g.scheduled_start, length, game_name, t.guid t_guid, team_name "
                +" FROM scheduled_game_team t, scheduled_game g "
                 + "WHERE g.scheduled_start > NOW() AND t.g_guid = g.guid;";
    {% endraw %}
