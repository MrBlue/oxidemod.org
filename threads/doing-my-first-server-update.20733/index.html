  2. nGw|Lumm

    Well i'll tell you how i handle my updates. First of all i Stop the server. after its stopped. i delete my XP files and my map files. then im going to the data folder where all the data of the plugins are located. Im deleting for example my building owners data and kits data and all the other stuff that you might need at a wipe. So it all starts from 0. Next step im taking is searching for another map. like on playrust.io gallery or something. search for a good and nice looking map and take the seed and the size. then i change that in the file. then you should update Oxide and rustio (if enabled) next step is start the server and very important! watch your console if every plugin is loading correctly. you can see if one plugin is not loading as it should. then i logg in and test everything if everything is working.

    if you didnt get anything that i wrote or you have any other question. just write me
