  2. Wulf Community Admin

    Generally with the forced wipes, you can't really restore your old map as the protocol is different.

    To automatically update, you'd need to have the host provide a script on TCAdmin that handles checking and downloading the latest Oxide snapshot. Keep in mind that doing this may result in your server being not working, as the latest Oxide build is not instantly compatible with each Rust update.
