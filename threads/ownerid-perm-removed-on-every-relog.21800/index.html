  2. vMetzey

    It should auto create that file, but if it's not there, create one and add your admin id in "ownerid *Steam64 ID Here*"

    Also the folder is "experimental/cfg/" just place the file named "users.cfg" in there and should be good to go.
    Another safety measure is granting yourself admin perms in console or RCON and then typing "server.writecfg".

    Hope this helps!
