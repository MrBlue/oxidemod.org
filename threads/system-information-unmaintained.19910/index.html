    theconezone submitted a new resource:

    System Information - Rcon commands to display current server memroy usage and uptime.

    Simple plugin that allows you to view current memory usage and server uptime through rcon, console, and chat commands. Uses permissions, or authlevel 1 or above, or commands through rcon. If users do not have permission, a fake "unknown command" error message is displayed.

    Permissions
    systeminformation.uptime - Permission to use sysinfo.uptime or /uptime
    systeminformation.uptime - Permission to use sysinfo.mem or /mem

  7. Wulf Community Admin

    @theconezone, one thing I meant to mention to you before approving this is that it should be using the Lang API from Oxide for messages so that users can localize them easily instead of being hard-coded or using the configuration file that is meant mainly for settings/options. Not all server owners have access to edit plugins directly, and they generally shouldn't be edited directly either. If you are unsure how to use the API, most of the recent new submissions and most of my recently updated plugins will have examples. Make sure to use the player's Steam ID when sending messages to a player, this will allow them to use their own language when available.
