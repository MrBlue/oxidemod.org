    Sydney submitted a new resource:

    Day Voting System - Players can vote to change the time to day

    Concept

    Any player can start a vote to set the time to day and after the vote commenced, there are 60 seconds for other players to vote too. After the 60 seconds are over, if the percentage of the players who voted yes equals/bigger than 50% the vote will pass. Once the vote has passed there is a 5 minutes cooldown before the someone can vote again.

    Commands

    /voteday - Will start a vote to set the time to day.
    (/y)es - Vote yes
    (/n)o - Vote no
