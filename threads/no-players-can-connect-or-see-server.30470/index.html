    Running SteamCMD would uninstall Oxide. The only thing happening when you install Oxide is that it adds the "modded" tag, which Rust uses to list your server under Modded. Other than that, there is nothing that Oxide does to affect your server listing in any way. This also would not affect players being able to connect or not.
