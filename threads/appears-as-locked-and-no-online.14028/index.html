    there are several reasons for this, the most obvious would be, that you messed up the permissions.cfg file.

    if there are any blank rows, or any brackets too many, the server will loop restart.
    to test if this is the case try the following:

    if you have a chance to monitor, how much ram is used by the server (like task manager on windows, or a server status tool from your gamehoster)
    check it, when the server launched correctly the server should use something like 1,53-1,62 GB of ram. if it uses only like 300-600 MB its most likely the permissions.cfg if it uses 600MB-1,3GB its usually a plugin that is compiled wrong and causes the server to loop. reply to this post with either a screenshot or at least tell me i it solved the problem. If it is the permissions.CFG post it here, i correct it for you .
