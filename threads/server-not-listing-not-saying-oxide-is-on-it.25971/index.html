  2. Wulf Community Admin

    If the oxide.version command works, Oxide is not installed. Make sure you are installing after any updates as it is a mod. Also, the server listing time can vary with Rust and is handled by Steam. If you have not changed any settings with Oxide's oxide.config.json, then your server would be listed under the Modded section once Steam can see it (make sure your main server port is open on your router if behind one) and your client gets the updates from Steam.
