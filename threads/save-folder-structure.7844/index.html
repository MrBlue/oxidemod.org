  2. Mughisi

    The server saves the world on a set interval, each time this autosave occurs 6.sav will be removed, 5 will be renamed to 6, 4 to 5, and so on and then the current state will be saved to 0.sav
    When loading one of the backups you should however only lose what was built between the time of the backup and the time when restoring the backup, you shouldn't actually lose buildings from before that.
