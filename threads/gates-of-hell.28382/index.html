    Gates of Hell - A Plugin to spawn enormous amounts of Plague Villagers and Werewolves

    With this plugin you can open the "Gates of Hell"

    as an admin or moderator you can type:

    /gatesofhell
    or
    /gatesofhell *number between 0 and 20*

    if the argument is left blank, it will use the one from the config file (5 by default)

    after being executed EVERY player on the server will have Villagers and/or Werewolves spawning randomly in a radius of 100 around him (depending on the config file) every 3 seconds.

    dead creatures give usual loot, and have normal parameters.
