    [DOUBLEPOST=1456035725,1455970742][/DOUBLEPOST]Just in case anyone was wondering this is how I did it:

    Code:
    {% raw %}
            void OnPlayerRespawned(BasePlayer player)
        {
            player.GiveItem(ItemManager.CreateByName("stonehatchet", 1));
            player.GiveItem(ItemManager.CreateByName("stone.pickaxe", 1));
            player.GiveItem(ItemManager.CreateByName("pistol.revolver", 1));
            SendReply(player, "You respawned!");
        }
    {% endraw %}
