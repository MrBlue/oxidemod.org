  2. CaptainNebulous

    is that for a custom map or something? That definitely isn't the correct name for an autosave using the V1 or V2 default map. It is likely that this is an issue with the custom map your server appears to be using. If it was one of the default maps it would be named "Deimansland" or "Mangatang" not just a bunch of numbers. Also you shouldn't have to redo your oxide plugins on a wipe, other than deleting the data files of things that you want to be reset when you wipe. Other than that Oxide plugins generally have no tie-ins to the save file. Most of their dependencies are in the Oxide folder itself and are unaffected by wipes.
