Discussion in 'Plugin Requests' started by Alpha Toon, Jun 25, 2016.

  1. Alpha Toon

    The idea is simple, but the coding may not be. So basically I would like to see a plugin that adds a command called /admin or whatever and what it would do is grant tool cupboard access and keypad access while the command is active. When the user types it again, it would remove all access and be like nothing ever happened. There should also be a message on screen that shows it's active, this way people won't forget that they have it on. And if necessary, or can be optional, have it set to a certain authlevel. Should probably also have a permission so that it can be assigned to only trusted people. This could be useful for refunding players if items were stolen via hackers or other means and for placing signs to let offline people know something isn't right or they broke a rule, or something similar. There can be actual legitimate use for this plugin, but some will abuse it. It all depends on the type of person you are. But if it does become a thing, I thank whoever makes it.
