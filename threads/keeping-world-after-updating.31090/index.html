  4. FuJiCuRa

    @AgentBallermann >> SaveMyMap

    For the Map:
    - save the server > quit
    - update the server & oxide
    - start the server with same size and seed as before, wait until fully loaded
    - run on console "smm.loadnamed proceduralmap.size.seed.164.sav" (replace by your current sav file)
    - restart

    Each month a new save file and map file is made it picks the next number up so if your save and map file is
    proceduralmap.3000.778.164.sav
    proceduralmap.3000.778.164.map

    Then it make sense the new one will be
    proceduralmap.3000.778.165.sav
    proceduralmap.3000.778.165.map

    This must be changed when server is offline and before you run the update each month but its only done on the big update each month
