  2. Wulf Community Admin

    Kits is the only plugin you listed that has permissions for it, which you can use the below commands along with the desired group and permission:
    To assign permission to a group:
    oxide.grant group <groupname> <permission>
    To add users to a group:
    oxide.usergroup add <username> <groupname>

    For information about what permissions the Kits plugin provides, see http://oxidemod.org/plugins/kits.668/. For further help with that plugin, please use its discussion thread.
