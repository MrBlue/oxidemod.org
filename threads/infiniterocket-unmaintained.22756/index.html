  5. deniskozlov

    deniskozlov updated InfiniteRocket with a new update entry:

    0.2

    Added more complex behavior for infinite rocket.
    New chat command:
    Code:
    {% raw %}
    /explode
    {% endraw %}
    raise explosion at now
    Code:
    {% raw %}
    /explode N
    {% endraw %}
    raise explosion after N seconds
    Now plugin have configuration section:
    ExplodeTimer - seconds after rocket will be exploded. Can get integer or "infinite" literal. By default "infinite".
    UsePermissions - true or false. Set use permission system or not. If using permissions and you don't have needed permission you can't run command /explode

    Hi,
    Timer limit is max value of C# float type but in fact it is much less because the rocket can't flying infinitely, it will be collide with something and then the entity(rocket) anyway destroyed without waiting for the timer.
    Default(Rust) time for rocket sets random between 10 and 20 seconds. In my plagin default value changes to "infinite" and it's can be changed in configure file.
    The tweak of the timer affects only on rocket timer. The rocket in fact it flying timer bomb.

    Good luck
