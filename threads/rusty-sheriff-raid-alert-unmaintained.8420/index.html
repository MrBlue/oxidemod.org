    Joe Edmunds submitted a new resource:

    Rusty Sheriff Raid Alert - Create perimeters around bases, and be alerted in-game and via an Android app when breached.

    Thanks for using the Rusty Sheriff Raid Alert mod.

    The basic premise of the mod is to be able to create a perimeter around your base and be alerted in-game and externally via an Android App in the case of a breach.

    By default
    - admins have no restrictions
    - the alert system is available to admins and users
    - the number of perimeters a user can set is 1.
    - the maximum number of points a user can set on a perimeter is 24.
    - the maximum height or width of a perimeter...

    Auth system and full in-game config options

    This update brings an authorisation system to the Raid Alert mod.

    Authorised users can either be regular users or Raid Alert admins.

    Regular users are constrained by the config options found in the /rs admin menu.

    Raid Alert admins have no restrictions.
