  4. Dream 5

    Like the others suggested consider either building your town in the water, using Build for Rust | Oxide to spawn the foundations safely, or probably the best option - using Zones Manager for Rust | Oxide and using the following values to protect him until he's finished building.

    /zone_edit (Number)
    /zone enter_message "Town currently under construction" (Example)
    /zone pvpgod true
    /zone pvegod true
    /zone eject true - This will stop players entering the zone while it's under construction.

    I would advise using the copy/paste plugin too, unless you're like me and like to make things totally different each time! - Copy-Paste for Rust | Oxide
