  2. iDeath

    Without plugins, you can disable all server events but that will also effect helicopter and ch47 events. A better solution would be to install FancyDrop and disable built-in air drop.
    Code:
    {% raw %}
    "Remove builtIn Airdrop": false,
    {% endraw %}
    Code:
    {% raw %}
    server.events false
    {% endraw %}
