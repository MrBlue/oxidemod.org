  2. Deicide666ra

    Go to the "config" tab on the gameserver site, you'll see the command line used to start Rust. In there should be the map seed and size, just change the parameters there.

    You can try random seeds until you find something you like... or use playrust.io to find a map you like and reuse it's seed. Be careful with map size, I hear it's buggy if you get too big or too small, not sure where the limit's at. I'd stick to the default size unless you have good reasons not to.
