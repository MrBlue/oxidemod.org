    The Oxide update for today's Rust release was made 15 mins after Rust updated. If your server is having issues, it's likely your server is not updated and you're trying to use an incompatible Oxide build with it. Stop your server, run the Steam update via your host's control panel, then install Oxide again via your host's control panel.

    For help with Notifier, please ask in its support thread.
