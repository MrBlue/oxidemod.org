  5. Ts3hosting

    how with the betterchat update can i add [ ] around my clan tags?
    answered my question

    edit this line in the cs
    Code:
    {% raw %}
     return $"[#{(clanTagColor.StartsWith("#") ? clanTagColor.Substring(1) : clanTagColor)}]{clan.tag}[/#]";
    {% endraw %}
    to this
    Code:
    {% raw %}
     return $"[#{(clanTagColor.StartsWith("#") ? clanTagColor.Substring(1) : clanTagColor)}][{clan.tag}][/#]";
