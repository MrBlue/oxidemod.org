  20. pookins

    use airdrop extended for airdrops and set betterloot to false for airdrops
    when you have installed airdrop extended use the OVERVIEW commands in your console to set up airdrop times/cargo limit/despawn time and use the oxide items list for
    cargo you want added Rust experimental list of items with pictures

    for instance if you want to add the targeting computer you would type
    aire.setitem 1490499512 targeting.computer 70.00 2 4 which would give you a 70% chance of the item with a min (2) and max amount (4)
    a word of caution if you set the cargo pods to more than 4 one or more will be empty.

    I have found the only problem i have with airdrop extended is that i have to retype console cmds after each server restart, but that only takes a few minutes

    I have found that Betterloot is the best plugin for loot tables and barrel/crate respawn and hope mine will be working again soon
    This is the first time i have had problems with it
