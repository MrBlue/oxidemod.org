    If you want to kill/remove the entity you're looking at, type in console (or bind to a key):
    Code:
    {% raw %}
    ent kill
    {% endraw %}
    If you want to remove all instances of a rowboat, type in console:
    Code:
    {% raw %}
    del assets/content/vehicles/boats/rowboat/rowboat.prefab
    {% endraw %}
