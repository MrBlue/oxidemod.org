    Sup,

    I was just working out how long a minute in Rust is for my plugin, and thought I'd share it here for people's future reference should any one ever need it.

    1 in game minute = 5.219 seconds.

    With that you can easily work out:
    1 in game second = 0.08698333333 seconds
    1 in game hour = 313.14 seconds
    24 in game hours = 7515.36 seconds

    I did this with a small plugin that checks the game time's minute every tick and outputs the server's time seconds and milliseconds when it changes.
