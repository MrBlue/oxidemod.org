  3. hijinx80

    It does make a difference. If you update before the Oxide update is out, you'll often be running a "vanilla" server until the Oxide update is ready. This isn't always the case (small patches don't usually break Oxide), but will definitely happen with larger updates. On the flip side, players that have the updated client won't be able to play on your server until you push the update out.
