  11. Wulf Community Admin

    Wulf updated Logger with a new update entry:

    Fixed conflict with other chat plugins and renamed file

    There is an issue when you have multiple plugins fighting to control what happens to the chat and logging, so to make other plugins happy, logging is no longer canceled via Logger. The only downside to this is that you may see duplicate chat in the console. An exclusion was also added for Chat Handler to prevent chat issues with it especially.

    Important: The filename was rename to 0-logger to give it priority control of chat, so you'll need to delete the old logger.lua when upgrading.

    Wulf updated Logger with a new update entry:

    Fixed command logging with new auth check

    A recent update broke the IsAdmin() shortcut for checking a player's auth level, so I've moved to using a more direct auth level check. You can configure the auth level to log via the config file. Rust has 3 auth levels, 0, 1, and 2. 0 being regular users, 1 being moderators, and 2 being admin.
