  2. Slut

    There's plenty of reasons that those messages appear, cheating being one but it's uncommon that when a message appears the player is cheating, firstly you should make sure that the player has a stable connection to the server, you can do this by constantly checking the players ping with the "players" command in Console. You should also ensure that the server FPS is above 30fps (fps command in console) and finally check the players fps (clientperf command in console) and make sure they're not dropping to a low framerate. If their ping isn't high, the server fps is decent as well as the clients fps and you are seeing A LOT of "Hack" messages then they're probably cheating. So in saying, those messages are mostly caused by serverlag/clientlag.
