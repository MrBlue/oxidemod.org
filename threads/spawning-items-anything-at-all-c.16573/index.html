  4. DylanSMR

    Also I can give you a bit of code on how to actually spawn stuff like foundations, campfires, etc. All you need is the list of prefabs
    Code:
    {% raw %}
    string name = "assets/prefabs/deployable/woodenbox/woodbox_deployed.prefab";
Vector3 pos = new Vector3();
BaseEntity entity = GameManager.server.CreateEntity(name, pos);
            entity.Spawn(true);
    {% endraw %}
    Replace whats in the name variable with your prefab, and inside the lines "Vector3 pos = new Vector3();" it would be like, "Vector3 pos = new Vector3(x, y, z);"
