    As far as i'm aware "PvpArena" isn't a recognised flag. check the options on the initial page. If enabled is set to 'True' then all flags listed will be active and all those not listed will - by default - be set to false.

    e.g.


    Code:
    {% raw %}
          "enabled": true,
      "flags": "PveGod, Pvpgod, Sleepgod, UnDestr, NoDecay, NoPlayerLoot, NoCorpse, NoSuicide, NoBleed, NoStability"
    {% endraw %}
