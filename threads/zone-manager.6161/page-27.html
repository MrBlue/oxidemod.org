    First you have to add your zone.
    Code:
    {% raw %}
    /zone_add
    {% endraw %}
    Afterwards you can configure your zone.
    Code:
    {% raw %}
    /zone <option 1> <value 1> <option 2> <value 2>  .. <option n> <value n>
    {% endraw %}
    Those options are listed on the overview page.

    Indeed your zones added already, you have to choose the one you want to configure via its id.
    Code:
    {% raw %}
    /zone_edit <zone id>
    {% endraw %}
    Then again you can use
    Code:
    {% raw %}
    /zone <option 1> <value 1> <option 2> <value 2> ... <option n>  <value n>
    {% endraw %}

    eg.
    Code:
    {% raw %}
    /zone radius 1
    {% endraw %}
