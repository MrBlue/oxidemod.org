    Attckdog said: ↑

    Haven't tested it but I can already spot 2 issues, args[] and player.transfor.position. Not sure what args[] will return, but the method requires a string array and args is a table so you have to use the utility function to convert the table to an array: util.TableToArray(args)
    And the last argument only has a typo, transform instead of transfor
