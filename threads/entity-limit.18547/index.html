    PaiN submitted a new resource:

    Entity Limit - Limit entities per player

    This plugin limits entities for each player. For example you can limit for players the usage of High External Walls to 5.

    For item short names check out : Oxide API for Rust

    Commands
    /limitswipe = > [NEXT UPDATE]

    Default Config [oxide/config]
    Code:
    {% raw %}
    {
  "ItemShortname-MaxAmount": {
    "gates.external.high.stone": 2,
    "gates.external.high.wood": 2,
    "wall.external.high.stone": 5
  }
}
    {% endraw %}
