  14. BBH Louie

    For the people who can't get it to work, you have to fix the item names in your config otherwise it can cause multiple items not to load. Easiest way to fix is follow these steps.

    1. Open the StackSizes.json in the config
    2. Press Crtl + A to highlight the whole config
    3. Press Crtl + F to pull up the Find and Replace Tool
    4. Enter "_" in the Find and "." in the Replace section minus the quotes and hit ok
    5. Save your file and upload it back to your config folder
    6. Reload the plugin on the server
     
    BBH Louie, Aug 6, 2015
