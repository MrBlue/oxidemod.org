  2. Wulf Community Admin

    Your server ran out of memory, that isn't an issue with the plugins. If your server is actually crashing (or if it's just a forced restart by your host), it's because it is out of memory and cannot continue. It just so happens that that extra 100mb used to compile plugins is causing the server to hit your imposed memory limit with your host.
