  2. Wulf Community Admin

    See Solved - What is garbage collection?. The actual hook warnings (not garbage collection) are indications that those hooks took longer than they generally should, which could be caused by server performance (or garbage collection as it happened during this period).

    If you're experience server performance issues, I'd suggest using less plugins that deal with those hooks, as those hooks are called very frequently.
