  1. Mersereau

    I'm using player.StopWounded() with the OnPlayerLanded hook to pick a player up on a certain condition, but if they die again within 60 seconds after the first time then they skip the wounded state.

    Is there a way to reset the timer so they don't insta die if I use this command?

  2. Shady757

    You can set their last wounded time back 60 seconds. Here's an example (I'm pretty sure lastWoundedTime is still private, so reflection is needed):

    Code:
    {% raw %}
     FieldInfo lastWoundedTime = typeof(BasePlayer).GetField("lastWoundedTime", (BindingFlags.Public | BindingFlags.Instance | BindingFlags.NonPublic));private void ResetWoundedTime(BasePlayer player) => lastWoundedTime.SetValue(player, UnityEngine.Time.realtimeSinceStartup - 60);
    {% endraw %}
     
    Shady757, May 31, 2017
