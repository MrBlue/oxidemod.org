  2. Calytic Community Admin Community Mod

    A sharing violation may mean that the folder or file itself is read-only. It may also mean that the file is already open and write-protected until the process writing the file is closed. It is entirely possible that a backup routine is causing this issue for you. Usually this only happens if multiple instances of the same Rust server are running at the same time.
