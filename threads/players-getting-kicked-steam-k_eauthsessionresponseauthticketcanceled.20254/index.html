  4. Rainbow™

    Timing out can be a number of reasons,

    1) Players have a poor internet connection
    2) Your server is running on a low setup.
    3) Memory leaks.

    I have noticed in-game that you spawn too many SupplyDrops,Helicopters or you spam SurveyCharges it creates a Memory leak in-game.

    What happens is the game doesn't know when to stop using RAM, so your players and yourself all end up timing out and crashing due to the leak.
    If you want to make sure it's definately a leak, open task manager, and watch your ram usage. If it increases massively (Mine was at 20GB) then you know what's causing it. (I lost 50 players because of it) so I don't do mass air drops, helis or spam surveycharges anymore.
     
    Rainbow™, Aug 11, 2016
