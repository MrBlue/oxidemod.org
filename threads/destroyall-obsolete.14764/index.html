    Koenrad submitted a new resource:

    DestroyAll - Destroy all objects by partial name

    Due to the fact that cracked servers are having issues with people spoofing their LumaEmu steamid to gain "rcon" access, the hurtworld devs have removed the removeall console command. This plugin restores it.

    Console Commands:
    removeall <partialname>

    Chat Commands:
    /removeall <partialname>

    Permissions:
    removeall.use

    You can use the console command count <partialname> to check how many objects exist before removing them all.
