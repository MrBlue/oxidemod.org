    Oxide plugins are never under that location. The default location is oxide/plugins, and used to be server/identity/oxide/plugins about 5 months ago. The RustDedicated_Data/plugins folder is related to Unity (the engine Rust uses), not Oxide. So unless you are specifically setting +oxide.directory to that other location, that has always been the wrong location for plugins.
     
    Wulf, Aug 27, 2017
