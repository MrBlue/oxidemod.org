    The first list shows that those resource sites couldn't be reached. That happens if the plugin is no longer on oxide (unmaintained or deleted) or oxide itself is not reachable.

    The other list, indicates that the plugins did not set their resourceID which is required for identification for those. It does not harm the functionality of those plugins, but disallows the updater plugin to get their latest version.
