    bawNg submitted a new resource:

    Building Grades - Allows admins to easily upgrade or downgrade an entire building

    This plugin allows admins to easily upgrade or downgrade an entire building by one grade, relative to the current grade of each building block.

    Extremely large buildings will cause a brief server performance spike.

    Chat Commands:
    /up     - Upgrade each block which is attached to the building you are looking at by one grade
    /down - Downgrade each block which is attached to the building you are looking at by one grade

    Development of this plugin was kindly funded by Trust.
