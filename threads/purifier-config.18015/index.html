    Purifier Config - Allows configuration of Water Purifiers

    This plugin allows configuration of Water Purifiers, with options such as changing how quickly it purifies the water, the ratio of salt water to clean water, and more.

    Config File:
    Code:
    {% raw %}
    {
  "FreshWaterRatio": 4,
  "WarmupTime": 10.0,
  "WaterToProcessPerMinute": 120
}
    {% endraw %}
    All config values are default and this plugin will not do anything if the config settings are left as is.
