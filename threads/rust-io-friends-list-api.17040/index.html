  1. Alphawar

    Alphawar submitted a new resource:

    Rust:IO Friends List API - Easily edit the Rust:IO friends list via chat

    This plugin enables people using the Rust:IO extension to easily add/remove friends.

    The reason for this is that other friends list mods use there own list, but due to my server having Rust:IO, Rust:IO Clans, Rust:IO Friendly Fire, and Lusty Map, I wanted a way that people could easily add each other as players didn't like/need a map site open at the same time.

    Commands:
    • /friend add/+ name
    • /friend remove/- name
    • /friend check/? name
