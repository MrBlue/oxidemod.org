  2. Wulf Community Admin

    Oxide's installation is pretty much a simple extract and overwrite install, nothing complicated. Just make sure the server is shutdown, and it is actually fully installed and not being overwritten by any update on startup. Plugins go under saves/oxide/plugins, and you can tell if they are running by checking the oxide/logs or using the "/plugins" command. You can check if Oxide is running by using the "/version" chat command.
