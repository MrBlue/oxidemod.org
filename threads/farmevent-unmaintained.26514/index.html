  6. Tunnleram

    This is a great idea! People love events!

    The one thing I really crave as a server admin are events that I don't have to start and stop.

    For instance, here's two examples I love:
    1- RunningMan runs on a fixed schedule
    2- Monument Radiation runs on a variable schedule with variable settings

    I'd love to see #2 for this mod.

    Example:
    - set event to run anywhere between every 30 and 90 minutes (example) (sometimes it's 40 minutes, sometimes it's 80)
    - set event to require any of the specified items (wood, stone, etc) (sometimes it's wood, sometimes it's ore)
    - set event to require any quantity between 250 and 1000 (example) (sometimes it's 300, sometimes it's 900)
    - set reward to be random or any of the specified items (items you set)

    This would be ideal and make the mod dynamic and require less administration. I'd love for event mods to take this approach although I realize it might require more work.
