  2. Calytic Community Admin Community Mod

    Haven't tested it but maybe something like this..

    Code:
    {% raw %}
    HashSet<T> getObjectList<T>() where T : UnityEngine.Object
        {
            return new HashSet<T>(UnityEngine.Object.FindObjectsOfType<T>());
        }
    {% endraw %}
