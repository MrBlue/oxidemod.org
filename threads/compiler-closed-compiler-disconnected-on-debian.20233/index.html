  2. Wulf Community Admin

    The Rust server is only 64-bit, you can't run it in 32-bit. The compiler used will be 64-bit as well on Linux.

    As for the "Compiler closed" issue, make sure you are running the server under the same user you installed Rust and the Oxide mod with. Also make sure the RustDedicated_Data/Managed and oxide/plugins folders are set to 755 or something similar. Also make sure you do not have any other software that may be interfering with it.
