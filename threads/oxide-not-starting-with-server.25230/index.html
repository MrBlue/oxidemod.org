  2. Wulf Community Admin

    The oxide.config.json is currently installed under the server root, and is included in each release build currently.

    The CSharpCompiler.exe is automatically downloaded when you start the server with Oxide, unless you have something interfering.

    If you do not see any indication of Oxide and yet the server runs, then Oxide is not installed. If you have any sort of updating scripts, they are likely uninstalling Oxide when you start the server. Make sure you are installing Oxide last, as it is a mod.
     
    Wulf, May 1, 2017
