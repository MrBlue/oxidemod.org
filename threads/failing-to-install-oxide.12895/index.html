  2. Wulf Community Admin

    1. Download Oxide for Reign of Kings | Oxide
    2. Extract it to your desktop in a folder
    3. Stop your Reign of Kings server (server, not client)
    4. Copy/move all of the files from that folder to the root of your server (where ROK.exe is)
    5. Overwrite all files if prompted
    6. Start your server, done
