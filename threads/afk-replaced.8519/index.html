    AFK Kick - Kicks players that are AFK (away from keyboard) for too long

    AFK Kick automatically kicks players that are AFK (away from keyboard) for too long. The default AFK time limit is set to 300 seconds (5 minutes).

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use oxide.grant user <username|steamid> <permission>. To remove a permission, use oxide.revoke user <username|steamid> <permission>.
