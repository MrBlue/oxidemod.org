    yurkin_ya said: ↑
    when you write /sethome 1, the circle appears, TELL me how to remove this circle? View attachment 30664

    That's the debug sphere that is displayed when you're an admin and you've set home.

    To remove this feature simply edit the .cs file and around the lines 1109 you should see this:

    Code:
    {% raw %}
                if (player.IsAdmin())
                player.SendConsoleCommand("ddraw.sphere", 60f, Color.blue, GetGround(positionCoordinates), 2.5f);
    {% endraw %}
    Simply delete that and save the file.
