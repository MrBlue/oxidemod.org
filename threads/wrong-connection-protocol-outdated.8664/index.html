  8. Wulf Community Admin

    MLGENIE said: ↑
    So what exact steps would i take to update my server?
    Click to expand...
    1. Stop the server and run SteamCMD with the validate option.
    2. Start the server, connect to it to verify you can connect.
    3. If you can't connect, open Steam and validate your Rust client.
    4. Download the latest, compatible Oxide build from the Downloads page.
    5. Extract Oxide-Rust.zip to your server's root and overwrite existing files.
    6. Start the server, and try connecting again.
     
    Wulf, May 22, 2015
