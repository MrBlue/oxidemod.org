  6. EnzoWillems

    Set this in your .bat
    Code:
    {% raw %}
    @echo off
cls
:startYourServer.exe -batchmode -nographics -exec "host 12781;queryport 12881;;maxplayers 240;servername "YourServerName"@echo.
@echo Restarting server...
@echo.
goto start
    {% endraw %}
