  7. PaCe

    If there is not enough population there a few ways to improve that. 3rd march is rust-wipe. You should go with it and start up after this with your server. For 3-4 days you dont need to wipe. Here are some tipps for you to get more players...

    Some tipps how we grew up:

    1) be active on your server
    2) support players, if they have any questions/reports
    3) do not buy a server because your reason is "i want to have an own one"
    4) tell all your friends to play there
    5) do not help your friends with any adminside help
    6) have a low/good ping and good name (best plugins listed in shortnames)
    7) kits are very important - most players leave, if there is none
    8) you need to know much about rust, you say, 200hrs of rust is enough? i say, no!
