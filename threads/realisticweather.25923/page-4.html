  4. TwistedHelixis

    Install timed exec, then in the oxide config folder you should have the timedexec.json file.

    Mine is like this
    {
    "EnableInGameTime-Timer": false,
    "EnableRealTime-Timer": false,
    "EnableTimerOnce": true,
    "EnableTimerRepeat": true,


    "InGameTime-Timer": {

    },
    "RealTime-Timer": {

    },
    "TimerOnce": {
    "weather.rain 0": 35,
    "weather.clouds 0": 40,
    "weather.fog 0": 45,
    "weather.wind 0": 50
    },
    "TimerRepeat": {
    "rw random": 7200
    }
    }
