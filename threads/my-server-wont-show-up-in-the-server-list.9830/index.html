  3. SweetLouHD

    Check your firewall settings. It might be blocking UDP 7350 traffic.

    It would not be port forwarding as the information is not being received by Code }{atch. Port forwarding only affects incoming traffic.

    You still need to have 7350 forwarded to actually allow people to connect, but this particular issue is most likely firewall.
