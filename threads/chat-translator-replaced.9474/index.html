    Chat Translator - Translate foreign messages to english in your chat

    If you run a English Only server but have a few people who still speak in a foreign language, this plugin will do its best to translate it into english and say it back to the server.

    You will need to do 2 things before you setup this plugin:

    You need to visit: https://detectlanguage.com/

    Create an account and get the API Key, once you run the plugin once, ensure you set your API key.

    You then need to visit: http://frengly.com/#!/translate
