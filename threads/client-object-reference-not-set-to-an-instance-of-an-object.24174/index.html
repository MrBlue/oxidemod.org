    I have had this error a few times, it usually had to do with my server not running correctly or me not connecting correctly. First make sure all settings and the ip is set-up correctly, then make sure the requiered ports are open (28015 and 28016 in most cases). If you try to console connect make sure you put the port behind the ip (connect IPGOESHERE:28015).
