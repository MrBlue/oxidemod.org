  6. toothless

    I just installed this for the first time and it did nothing (like comments here in recent weeks). No errors thrown, the reply in game announced a successful copy and successful paste, but pasting didn't actually do anything.

    I found the created json files for my structures had no entities in them at all.

    The problem was fixed by editing the oxide/config/CopyPaste.json file and restarting the server. For me, I used these:

    Code:
    {% raw %}
    {
  "Copy Options": {
    "Buildings (true/false)": true,
    "Deployables (true/false)": true,
    "Inventories (true/false)": true,
    "Share (true/false)": true,
    "Tree (true/false)": false
  },
  "Paste Options": {
    "Auth (true/false)": true,
    "Deployables (true/false)": true,
    "Inventories (true/false)": true,
    "Vending Machines (true/false)": true
  }
    {% endraw %}
