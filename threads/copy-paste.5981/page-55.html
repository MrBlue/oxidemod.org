  18. Mr.SnivelUMPagus™

    coopalooop22 said: ↑
    ^^ if you try to paste in an old file...it says that the file is empty
    Click to expand...
    At the top of the file it says where it says "deployables" you can change to "entities". Also where it says "yrotation" you can change it to "rotationy".
    If you do these changes, it will no longer say the file is empty. It will say that you have successfully pasted the structure, but there will be nothing there.

    That's as far as I got on trying to convert an old copy into the new format. I guess you can go through the whole file and find all the other issues, but it will be a lot of work.

    There is no way to get the old copypaste to compile and work on the newer Rust updates, so the only possibility of recovering an old base file is to try to correct all the things that changed in the file.

    The only thing I didn't try is to import the old base file in fortify, and make a minor change, then hope it saves the file that will work in the new copypaste. Of course you need the steam version of fortify, and I don't know if it will work.
