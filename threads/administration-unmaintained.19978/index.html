    Spicy submitted a new resource:

    Administration - Full administration plugin.

    This plugin serves as a full administration plugin for Rust Legacy. It's like Oxmin from Oxide 1.18. It may override some other plugins, if you do not want this, you may disable certain commands via the config.

    Commands:
    Code:
    {% raw %}
    /info - plugin info./kick [playername] - kick this player from the server.
/ban [playername] - ban this player from the server.
/banid [SteamID64] - ban this SteamID64 from the server.
/unban [SteamID64] - unban this SteamID64 from the server.
/close...
    {% endraw %}
