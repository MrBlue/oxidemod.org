  2. Freaky

    N Teleportation for Rust | Oxide

    "HomesEnabled": false,
    "TPREnabled": false,
    "TownEnabled": false,

    and you have only:
    /tp player (admin tp to player)
    /tpn player (admin tp near the player)
    /tp player1 player2 (tp player1 to player2)


    see the Admin commands on Overview !
