  4. StreetDog

    BetterChat

    go to DATA then BETTERCHAT then open Groups.json

    here you have:


    Code:
    {% raw %}
        "GroupName": "default",
    "Priority": 0,
    "Title": {
      "Hidden": false,
      "HideIfNotHighestPriority": false,
      "Size": 15,
      "Color": "#9EC326",
      "Text": "[Player]",
      "InheritSize": false,
      "InheritColor": false
    },
    "PlayerName": {
      "Size": 15,
      "Color": "#9EC326",
      "ClanTagFormat": "[{Clan}] {Name}"
    },
    "Message": {
      "Size": 14,
      "Color": "white"
    },
    "Formatting": {
      "Console": "{Title} {Name}: {Message}",
      "Chat": "<color=red>({Level})</color> {Name}: {Message}"
    {% endraw %}
    This will show the level on red on chat
    Feel free to change it
