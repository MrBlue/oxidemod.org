    Yes, you are correct. You grant permission through the console. I'm not sure if it's also necessary to reload the plugin, but I do anytime I make changes to any plugin. So, you would type oxide.reload AntiOfflineRaid in the console to reload the plugin.

    If you want it granted to admins and moderators as well, you also need to add the permissions for those groups as well.

    So you would use these permissions:

    grant group player antiofflineraid.protect

    grant group admin antiofflineraid.protect

    grant group moderator antiofflineraid.protect

    I hope that helps.
