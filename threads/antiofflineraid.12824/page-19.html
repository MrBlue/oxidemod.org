    No it just means that if you're in a clan and your minmembers is set to 1, if anyone from that clan comes online, then protection disabled. likewise it wont turn on until the last member of that clan goes offline. Be aware people can abuse the system by 1 guy building the base and other people live in it but not using clans, this means if someone tries to raid that base and the guy who built it is offline but the other guys are on, they are kind of invincible inside the base depending on protection level. I've set a rule that if you live with someone you have to be in a clan to avoid this exploit.
