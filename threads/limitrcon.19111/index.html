    Wulf submitted a new resource:

    LimitRCON - Limits RCON access to specific IP addresses

    LimitRCON is a basic plugin that limits RCON access to specific IP addresses listed in the config file. It will allow all local and LAN addresses automatically as well.

    Note: To use this, you'll need to have the latest Oxide release 2.0.2104.
