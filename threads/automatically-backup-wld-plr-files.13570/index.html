Automatically backup .wld & .plr files

    Hey guys, I've been having a problem with my save files getting corrupted on server restart. Here is my solution for backing up the saves. This solution will probably only work for people with RDP access to their server.

    1. Download and install Beanland: AutoVer.
    2. Open it up and click the button to "Add new Watcher".
    3. Go to the "General" tab - http://i.imgur.com/p6TmqRI.jpg
    • Name: Whatever you want
    • Watch folder: Select your Hurtworld directory
    • Backup to: directory for all backups
    • Uncheck the "Include sub folders" option.

    4. Go to the advanced tab - http://i.imgur.com/72qQr15.jpg
    • Include Files: autosave*.*
    • Exclude Folders: appcache;config;Hurtworld_Data;logs;oxide;steamapps
    5. Leave everything else default and click "OK".
    6. Test to make sure it's working by running the "saveserver" command. Once the server is finished saving, you should see two new backups of your .wld and .plr files.

    Every time the server saves, it will create a new backup.

    **UPDATE: After using this for about 4 days, and my server save growing to over 18mb, the backup folder is about 20gb. I've also noticed that as the server is saving larger and larger files, autover is detecting the wld save as being changed 2 or 3 times per save, so make sure you use the largest file for each given save event.
