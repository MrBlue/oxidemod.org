    A Server Essentials and Admin Panel Plugin
    If you've ever played Minecraft or Garry's Mod, then you know exactly what I'm about to say. I'd love to see a Server Essentials plugin. What's that, you may ask? Well, the basic tools for staff and users alike to make a server more user-friendly and useful! Adding/combining plugin commands like !kick/ !warn/ !ban/ !smite/ !motd/ !help/ !admin / !tp(without appearing asleep)/ !homes/ !warps/ !me/ !info /!whois / etc etc. A bunch of these are all separate plugins, and a lot of the times they don't work well with each other. It just makes much more sense to me to have all that sort of "expected" plugin material all bunched into one beautiful, well-crafted, and compatible plugin, and just called it Oxide Essentials!
