  2. Wulf Community Admin

    1. Would require a client-side mod, so not possible via the server. This would be more possible with Steam workshop if they support it further.
    2. Not possible, the Rust CUI only supports adding custom GUI elements onto the screen.
    3. Player speed is client-side, so not really possible, or at least not without disabling some anti-cheat and being a bit hacky.
    4. Not sure if this is possible, but likely would be with a little work.
    5. You could probably hack something up, but it'd be mostly behind the scenes, not visually the same.
    6. May be possible, but not visually most likely.
    The Docs that Oxide provides are mainly for Oxide, not for each game. The properties/methods/functions that are in games are found in each game's DLL, they aren't provided by Oxide itself. We may provide some generated docs eventually for each game, or just wait until we wrap a lot more.
