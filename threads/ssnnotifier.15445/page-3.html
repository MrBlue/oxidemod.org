  3. Umlaut

    Was added JSON-API:
    • api/server_name/statistic/killers
    • api/server_name/statistic/victims
    • api/server_name/statistic/distances
    • api/server_name/statistic/tattlers
    You can use the timerange_type variable to restrict statistics on time.
    timerange_type = 0 - without restricts
    timerange_type = 1 - statistic for a day
    timerange_type = 2 - statistic for a week
    timerange_type = 3 - statistic for a month

    Like this: http://survival-servers-network.com...tic/time_spent_on_the_server?timerange_type=1

    timerange_type by default is 0.
