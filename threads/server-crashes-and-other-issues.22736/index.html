  5. Mughisi

    Oxide doesn't affect damage dealt by players in any way. The only thing that could be altering the damage dealt is a plugin or the server settings (server.meleedamage, server.arrowdamage, server.bulletdamage and server.bleedingdamage). Default value of those settings is 1.

    If the problem persist and all those server options are set to 1 you can always unload all the plugins and load them back in one at a time to figure out which plugin is causing it.
