    SetHome - Allows players to set a home teleport point

    SetHome allows players to create unique home points for teleportation. This plugin uses HelpText for printing a list of available commands, which includes:
    • /togglehome - Admins may enable and disable SetHome at any time.
    • /sethome - Players create a home position to which they can tp.
    • /removehome - Players must remove a home position before setting a new one.
    • /home - Players tp to their previously set location.
