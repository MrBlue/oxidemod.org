  2. sqroot

    They built Oxide themselves.
    Keep in mind that this can get you blacklisted.

    Hosting it via an external FTP server is not possible because Rust:IO serves a live map, and as such it has to constantly communicate with the game server.
    To generate a non-live map, simply generate the map you want on playrust.io and link it somewhere.
