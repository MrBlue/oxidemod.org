  1. Domestos

    Domestos submitted a new resource:

    Player List - Allows users to see who or how many are users are currently online

    Description
    This plugin allows your users to see what players or only how many players are currently online.

    Usage
    /who - shows either how many or what players are online

    Default config
    Code:
    {% raw %}
    {
  "Settings": {
    "MaxPlayersPerLine": 8,
    "OnlyShowPlayerCount": true
  },
  "Messages": {
    "PlayerCountMessage": "Currently are {count} players online",
    "OnePlayerMessage": "You're the only player online",
    "PlayerNameMessage": "Players online: "...
    {% endraw %}
