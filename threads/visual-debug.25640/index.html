  1. Jake Rich

    Jake Rich submitted a new resource:

    Visual Debug - Allows you to view all variables, and call functions inside any running plugin.

    You need to have ownerID on the server to use the command.

    Use "/visualdbg" to show, or hide the explorer.

    Click the green square to explore the object, red square to go to the parent object, and blue to call a function.

    If the function returns an object, you will explore it. Otherwise it will print the result in chat.
