  1. dcode

    dcode submitted a new resource:

    CupboardRadius - Makes the cupboard radius a sphere and its radius configurable.

    The August 27th, 2015 update changed the cupboard build zone shape so that it's sometimes possible to place another cupboard bellow an existing one, in some configurations resulting in a situation where a player is able to demolish lower levels of a building.

    This plugin changes the cupboard build zone back to a sphere and also makes the cupboard radius configurable (defaults to 25).
