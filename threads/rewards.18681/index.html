  1. Tarek

    Tarek submitted a new resource:

    Rewards - Reward players with money (Economics) for killing animals, other players, and activity.

    Reward players with money (using Economics plugin) for killing animals, other players, and activity.

    Console Commands:
    /setreward <reward name> <value> - Sets rewards config and saves it.
    /showrewards - Shows current rewards config.

    Default Configuration:
    Code:
    {% raw %}
    {
  "human": 50.0,
  "bear": 35.0,
  "wolf": 30.0,
  "chicken": 15.0,
  "horse": 15.0,
  "boar": 15.0,
  "stag": 10.0,
  "ActivityRewardRate_minutes": 30.0,
  "ActivityReward": 25.0
}
    {% endraw %}
