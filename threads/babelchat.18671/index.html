    Wulf submitted a new resource:

    BabelChat - Translates chat messages to each player's language preference or default language

    BabelChat automatically translates chat messages to each player's language preference or the server's default language. By default, the plugin will use random colors for player names, but you can change that in the config.

    Configuration
    You can configure the settings in the BabelChat.json file under the oxide/config directory.
    Code:
    {% raw %}
    {
  "ForceDefault": true,
  "PrefixColors": true
}
    {% endraw %}
    To set the default server language, use the "lang" console...
