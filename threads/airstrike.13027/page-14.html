  3. k1lly0u

    Lol that diagram! You assume correct, though I am pretty sure it never dropped rockets directly above the target.
    Yes the old version did send them all to one position but I felt a blanket coverage was much more appropriate for the squad strike. The accuracy you set for it will only determine the grouping of rockets from each plane, so a very low accuracy on a squad strike will still have 3 groups of rockets, but all the rockets in the individual groups will be tighter together. The distance between the groups is 1/5 of the distance between the planes
