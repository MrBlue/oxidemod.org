    There you go. The last thing you need to add is the zoneid from the zone you made around your arena. Type /zone_list and next to the zone you created will be a zoneid #...usually 8 digits long but can be shorter sometimes. Make sure you enter that number in the config in all 4 autoevent setting. Right now they all say "null"I just set each event as tdm because honestly it doesn't do very well if you try to mix game types in autoevents. Also you have to make sure you follow the directions for the tdm plugin as far as setting up and naming the 2 separate spawnfiles. I just used the default name for the tdm spawnfile to be used in your config. Also make sure you have the kits plugin and you create a kit called tdmkit with whatever you want people to spawn with. A few things to note...1. Right before you're ready to do the /event launch to get them going make sure you reload the EventManager plugin 1st or there's all sorts of different bugs that can happen sometimes. 2 Once you start autoevent it will run through it 4 times like in the config but then it stops and needs to be launched again after the 4th one.. I don't know about tdm but I run Gungame on my server and the maxplayer setting never works so make sure you have plenty of spawnpoints if you have a lot of people playing. I set it up so 2 of them run until some team hits the kill limit and the other 2 will run like battlefield style for 900 seconds so make sure and set a decent kill limit in the tdm plugin config
