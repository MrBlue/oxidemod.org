    3e) quit

    2. Now download the latest Kortal client and open the "Hurtworld_Launcher.exe" once. After that you can close it again ^^ the first start create some other files
    that you will need now.

    3. Copy everything from the Kortal client except "Hurtworld_Launcher" and the folder "HurtworldClient_Data"
    (if you can see invisible data, you will see two other exe's : "Hurtworld.exe" and "HurtworldClient.exe" - don't copy these,too - so if you see these copy everything except Hurtworld_Launcher.exe, HurtworldClient_Data, Hurtworld.exe, HurtworldClient.exe)

    4. Paste all the copied stuff in the server folder you downloaded in step 1

    5. Click on "host_server.bat" / "host.bat"

    for me didn't this worked so I found another easy solution:

    6. Download Oxide
    Oxide for Hurtworld | Oxide
    It's an API for hurtworld servers.
    (you have to register yourself to download...^^)

    7. Copy the Oxide files into your server folder

    8. Start the server with the oxide bat file

    9. You have to execute the command in the server command line to deactivate the easy anti cheat module.
    If it's an "offline" server only for yourself or your friends you don't have to fear cheater so I think there won't be a problem if you deactivate it

    10. Play with your kortal client and connect to your ip (yourself 127.0.0.1 or for lan your lan ip)
