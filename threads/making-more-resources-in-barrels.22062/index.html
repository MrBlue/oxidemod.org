  9. StreetDog

    Dima1809 said: ↑
    in my language there is no overview, but English very hard for me to look
    Click to expand...
    For a new as you, will be a lot better to use MagicLoot:
    Magic Loot for Rust | Oxide

    Code:
    {% raw %}
    {
  "Developer": {
    "AmountChange": false,
    "Debug": false,
    "ExtraItem": false,
    "Skins": false
  },
  "exclude": {
    "list": [
      "supply.signal",
      "ammo.rocket.smoke"
    ]
  },
  "Loot": {
    "AmountMin": 1,
    "Enabled": true,
    "ItemsMax": 3,
    "ItemsMin": 1,
    "PreventDuplicates": false,
    "WorkshopSkins": true
  },
  "Settings": {
    "Multiplier": 1
  }
}
    {% endraw %}
    Just adjust the
    "Settings": {
    "Multiplier": 1
    to 10 then you will have x10 loot
