    You can run Rust Dedicated Server from any OS, the difference between Windows 7, all the way to Windows 10 is negligible, and just because it's Windows Server, doesn't mean it's automatically giving you a massive boost, because 2016 is very much on par with Windows 10.

    It all comes down to how it's configured and setup, a poor configuration of Windows on any hardware will still give you poor performance.

    As for the Ovh setup, moving from DDR3 1333 to DDR4 2400 will give a nice boost (nothing massive), and the difference between the CPU's is around 15/20% depending on what you are bench marking.

    Going from standard SSD's to NVMe would be one of the biggest changes, smoother saves and faster restarts. As it can be upwards of 6-7 faster for Read / Write. Just remember you will need to have a OS that supports this, I know personally that Windows 2008 R2 and Windows 7 can't be used unless you slipstream the driver into the OS before loading it, and even then you might run into problems.
