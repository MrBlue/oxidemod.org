Discussion in 'Rust Development' started by Skipcast, May 2, 2017.

  1. Skipcast

    The hook is only called when a player gets helped up by another player. There is still at 20% chance that the player will get up on his own. In this case the hook is not called.

    You can see where to inject the hook in the BasePlayer.WoundingTick method.

    The code is currently:
    Code:
    {% raw %}
    if (UnityEngine.Random.Range(0, 100) < 20)
  this.SetPlayerFlag(BasePlayer.PlayerFlags.Wounded, false);
else
  this.Die((HitInfo) null);
    {% endraw %}
    Suggested fix would be:

    Code:
    {% raw %}
    if (UnityEngine.Random.Range(0, 100) < 20 && Interface.CallHook("OnPlayerRecover", this) == null)
  this.SetPlayerFlag(BasePlayer.PlayerFlags.Wounded, false);
else
  this.Die((HitInfo) null);
    {% endraw %}
