  2. Calytic Community Admin Community Mod

    Hey Inofix,

    So, it looks like the answer to this question can be found in the Vis.Colliders method (in Assembly-CSharp).

    Here is the code..
    Code:
    {% raw %}
      public static void Colliders<T>(Vector3 position, float radius, List<T> list, int layerMask = -1) where T : Collider
  {
    foreach (Collider collider in Physics.OverlapSphere(position, radius, layerMask))
    {
      T obj = collider as T;
      if (obj.transform.CompareTag("MeshColliderBatch"))
        obj.transform.GetComponent<MeshColliderBatch>().LookupColliders<T>(position, radius, list);
      else
        list.Add(obj);
    }
  }
    {% endraw %}
    Below is example usage of that method. There should really be a PSA about people using Oxide with collider batching before the majority of mods have been patched. I have identified over half a dozen very popular mods that haven't updated and simply will not work with collider batching enabled.
    Code:
    {% raw %}
    List<Collider> colliders = new List<Collider>();
Vis.Colliders<Collider>(player.transform.position, distance, colliders);
    {% endraw %}
