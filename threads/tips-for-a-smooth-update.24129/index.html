  1. LoneSurvivor

    Hey, been hosting a server for almost two months now and going strong but I was wondering if there were any other server owners or anyone for that matter that had a smooth process for wipe/update day. It normally is about an hour process for me and my guys that host the server just due to the fact that after we update the servers plugins don't work and we then have to update oxide. I was just wondering if someone had a bullet list or a quick tip on how you smoothly go about wipe day that isn't stressful and doesn't take much time.

  2. Wulf Community Admin

    • Some recommend waiting for the client update to hit, otherwise players can't join if the network protocol changed
    • Wait for Rust update and compatible Oxide update (announced in Release Changelog thread)
    • Once Oxide update is released, shutdown server and update Rust
    • Shutdown server again if host autostart it, then update Oxide
    • Done! Unless there are breakages, plugins should be working
     
    Wulf, Feb 27, 2017
