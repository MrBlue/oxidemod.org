  2. Wulf Community Admin

    Wipes are only done when Rust updates or when hosts do it on their own accord. The Rust world save has nothing to do with Visual C++, so it sounds like they're feeding you incorrect information. Visual C++ is only used for Lua and CSharp plugins, and if they are using the latest builds, installing Visual C++ shouldn't be needed.

    To answer your question though, you can backup your world saves and player data by copying the save and persistence folders from server/yourserveridentity and then restoring them after. Keep in mind that if the Rust server says your map is outdated, it will force a wipe anyways. Wipes are almost an everyday thing with Rust Experimental right now.
