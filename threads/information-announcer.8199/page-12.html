    How Do I Edit The Announcer.cs To Put My Own Rules And Everything In For Rp Server

    When the plugin runs for the first time it will generate a file named Announcer.json in the config folder. You can simply edit the messages in that file to change them. After making the changes make sure your file is still valid json by validating it (www.jsonlint.com) and then update the server file and reload the plugin. Then the plugin should be using your own messages.
