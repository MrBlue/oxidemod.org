  2. Wulf Community Admin

    The config folder is created by Oxide, so if you don't see that, it's possible that you do not have Oxide installed. Make sure you've installed Oxide while your server is shutdown, and it has been started once with Oxide installed. Once it has run successfully, it will create the oxide/config folder.
