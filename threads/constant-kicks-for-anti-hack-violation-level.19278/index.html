  10. Colon Blow

    If you are using the FauxClip plugin to simulate Noclip, then yes, it is being seen by antihack. And you will have to either turn Antihack off or turn off the enforcement. Because this plugin has nothing to do with the Admin Noclip.

    Add one of these to server startup.
    to turn antihack off.
    Code:
    {% raw %}
    antihack.enabled false
    {% endraw %}
    or to turn off enforcement (aka being kicked)
    Code:
    {% raw %}
    antihack.enforcementlevel 0
    {% endraw %}
    The enforcement is what changed a few weeks ago. they defaulted from 0 (do nothing) to 1 (kick after reaching max level)

    But, if you are actually using the admin noclip function. That is different. it should not be caught by antihack. Me and my admins have not seen any issues even with antihack enabled.
     
    Colon Blow, Feb 28, 2016
