  1. k1lly0u

    k1lly0u submitted a new resource:

    XP Equalizer - Automatically sets a new players XP to the current server average and force set any player

    A simple plugin to automatically set a new players XP and Level to the current server average in hopes of giving new players a chance against long standing players. With a option to force reset any or all players to the average level

    Chat Commands
    /xpe reset all - Calculates the current server average and resets all players XP to that amount
    /xpe reset <playername> - Force reset <playername's> XP and level to the current server average

  "IgnoreAdmins":...
