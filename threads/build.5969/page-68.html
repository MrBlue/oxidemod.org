    The plugin doesn't currently have a way to print/display the entire list but I do believe the plugin dev is working on it as he mentioned on page 67. As far as I know the best way to get a full list is install PrefabSniffer available here--> PrefabSniffer It has a command specifically for the Build plugin (prefabs build) and will print out a list to the oxide/logs directory.
