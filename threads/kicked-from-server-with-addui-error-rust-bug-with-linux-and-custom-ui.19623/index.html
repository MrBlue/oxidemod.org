  2. Wulf Community Admin

    The Linux client on Rust does not handle non-English languages for the OS with the custom UI it supports. The reason it only happens on modded servers is that vanilla servers do not use the custom UI in Rust. You can fix it by setting your OS locale to en-US or en-UK.
