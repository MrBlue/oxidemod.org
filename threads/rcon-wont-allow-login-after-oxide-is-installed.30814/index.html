  2. Wulf Community Admin

    Oxide doesn't control or touch Rust's RCON unless you have Oxide's RCON enabled in oxide.config.json. For Rust, there's not really any reason to enable Oxide's RCON as it would essentially be the same as Rust's, but a bit more limited.

    Since you have +rcon.web set to 1 (this setting is from Rust and is actually default to 1 as of a few months ago), make sure you're using an RCON tool that is compatible with that.
