  1. FuJiCuRa

    Fujikura submitted a new resource:

    Magazin Booster - Can change the default magazin size of most weapons

    This small plugin can change your weapons default magazine size.
    The code is mainly based on StackSizeController (thx to this) and inspired by ItemConfig (thx no Nogrod). The intention was the simple need to take this config from map to map without the need to update the file with every new build.

    It will create a config-file in first start, this needs to be changed by your need and then the plugin needs to be reloaded. Finish.
