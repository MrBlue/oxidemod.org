  2. Wulf Community Admin

    No, there is no way to change the default group, that is managed by Oxide. You can change the title of it, the rank, etc. if you'd like, but the name should remain the same, otherwise it will not work. If you want to automatically add users to another group, you can do so using the "parent" system in permissions, or use a plugin to add them to it.
