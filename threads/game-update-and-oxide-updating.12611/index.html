  2. Wulf Community Admin

    Oxide is a mod, so you'd need to re-install the latest compatible build after each Rust update.

    Steps: Wait for Oxide update > Stop server > Update Rust Dedicated with SteamCMD > Download and install Oxide update > Start server.
