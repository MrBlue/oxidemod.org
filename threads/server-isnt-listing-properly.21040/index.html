  2. Wulf Community Admin

    Which permissions file are you referring to? The permissions.cfg from Reign of Kings is unrelated to Oxide. The permissions from Oxide are under oxide/data, and are only used by plugins, nothing else.

    As far as the server listing goes, Oxide doesn't manage or change the listing at all. If you can't connect to the server at all, then that sounds like your server isn't actually starting, in which case you'd need to check your server logs to resolve the installation/setup issue.
