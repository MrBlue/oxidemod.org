    I don't think a plugin already exists for this that's public, so you may have to edit existing ones. This is roughly how.
    Code:
    {% raw %}
    List<ulong> skins = new List<ulong>
{
    89031749887,
    30298740890
};// Then inside the craft hook, you need to override the skinID to be the result of this
var randomSkin = skins.GetRandom((uint) DateTime.Now.Millisecond);
    {% endraw %}
