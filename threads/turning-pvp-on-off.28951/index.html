  8. Diesel 42o

    In TruePVE config is where you will find that mapping, the zoneID comes from zonemanager. So any zone you want to be pvp, enter the zoneId into TruePVE config, save and reload TruePVE
    Code:
    {% raw %}
    "Mappings": {
"default": "default",
"pvpzoneid#1": "exclude",
"pvpzoneid#2": "exclude"
    {% endraw %}
    and so on..

  10. Diesel 42o

    remove the , after "36024149": "exclude", and add it to "default": "default"

    The last line of the section always has no comma, the ones above it need them.

    like this:
    Code:
    {% raw %}
    "Config Version": "0.8.8",
"Default RuleSet": "default",
"Configuration Options": {
"handleDamage": true,
"useZones": true
},
"Mappings": {
"default": "default",
"36024149": "exclude"
},
"Schedule": {
"enabled": false,
"useRealtime": false,
"broadcast": false,
"entries": []
    {% endraw %}
    No code changes required anymore ZoneManager was updated.
