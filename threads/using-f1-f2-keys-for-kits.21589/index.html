    It's a console command/setting provided by Rust.
    Code:
    {% raw %}
    bind key command
    {% endraw %}
    Examples:
    Code:
    {% raw %}
    bind m "chat.say \"/map\""
    {% endraw %}
    Code:
    {% raw %}
    bind p status
    {% endraw %}
    Most likely they used a plugin to send these to your client as a command.
    Code:
    {% raw %}
    rust.RunClientCommand("bind f1 \"chat.say /kits starter\"");
    {% endraw %}
