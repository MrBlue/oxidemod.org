    Discord Group/Role Sync
    There's many amazing, useful Discord plugins out there... But none of them, at least to my knowledge, have the ability to sync roles with it. What do I mean? I mean, usergroups are like roles for Discord; just an in-game form of it. Wouldn't it be fantastic if the roles in the Discord matched the usergroups in-game? I know it would make things SO MUCH easier for my and my community when it comes to assigning staff, support, donors, regulars, vip, etc. So for instance, if I set someone to "usergroup donor" moments later it would apply the "donor role" in Discord. And vice-versa (if I set someone to "donor role" in Discord, it would sync up with the Rust server and assign them "usergroup donor" in-game! The idea is sounds, but is it doable? Has it already been done? If-so where and how!? Also, sidenote, Syncing bans would be nice as well, if possible.
