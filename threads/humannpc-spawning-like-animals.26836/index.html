  2. Steenamaroo

    Hi Yannick.
    I'm working on BotSpawn for Rust | Oxide and introduced custom spawn locations last night.
    It is written with groups and specific areas in mind, but you can set the spawn radius to very large and the number of bots to one (per location), if you want to have custom, randomly placed, single bots.

    It uses the upcoming FacePunch 'scientist' AI which, relatively speaking, is pretty smart.
    It'll take cover/retreat/rush etc.

    You can customise firing range, damage, health, accuracy, clothes and weapons, ground spawn or parachute in.....Lots of cool stuff.
    The plugin will also generate random names for each bot, if you want.
    If not, you can override.

    It will be actively developed as the AI improves, so feel free to get your suggestions in.
