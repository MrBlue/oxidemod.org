    Beds Limit - allows you to limit the amount of beds placed per base

    Permission for non admin use
    oxide.grant user userName bedslimit.allowed

    Chat commands
    "    /bedslimit" shows current values
    "/bedslimit maxbedsallowed" set the amount of beds allowed
    "/bedslimit radius" set the radius in which beds will not be allowed
    "/bedslimit update" use this command after editing the config file
