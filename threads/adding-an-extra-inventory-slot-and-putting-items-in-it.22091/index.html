  4. Calytic Community Admin Community Mod

    Look at "Item.MoveToContainer"

    Code:
    {% raw %}
    public bool MoveToContainer(ItemContainer newcontainer, int iTargetPos = -1, bool allowStack = true)
    {% endraw %}
    Note the second argument. By default, MoveToContainer checks to capacity of the container you are moving the item to with a "CanMoveTo" check.

    You could try writing your own version of MoveToContainer without the CanMoveTo check.
