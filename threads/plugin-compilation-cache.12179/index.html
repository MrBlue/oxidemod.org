  2. Wulf Community Admin

    C# plugins (.cs) are compiled and then cached. In the event that you try to load a non-working version, Oxide will try to use the previously used working version. There is no need to disable anything, plugins will update if they compile successfully. If you are having issues, please check your oxide/logs for errors, and provide them here if so. Also make sure you are fully updated to the latest Oxide builds, including all DLLs.
