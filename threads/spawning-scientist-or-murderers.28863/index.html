  3. Quantum

    Scientist:
    Code:
    {% raw %}
    nav_grid true
    {% endraw %}
    Murderers:
    Code:
    {% raw %}
     halloween.enabled true
halloween.murdererpopulation 2 (Population active on the server, per square km (0))
    {% endraw %}
    Add those to your startup or put it in the console. NOTE: if you put it in the console you also need to type server.writecfg after putting it in the console to save it for next restart.
