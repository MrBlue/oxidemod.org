  1. Mordeus

    Mordeus submitted a new resource:

    Day/Night Message - Broadcasts a custom message at dawn and/or dusk

    A very simple Universal Day/Night message plugin.

    Works with Rust, Reign of Kings, 7 Days to die, Hurtworld ItemV2, and probably more...

    You can turn off either the Day or night message in the config if you so desire. Dawn and Dusk time can be changed in the config, messages are changed in the Lang file.

    Config:
    Code:
    {% raw %}
    {
  "Dawn Time": 7.0,
  "Dusk Time": 19.0,
  "Title": "[#00ffff]Server:[/#]",
  "Use Day Message": true,
  "Use Night Message": true
}
    {% endraw %}

    Localization is per user. If a server only has one language, then only that language is used. If a server is forcing another language, then that language is used. If a user sets their language to something other than the server, then they will see those messages if available in that language instead of the rest of the server.
