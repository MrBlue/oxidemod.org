  2. Crytex

    All easy.
    Start server and in console type server.writecfg
    Looks here file server/%IDENTITY%/cfg/users.cfg
    And add in file line - ownerid 76561497564366699 "your game name" "admin"
    Restart server. When you starting connected again in console you will see
    76561497564366699: Manifest file differs (possibly out of date)
    your game name has auth level 2
    your game name joined

    Your <steamid64> you can find in log, add -logFile "log.txt in server start.bat file. Log file located in server main folder.

    You are admin now. Hope help
