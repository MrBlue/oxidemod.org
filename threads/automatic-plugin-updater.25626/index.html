  5. Togoshige

    Ive used PluginUpdater Tool ----> Plugin Updater (Windows) for Rust | Oxide

    Cons:
    - some plugins that are updated have breaking config file changes, so auto updating the plugin may break/reset the config file
    - some plugins do not have a ResourceID so you arent able to check for updates for those plugins (but most plugins seem to have a ResourceID)
