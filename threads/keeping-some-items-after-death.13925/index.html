    Make sure you use rcon before the command, also check your autoexec.cfg file to see if it has been added. If the command is not listed add it to the file.

    It should look something like this:
    Code:
    {% raw %}
    quit 86400
loadbalancerframebudget 100
playerlootmode 0
servername New Hurtworld Server
    {% endraw %}
    Once you add it, restart the server and you're good to go!
