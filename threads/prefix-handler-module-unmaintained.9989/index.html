    Prefix Handler Module - Prefix Handling Module for RanksAndTitles, handles multiple prefixes that users can choose from

    Prefix Handler is a Module or "Add on" for RanksAndTitles it WILL NOT work without RanksAndTitles and it does not work with other plugins.

    This Module will allow your users to have multiple plugins (based on groups) and allows the user to switch between prefixes they have access too.

    This Module has ALL of its configurable info inside the RanksAndTitles Config file, it uses this and the data file to work properly. So it will not use any of its own.

    Code:

    However since my RanksAndTitles uses prefixes as a permission controller and group permissions handler it may cause issues since it uses the oxide.user file to find permission groups to set available. Since it's also based off the config structure of RanksAndTitles that also may pose a problem.
