  4. sqroot

    You can't rewrite the code of a Rust class. Rust is explicitly referring to the class in the DLL, not to your class. If C# allowed that, it'd be seriously confusing, and the equivalent of the following: COMEFROM - Wikipedia, the free encyclopedia.
    There's maybe some reflection wizardry that allows you to do this, but you really shouldn't.
