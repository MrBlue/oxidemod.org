  2. Dustimittens

    In your config file there is a setting:

    Code:
    {% raw %}
    steamAuthTimeout = ''   # The amount of time before the steam authentication fails when a player connects.
    {% endraw %}
    Increase the amount of time the server waits before it gives up on authenticating. I set mine to 300. You can set it longer if you like.
