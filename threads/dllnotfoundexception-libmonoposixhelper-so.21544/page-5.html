    That will continue to show as long as zlib is not installed, or as long as CSharpCompiler and/or CSharpCompiler.x86 is not set as executable. Linux requires proper permissions to be set for your user to be allowed to execute programs.

    You can fix the permissions by using chmod +x CSharpCompiler (might need x86 if 32-bit server), or via your FTP properties on the file. If you don't have access to either of those, your host would need to set it up properly for you.
