  7. Smiddy

    ah that sounds like they are just hosting it on a linux server for you, it helps keep the costs down.

    Are you able to down through FTP from the server? most game hosts provide it.

    In the folder named 'Server' you should have a folder in there with the name of your server or it might just be called 'My Rust server' if you can download that it has all the save data / plugins / configs etc in it. Then all you need is to setup a rust server on your Windows PC and install oxide.

    Once you have done that make the startup batch file (there is an example one with the oxide download) just make sure it has your server name correct and the right seed etc, use that to start the server and will make the right server folder etc. once its started up stop it again.

    Then copy your server folder that you download over the top of the new one its made. Use the batch file again to start it up and you should see it finds the saved game and goes through deleting the entities it needs to.

    If you get stuck setting up the windows server here is a video to help you and there is also one on how to install oxide

    If you need more step by step on setting it up with your data let me know.
     
    Smiddy, Sep 29, 2016
