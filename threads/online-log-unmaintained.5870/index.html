    Wulf submitted a new resource:

    Online Log - Creates a JSON formatted log of all online players for use with external scripts


    Online Log is a simple plugin that creates a JSON formatted log of all online players for use with external scripts. The log file is located under oxide/cfg/onlinelog.json and is updated automatically when a player connects and disconnects from the server.
