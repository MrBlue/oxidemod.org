  5. Calytic Community Admin Community Mod

    AFAIK, You can't key by net ID or instanceID to keep box references across restarts.
    Code:
    {% raw %}
    public string GetEntityID(BaseEntity entity){
    Vector3 position = entity.transform.position;
    return "x" + position.x + "y" + position.y + "z" + position.z;
}
    {% endraw %}
    Once you can successfully store and retrieve StorageContainer instances, you can use the inventory property of StorageContainer to interact with items.
    Code:
    {% raw %}
    foreach(Item item in container.inventory.itemList) {  // do stuff
}
    {% endraw %}
