    You have an open process of Hurtworld.exe causing your port to be in use, causing the socket error.
    You can either restart your machine or run this command in your Run (Windows+R):
    taskkill /f /im Hurtworld.exe

    Make sure you've updated the server and Oxide before launching the server. Then try again.
