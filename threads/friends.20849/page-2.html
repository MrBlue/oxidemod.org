  14. dcode

    The plugin aims to be API compatible with FriendsAPI. However, if an already available plugin uses internal functionality of the latter that I am not yet aware of, there are three options: Either I add a shim for the respective internal API to this plugin, or the plugin author of the other plugin resorts to using the public API instead, or, last but not least, there is still a bug I have to fix.

    In theory, this plugin should work with everything that uses FriendsAPI. Friendly fire for Rust, however, is already included and configurable.

    If you have any additional information, let me know!
