    This plugin will trigger other chat commands when player write the command. For example, you are a server admin from Germany and you want to translate the chat command in to "Deutsch" and you don't want to deal with it again after plugin updates. This plugin will read other plugins and add the chat commands in to config file and you will just add what you want. Like, ChatCommand: help, -> TriggerCommand: hilfe , like this. and when the player write hilfe, it will trigger help.
