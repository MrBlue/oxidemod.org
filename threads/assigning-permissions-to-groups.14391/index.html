  2. Wulf Community Admin

    Oxide provides groups by default, for anything else check each plugin's Overview and post in their thread if you are confused about how to use them.

    Permission commands:
    Code:
    {% raw %}
    /grant <group|user> <name|id> *|<permission>
/revoke <group|user> <name|id> *|<permission>
    {% endraw %}
    Group management commands:
    Code:
    {% raw %}
    /group <add|set> <name> [title] [rank]
/group <remove> <name>
/group <parent> <name> <parentname>
/usergroup <add|remove> <username> <groupname>
    {% endraw %}
