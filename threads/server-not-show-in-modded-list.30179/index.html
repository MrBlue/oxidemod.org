  9. Razor14150

    ok I see this alot so Im going to post the host of solutions I have seen around.
    If your server isnt showing up theres a few possible issues
    A: If self hosting you most likely didnt open ports 28015 (for rust) or 28017 (for RCON)
    B: If being hosted by a server provider its possible the last person renting from them with that IP had gotten the IP blacklisted for fake populations or other blacklisted mods. Contact your server provider and ask them if this is the possible problem.
    C: There is a known issue between Steam and Rust where some servers just dont show up. The only possible solution Ive had with this issue personally is having my provider transfer what server my rust server is hosted on.
     
    Razor14150, Apr 10, 2018
