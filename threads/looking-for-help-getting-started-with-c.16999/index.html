  2. Calytic Community Admin Community Mod

    Generally you want to start by decompiling the Rust assembly (RustDedicated_Data/Managed/CSharp-Assembly.dll). I use dotPeek to accomplish this but there are some other good decompilers available.

    Free .NET decompiler :: JetBrains dotPeek

    You can add the Rust assembly and the other dlls in the Managed folder (mentioned above) as References in your VS.NET project for autocomplete, syntax highlighting, object browser.

    How to: Add or Remove References By Using the Add Reference Dialog Box

    Then review the oxide documentation and public plugins to get an idea of how plugins work.

    Oxide API Reference
    Rust | Oxide

    Aside from that all you need is a rust server to test it on.
