  2. KillParadise

    In the alert section you will get a notification whenever a new oxide update is released.
    you can check the oxide version by type oxide.version into your console, this will display the version you are running.

    Oxide already has all of the language extensions installed when you download/install it. The only one you need to download IF you choose to use it is Rust:IO, which has a good explanation of how to do so on its page.

  3. Wulf Community Admin

    Everything you download other than Rust:IO and Oxide itself would be a plugin, not an extension. If you'd like to get alerts on when Oxide is updated, you can watch it and receive email notifications by using "Watch this Resource" on the Downloads page.
