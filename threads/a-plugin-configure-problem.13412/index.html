  2. Wulf Community Admin

    Some plugins have data files and config files, they are not the same thing. The contents of the files should vary. The files under oxide/data should generally not be modified or touched, as they are only used for data storage by plugins. If you do see any config files under that folder, you may want to contact your host as they may be changing the location when they shouldn't be.
