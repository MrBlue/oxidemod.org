  2. Sir Eb

    Hello,

    did you connect your server already with a rcon - tool like rusty ? You have to execute the following command in there:

    Code:
    {% raw %}
    ownerid yoursteam64id admin
    {% endraw %}
    or place that users.cfg in "server\my_server_identity\cfg\users.cfg" put in there your steam64 id, and then boot your server and you should have admin permissions.

    Code:
    {% raw %}
    //Below this lines you can insert your Admins / Moderators , please do not use "//"
//Exapmle:   ownerid 76561198042113938 "admin" "no reason"
//            moderatorid 76561198042113938 "moderator" "no reason"
//Where 76561198042113938 has to be replaced by your own SteamID, only start your server after saving this file.
    {% endraw %}
    You'll find your steam64 id for example by logging into steam via your webbrowser and then going to "Profile". It's the number starting with 7 in your http - address line.
