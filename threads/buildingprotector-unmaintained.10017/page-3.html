    As a big fan of this plugin I suggest you to :

    1) Delete the buildingprotector in the plugin folder and config folder ( on FTP )
    2) Edit the .cs file directly with good hours using notepad++
    3) Restart your server without the plugin
    4) Put the buildingProtector.cs (with the correct hours you have introduced directly in notepad++) in the plugin folder on your FTP

    It should work properly.

    Mine is still working and I do this whenever the plugin starts to behave incorrectly.
