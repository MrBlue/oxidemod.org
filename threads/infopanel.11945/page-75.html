    Change the backgroundColor and make the dock not visible.
    Code:
    {% raw %}
    "BottomRightDock": {
      "Available": true,
      "AnchorX": "Right",
      "AnchorY": "Bottom",
      "Width": 0.19,
      "Height": 0.03,
      "BackgroundColor": "0 0 0 0",
      "Margin": "0.005 0.005 0.005 0.165"
    },
    {% endraw %}
    "BackgroundColor": "0 0 0 1", => 100% visible
    "BackgroundColor": "0 0 0 0", => not visible
