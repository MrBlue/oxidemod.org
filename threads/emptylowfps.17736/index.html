  1. Wulf Community Admin

    Dezito submitted a new resource:

    EmptyLowFPS - Set low fps.limit when no players connected

    A simple plugin that lowers the fps.limit (CPU usage) when no players is connected to the server

    In the config file you can specify:
    MaxFPSLimit: Max fps.limit when players connected (Default 256)
    EmptyFPSLimit: Max fps.limit when no players connected (Default 30)
