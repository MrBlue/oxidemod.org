    any text editor works (even notepads does the job), just make sure it ends with .json and not .txt.json.

    As for format, to put it very simple is based on "id":"value" (in this case due to be made as a config file and not "objects"). Most configurations requires to change "values" unless stated differently by the plugin dev him/herself.

    Edit: added a simple example of a random json schema

    Code:
    {% raw %}
    {
    "number_non_decimal": 1,
    "text": "A green door",
    "number_decimal": 12.50
}
    {% endraw %}
