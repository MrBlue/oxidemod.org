  4. Feramor

    You may not able to do that bcs you will need referances able to compile so you should use CallPluginFunction It will return the function's result anyway.It will return null if Plugin not found or got error.

    But you can
    Code:
    {% raw %}
    Core.Libraries.Plugins loadedPlugins = Interface.GetMod().GetLibrary<Core.Libraries.Plugins>("Plugins");
            Plugin CurrentPlugin = loadedPlugins.Find(PluginName);
    {% endraw %}
    CurrentPlugin will act like API in lua but you will need to call functions like
    object retrun_val = CurrentPlugin.CallHook("FunctionName", new object[] {"Call Faction Test"} );
