    Kappasaurus submitted a new resource:

    CupboardList - Displays a list of who is authorized on tool cupboards.

    CupboardList is a plugin which displays a list of who is authorized on a target tool cupboard.

    Command
    /cupauth - Displays who is authorized on the tool cupboard you're looking at. cupboardlist.able permission required.

    Permission
    cupboardlist.able - Required to use the command.
