    Compound Options - Allow to configure Compound Monument options

    Compound Options plugin - allows you to configure some options for a new monument "Compound" (scientists town)

    It's can:
    • Disable Compound SafeZone trigger (attack player when enter to area with loaded gun or attack any NPC), defaults - off
    • Disable Compound Turrets, defaults - off
    • Disallow Compound NPC (for rollback need restart server/respawn npc), defaults - off
    • Flush Compound vending machines inventory (disables), defaults - on
