  4. sililia

    Assuming you already know how to make a plugin, I'll skip the basics.

    You need to pull the Oxide.core repo from git and look at the code and comments because you'll be intimately working with it; if you make an extension (Here)

    Get the Oxide.rust repo too and look at how that is implemented since it's an extension of oxide.

    You'll need to make a class which inherits from Extension class and override.

    Name{get;set;}

    Version{get;set;}

    Author{get;set;}

    Load();

    You can override other functions but those are the most basic ones. For a description of what they do, look at the comments.

    You’ll need to add references that you’ll normally need for making plugin. You can look at the Oxide.Rust nugets to see which nugets you’ll need in order to develop you extension and compile it. You must compile it as .net framework 3.5 class library. Ones you’ve you compiled it, any dependencies that are not part of oxide must be added together with your extension in the managed folder
