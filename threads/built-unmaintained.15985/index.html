    Built - Provides entity ownership information without being dependent on external plugins

    !IMPORTANT!
    Do not use this plugin if you haven't wiped since the Rust update from the 11th of February, as Rust has only been tracking owner IDs since then.
    You will not be able to check ownership for buildings built before the patch from the 11th of February.

    Built provides entity ownership information for entities while not being dependent on any external plugins that track ownership.

    To check who built an entity, look at the entity and enter the chat command /built....
