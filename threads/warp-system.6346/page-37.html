    Click to expand...
    There is no permission per se....The permission is typed in when creating a Warp Point.

    Example: /warp add (warp name) (WarpTimer) (WarpRange) (WarpMaxUses) ("WarpPermissionGroup")

    "WarpPermissionGroup" = specific group and/or user/player/player's, you have created on server.
