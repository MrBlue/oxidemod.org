  2. Wulf Community Admin

    That means you're trying to use a build of Oxide intended for a different version of Rust. So you're server would either be outdated with Rust and trying to use a newer Oxide build with newer Rust DLLs, old you're server is updated with Rust but trying to use an older build of Oxide with older Rust DLLs.

    The door lock issue would be a direct indication of this, so you're likely outdated with Rust.

    Solution: Stop the server, run Steam to update Rust, download and install the latest version of Oxide.
