  6. › Genocide

    You can have the most badass server ever and still make a rust server lag with plugins. Sometimes your network connection at your server host will go wonky and cause lag. Sometimes there's a regional routing issue on the internet which causes several of OR a large handful of your players to lag. Sometimes it's a code issue with the game. So many reasons for lag...

    Best to start out small with plugins, then slowly add more using some logic on how heavy would this plugins' functions affect the server.. does the plugin do a lot of checks? Or does it only run every once in a while.. You will never be 100% lag free but you can prob be 95-99% lag free.
