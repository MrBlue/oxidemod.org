  2. PaCe

    I think this will happen because your server did not save nearly at the restart. So the players have enough time to put the items in their inventory after a finished save and at the time the server saves the map the items were in the quarry.

    Best way is to announce the restart.. type global.restart 300 as example and save it some seconds (20-15seconds before should be ok, in case use 30seconds first, 15 seconds again before restart) with save command and close the server manually some seconds later by hand (if you have a vps/root)
