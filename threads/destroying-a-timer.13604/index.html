  2. Wulf Community Admin

    It's a timer.Once, so it'll be automatically destroyed at the end. If you did a timer.Repeat, you'd either need to store them in a dictionary and then later remove it from that and destroy it, else set it as a variable and destroy the variable.
     
    Wulf, Dec 20, 2015
