    That's exactly what you need to do, do keep in mind that sometimes a config needs to be removed as well when there are major changes to the plugin but this should always be stated by the plugin developer in the update post.
    duker733 said: ↑
    Also, when you make changes to a config file for a plugin, the only way I know how to get the changes to take effect is to restart my server. Are there in game chat commands that can do this?
    Click to expand...
    You can use the console command oxide.reload, i.e. for m-Teleportation.lua you can use "oxide.reload m-Teleportation" (without the quotes ofcourse).
