    Each Oxide build includes both the SQLite and MySQL extensions and their dependencies, they have not be removed and we have not stopped bundling them, so your host is mistaken or confused regarding that.     For the OnProjectileAttack error, can't really say much without seeing the full and actual error.

    Without any useful information though, we can't really help with either issue. Useful information would be logs, any errors received, version information, if you've verified that Oxide is running (ex. testing the oxide.version command), etc.
