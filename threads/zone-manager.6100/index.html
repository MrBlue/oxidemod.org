  4. Reneb

    Coming on friday (it's already done, just need to wait for the update ):

    Zone Manager with:

    Enter MSG => Sends a message to the player when he enters the zone
    Leave MSG=> Sends a message to the player when he leaves the zone
    Godmode => all players are invul
    Sleeper Godmode => all sleepers are invul
    No Suicide => can't suicide
    Radiation => If you are in the zone you will get radiated
    Indestructible Buildings => no raiding
    No Building (with the Building Block box ) => Can't build in the area
    No kits => can't request a kit in the area
    No TP => can't TP out or in the area.

    And all that: LAG FREE !!

    Basically it uses the ingame zone manager that exist. Zones from ingame are:
    radiation zones
    building privileges zones
    temperature zones
    hurting zones (like the fires and stuff)
    So what this plugin does: makes a NEW zone.

    So the server manages the zone as if it was one of it's own, making the plugin lag free and timers free
