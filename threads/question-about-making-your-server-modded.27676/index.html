    Quick answer no.

    When you install oxide, by default your server will be set to modded, but you can edit oxide.config.json, there will be a option called modded (I think) and it will be set to true, change that to false and your server won't be modded no matter what plugin you put. But be careful as you may get blacklisted if you use plugins that change gameplay
