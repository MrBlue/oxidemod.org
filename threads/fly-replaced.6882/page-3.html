  12. VVoid

    Here's my quick workaround. Insert into Update method (after 55 line) and use E key to stop flying.

    Code:
    {% raw %}
                    if (input.IsDown(BUTTON.USE))
                {
                    enabled = false;
                    player.ChatMessage("Fly deactivated!");
                    return;
                }
    {% endraw %}

    p.s
    Basically almost any key can be used, so even BUTTON.CHAT will stop flying when you press T.
