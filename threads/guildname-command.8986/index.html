    Guildname command - Adds /guildname command to find the guild of an online player

    This simple plugin allows all users to type '/guildname playername' to find out the guild of an online player!
    The config file is auto generated, and if you want to make the command only able to be used by admins, you can add the admin's name to the file and set adminOnly to True.

    Current issues i'm working on:
    - if the guild's color is white, a [0] appears before the guildname in chat
    - there is a few unused members that don't need to be there
    - You should be able to get a list of admins...

    DumbleDora updated Guildname Command with a new update entry:

    Permissions fix!

    The plugin now grabs your admins from your RoK privileges itself so you no longer have to specify them by name in the config file.
    There is still a config file but it just has one True/False field for whether or not you want to limit the command to admins. This is by default set to False (all users may access the command.)

    The code has also been massively tidied up! For the clean up and sorting out the Permissions I have to thank CrZy!
