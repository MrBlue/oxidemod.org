    Canopy Sheep submitted a new resource:

    Staffmode - Toggle on/off staff mode

    Staffmode is a plugin that allows the toggling from one group (or set) to another group (set). This plugin is intended for staff groups, but it can be used to toggle other groups.

    Features:
    • Toggles to/from different groups
    • Logging on toggle
    • Multi-Group toggling supported
    • Toggles auth level/admin status
    • Dynamic data file for set groups
    • Customizable permissions
    Commands:
    • /toggle - Toggles staff status if granted permission(s)....

  17. Canopy Sheep

    Legit l IBP said: ↑
    What permission do I give to users to allow them to use the staffmode command?
    Click to expand...
    Grant the "PermissionNode" in the group data file, if there isn't one, anyone will be able to toggle with that group. In the snippet below, the permission would be "staffmode.mod"
    Code:
    {% raw %}
      "Groups": {
    "Mod": {
      "GroupName": "Mod",
      "AuthLevel": 1,
      "OffDutyGroup": "Modoffduty",
      "OnDutyGroup": "Mod",
      "PermissionNode": "staffmode.mod"
    },
    {% endraw %}
     
    Canopy Sheep, Jan 29, 2017
