    Hello. I'm a server owner and i have a really wierd problem with the playerbase on my server.

    We have 5-6 people that are hardcore (crazy hardcore, 10-14 hours per day). They rad more or less the entire map during this time. My server is stable with 25-30 players more or less all day long. I banned these hardcore players a couple of days ago and my player population grew to 50 players stable for a long time. I decided to unban these hardcore players and the raiding started. The server population dropped to a stable 25-30 people.

    I have been doing some caluclations and chart analysis when it comes to how many new people are joining the server each day, and how many stay for several days. And there is no doubt that this agressive raiding by the hardcore players are messing up with the playerbase on my server.

    So... Here is my issue. I don't want to ban harcore players. I would like to request a plugin that allowes raiding during xx hours with a xx hour break.

    Example. Raiding is allowed for 1-2 hours. Then abreak that last about 3-4 hours. When the break is over it all starts over with raiding allowed for another 1-2 hours untill the next break comes.

    I have seen a plugin that allows raiding only during night time, but this is not what i am looking for.

    Any suggestions? Can anyone make this kind of plugin?

    Thank you guys.


    EDIT: Optional PvP with a cooldown would be an good idea as well. If a player goes from PvE to PvP there will be a 5-10 min cooldown before he can go back to PvE.

    EDIT2: I forgot to mention that this is a raiding issue. If there will ever be such a plugin i am requestion, i would like the buildings to be unraidable(undestructable) as well.

    BEST SOLUTION:
    This can be helpfull .. http://oxidemod.org/plugins/indestructable-buildings.966/ and just unload it with my TimedExecute Plugin

    so my plugin's config should look like
    Code:
    {% raw %}
    {
  "RepeatTimedCommands": [
    [
      "save.all",
      300
    ],
    [
      "say 'hello world'",
      5
    ]
  ],
  "ShowTimedCommands": "true",
  "TimerOnceCommands": [
    [
      "oxide.load IndestructableBuildings'",
      1 -- It's better to execute it after you load the plugin
    ],
    [
      "oxide.unload IndestructableBuildings'",
      43199 -- Now after the load has executed the plugin will execute the unload command in half day
    ],
    [
      "reset.oncetimer",
     86399 -- Remember that these timers are connected to eachother so you need to do (before's command secs + the secs that you will wait untill it will reset the timer and start again) so you will wait 43199 not 86399 bcs the timer continues as i said
    ]
  ]
}
    {% endraw %}
