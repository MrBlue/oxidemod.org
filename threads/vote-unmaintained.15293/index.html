    Vote - Start a Yes/No vote

    Permission:
    Vote.admin

    How to Add Permission:
    /grant user UserName vote.admin
    or
    /grant group admin vote.admin (If you have an admin group)

    Commands:
    /vote (what you want to vote for).

    How to vote:
    Simple type yes or no in the chat. It will count towards the vote if there's one active. Votes default last 30 seconds. You can only votes once.

    Configuration File:
    Code:
    {% raw %}
    {
  "VoteEnded": "<color=yellow>[VOTE]</color> Voting has ended. Yes:...
    {% endraw %}
