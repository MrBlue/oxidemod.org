    [DOUBLEPOST=1515210170][/DOUBLEPOST]PreventLooting provides protection for entity of player (boxes, furnaces, careers, etc.) from looting by other players. Ideal for PVE servers, but can be used for PVP.

    You can configure the plugin to work only in the range of the cupboard or not. And all who will be authorized in the cupboard will have the opportunity to loot.
    The plugin can use FriendsAPI, ZoneManager and use permissions.
    Using FriendsAPI allows you to allow your friends access to your entities.
    Using ZoneManager allows you to disable the plugin in certain zones.
    If you want to share an entity with other players, you can use chat commands.
    The plugin has default messages in English and Russian.
