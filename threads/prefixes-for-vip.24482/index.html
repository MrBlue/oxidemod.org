    Betterchat does that in the data folder there is a file BetterChat.json open this in notepad++ and make a group for each one you have then edit this line
    "Text": "[Player]",
    and change whats inside [] for each group
    you will also need to set each group to true for
    "HiddenIfNotPrimary": true
    and set the Priority for each group
