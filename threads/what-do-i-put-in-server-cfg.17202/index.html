  3. Freaky

    Hi!

    The server.cfg needs to be in your server\{server.identity}\cfg

    In this file you can place for example:

    server.hostname "My Rust Server"
    server.port 28015
    rcon.port 28016
    server.maxplayers 50
    server.level "Procedural Map"
    server.worldsize 8000
    server.seed 439876134
    server.saveinterval 600
    server.headerimage "http://www.your-server.com/header.jpg"
    server.url "www.myamazingrustserver.com"
    server.description "My server is amazing!"

    if you change something, you need to save the file and after write in the console: server.readcfg (or readcfg), except the seeds and worldsize - need to restart the server!
