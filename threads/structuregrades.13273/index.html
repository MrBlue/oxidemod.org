  1. Wulf Community Admin

    Norn submitted a new resource:

    Structure Whitelist - Enable/disable specific building grades

    What is this?
    This is a simple plugin to block specific building grades, simple change true to false and the grade will be disabled.

    Default Configuration:
    Code:
    {% raw %}
    {
  "Grade": {
    "Metal": true,
    "Stone": true,
    "TopTier": true,
    "Wood": true
  },
  "Messages": {
    "NotAllowed": "Structure grade: <color=yellow>{grade}</color> has been <color=red>disabled</color> on this server."
  }
}
    {% endraw %}
