  2. Wulf Community Admin

    The Oxide downloads are not hosted on oxide mod.org, they are hosted on GitHub.com. While it was quite the coincidence that they both had issues at the same time, the reasons were unrelated. The change with GitHub.com had to do with them now requiring TLS 1.2 for connections to the site, which some older software and such does not support without updates. If you are having an issue with this, you'd really need to contact your host for them to update to support the changes with GitHub. I doubt your host allows uploading all of the files via the File Manager though.
