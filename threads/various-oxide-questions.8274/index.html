    1. The symbols are unicode symbols. For a good reference, see http://fsymbols.com/all/. You copy/paste them in your hostname.
    2. Groups for what? Oxide's permission system has groups, but unless something uses that group, there's no point.
    3. The hostname size in the server listing is based on how long your hostname is, there is no control over it.
    4. Search the plugins section for "airdrop" plugins.
