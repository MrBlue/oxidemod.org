    ProfanityFilter - Filters profanity.

    Filters bad words that typical players love.

    Default config:
    Code:
    {% raw %}
    {
  "BannedWords": [
    "fuck",
    "shit",
    "cunt"
  ]
}
    {% endraw %}
    Default lang:

    English:
    Code:
    {% raw %}
    {
  "BannedWord": "That's a banned word."
}
    {% endraw %}
    French:
    Code:
    {% raw %}
    {
  "BannedWord": "Ce mot est interdit."
}
    {% endraw %}
    Spanish:
    Code:
    {% raw %}
    {
  "BannedWord": "Esta palabra es prohibida."
}
    {% endraw %}
