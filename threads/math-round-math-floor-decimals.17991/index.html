  3. sqroot

    To add to what k1lly0u mentioned, you can also easily write this yourself.
    When casting a number to an integer, it is truncated. So we only want to truncate the last two digits.
    If you can assure that 0 <= 100*x < 2^31 then the following works:
    rounded = (float) (((int) (100 * x)) / 100)
    Brackets only for clarity, you can ignore some of those.
    If 0 <= 100*x < 2^31 is false then stuff gets a bit more complicated and you should definetly use Math.Round in either case.
