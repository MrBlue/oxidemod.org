    RealtimeWipeInfo - Updates title dynamically depending on wipe time as well as preventing wipe chat spam.

    This plugin has 2 main features
    • Dynamically updating title with details on the last wipe
    • Anti wipe chat spam with a reply of the last wipe date.
    I have included my current setup that I use this plugin for in the configurable Lang messages. I suggest not changing much about the title messages because it can fit a reasonably long title with this. If you change the title messages, make sure it can fit along with your title.
