    IRBench - Repair items without durability loss

    A simple plugin to modify repair benches so items don't lose durability when repaired.

    Default config:
    Code:
    {% raw %}
    {
  "ConditionLostOnRepair": 0.2
}
    {% endraw %}
    Setting ConditionLostOnRepair to 0 will make the item brand new after it's repaired. Valid settings are 0 through .99. Higher settings = lower durability after repair.

    Note: Will not increase overall durability, so items repaired before installing this plugin will maintain their current durability.
