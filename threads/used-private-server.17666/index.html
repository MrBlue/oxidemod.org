  2. sqroot

    To copy absolutely everything, just copy the server identity directory (this includes configurations, the map, blueprints, ...).
    To copy blueprints, copy only Storage.db in your server identity directory.
    To copy the map, copy the most recent save file (0.sav) from the map directory in the save directory in the server identity directory to a map directory with the same name (since Rust identifies maps via that directory name).
