    Skrallex submitted a new resource:

    ConsoleMessages - Send messages to individual players with console commands

    Easily send messages to individual players (Private Messages-esque) using a console command. Super simple, and Lang files included.

    Made this for integration with ServerRewards.

    Console Command:
    • cm.say PlayerName Message: Send a message to the specified player.
