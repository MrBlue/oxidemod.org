    Requirements:
    • Most current version of the Oxide.
    • plug Kits
    Commands to create
    1 step: /group add <namegroup>
    2 step: /grant group <namegroup> <permission>
    3 step: /kit add <kitname>
    4 step: /kit permission <permissionname>
    5 step: /kit items (your options and Description)

    Turning players in VIP
    /usergroup add <nameplayer> <namekit>


    I hope this helps!
