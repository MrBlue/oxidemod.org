  2. Wulf Community Admin

    Rust automatically saves before shutting down if you are using the 'quit' command. Forcefully shutting down your server via a control panel would not provide a save though. If you use the control panel method, you would need to use a plugin such as TimedExecute to schedule when the save happens.
