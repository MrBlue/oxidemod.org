  2. Wulf Community Admin

    Dates are what the Oxide 2 builds are linked with, as there isn't really any versioning to the Rust server. https://steamdb.info/depot/258551/history/

    You can see the matching date by looking in the version.txt included with every Oxide 2 build, or by looking at the .zip filename or update page when downloading.
