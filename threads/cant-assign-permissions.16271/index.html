  2. Wulf Community Admin

    The addadmin steamid needs to be added to your autoexec.cfg or your startup .bat/script, otherwise it will not persist between restarts.

    You won't be able to use the /grant command without admin being assigned properly. Once you have admin, you can assign permissions with this command for the Teleporation plugin and others.
