    URLs are actually never stored, when you run the sil command it downloads the image and stores the bytes of the image into the server's storage folder where all images for signs are stored. The restore command will grab those bytes, create an image in memory and resize it and then store the bytes again so the image has the correct width and height to be able to be displayed.
