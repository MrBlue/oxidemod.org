  2. Boostmeister

    you need to make sure the image is 512x256 and i believe it has to be jpg jpeg or png i think.
    then you add this line to your bat file +server.headerimage "linkofimagehere"

    so broken down
    1.Resize your image
    2.Make sure its the correct format.
    3.Host it on a site like imgur and click copy image address.
    4.Add that line with your copy link to your bat file.

    If i made any errors im sorry hopefully Wulf or someone can correct me.
