    Simple Gambling - Allows admins to create item gamblings

    Permission for players to gamble (needed to play)
    oxide.grant user userName simplegambling.allowed

    Chat commands
    "/gamble"     shows current gambles
    "/gamble play <GambleName> <Number>" choose the name of the Gamble and the Number you want to bet on, to use this command you need to have the item that the Gamble requires

    Chat commands (for admins)
    "/gamble" shows help
    "/gamble set" shows help on how to create a gamble...
