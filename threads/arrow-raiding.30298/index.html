    Bruno Puccio submitted a new resource:

    Arrow Raiding - Allow players to break wooden doors using bow and arrow

    Permission for non admin use (needed to change plugin's settings)
    oxide.grant user userName arrowraiding.allowed

    Chat commands
    "/arrow"     shows current values
    "/arrow bow" set the amount of bow arrows needed to break a wooden door
    "/arrow crossbow" set the amount of crossbow arrows needed to break a wooden door

    How to use it
    if you want to let players break wooden doors with 300 bow arrows you have to type
    "/arrow bow 300"
