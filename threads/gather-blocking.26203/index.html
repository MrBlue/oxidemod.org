    Gather Blocking - Blocks gather by certain tools.

    This plugin simply blocks gather by certain tools. Such as a bone knife.

    Default Config:
    Code:
    {% raw %}
    {
  "The list of tools that cannot gather. Based on their item name.": [
    "knife.bone",
    "bone.club"
  ]
}
    {% endraw %}

    Default Lang:
    Code:
    {% raw %}
    {
  "CannotGather": "You may not gather with the tool of: {0}"
}
    {% endraw %}
