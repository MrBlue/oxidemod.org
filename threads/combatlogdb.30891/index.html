    CombatLogDb - Use this admin tool to translate CombatLog IDs into playernames and lookup players' combat logs.

    This admin tools allows players with the permission combatlogdb.use to translate IDs found in a player's combat log into the player's name.

    You can also display a player's combat log, and the plugin will automatically translate any IDs found in the combat log to the relevant player name.

    Permissions
    combatlogdb.use > Allows the user to use the /cid and /combatlog command

    Commands
    /cid ID > Check who a CombatLog ID belongs to.

    /combatlog <name or...
