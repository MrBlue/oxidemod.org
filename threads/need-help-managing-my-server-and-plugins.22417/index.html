  4. Freaky

    1. https://github.com/OxideMod/Snapshots/raw/master/Oxide-Rust.zip
    2. Unzip and copy the files in your installation directory (where do you have RustDedicated_Data directory, not into it)
    3. Launch you server for the first time with Oxide (for creating oxide directory).
    4. Copy your plugins into /server_installation/server_identity/oxide/plugins (the plugins configs will be created automatically --> ./oxide/config)
    5. Modify only the configs, not the plugins, save your modifications and reload the plugins --> Console: oxide.reload <plugin_name> !
    6. You can use RustAdmin to manage your server : https://www.rustadmin.com/release/RustAdminRelease.zip
    7. Enjoy!
