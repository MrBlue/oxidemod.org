    Oh my the dreaded RPC error - this error normally crops up when trying to add the UI to the player, the fix for it is to reload the adminmenu plugin on the server - using :
    oxide.reload AdminMenu

    If that doesn't fix it - make sure you're on the most recent update (2.0.5) and reset the config (/adm_resetconfig) (this will need adminmenu.allow.config permission)
    Good luck and sorry for the inconvenience
