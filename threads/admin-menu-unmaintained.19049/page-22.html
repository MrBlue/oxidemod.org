  13. FlamingMojo

    The Notorious said: ↑
    Could you please clarify how to give my moderators the ability to do "all" commands, or how to select individual commands (ie kick, ban, etc.)... My mods can currently only change time of day, but not tp, kick or ban.
    Click to expand...
    You need to give the moderator group either the permission adminmenu.allow.all i.e. by using the console command
    grant group moderator adminmenu.allow.all
    OR by using the Groups Menu part of AdminMenu so long as you have the permission to that (adminmenu.allow.permissions)
