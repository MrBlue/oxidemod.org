  1. FlamingMojo

    FlamingMojo submitted a new resource:

    AdminMenu - In-game GUI for common admin commands

    My first plugin so this is a work in progress!

    This shows an in-game graphical interface for admins, allowing them to do one-click commands on any player or sleeper. currently supported commands are:
    • Ban - This is set to permaban without reason at the moment
    • Kick - This is set to kick without reason at the moment
    • GoTo - This teleports the admin to the player
    • Bring - This teleports the player to the admin
    • Kill - This instantly kills the player.
