    Time Control - Control time, day and night duration on your server

    Time Control for Hurtworld Legacy and Itemv2: allow you to control the time and customize day and night duration on your server.

    Permissions:

    • timecontrol.use

    How to grant:

    • /o.grant group admin timecontrol.use
    • /o.grant user <yourname/steamid64> timecontrol.use

    Chat commands:

    • /settime day - set day time via chat command
    • /settime night - set night time via chat command
