  2. EnigmaticDragon

    You can use AddLiquid(ItemDefinition, Amount):
    Code:
    {% raw %}
    Item item = ItemManager.CreateByItemID(68998734);
player.GiveItem(item);BaseLiquidVessel bota = item.GetHeldEntity().GetComponentInChildren<BaseLiquidVessel>();
bota.AddLiquid(ItemManager.FindItemDefinition(112903447), 10);
    {% endraw %}
    Note: It's important that you first give the item to the player and then fill it with water. Otherwise you get a null reference exception.
