  11. Mughisi

    The plugin should work without issues, I just crafted 11 hoodies with the default settings which resulted in the following: http://i.imgur.com/7C7UUHH.jpg?1

    Keep in mind that the plugin will not select a random skin if you selected a skin other than the default one in the crafting interface (this is done to allow people to still craft items with the skins they own without having to get lucky with a random one).

    Another thing to keep in mind is that there are a few other plugins out there that actually cancel out the skin randomization from this plugin.
