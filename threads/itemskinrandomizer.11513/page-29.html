  5. PunJedi

    After reading through MANY pages of this, I found a fix for my issue that may help others.

    I'm running BOTH InstantCraft AND CraftingController and I found that if you set THIS line in InstantCraft
    "Randomize item skins if skin is zero": true, <--By default is false

    Then the randomizer works great. I also have my config for Randomizer set to True on both Permission and Use Default.

    Hope this helps and sorry if it was already posted as I made it through about 19 of 29 pages
     
    PunJedi, Dec 18, 2017
