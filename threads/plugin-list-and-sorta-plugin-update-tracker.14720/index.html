    So I was bored today, decided to whip up a quick website to help me keep track of new plugins, and help finding plugins for my server a bit easier.

    The index has more information on what it does and how to use it: -> Oxide Plugins <- (Ignore the subdomain shit, I didn't want to buy another domain for it right now)

    You can view Rust plugins here -> Oxide Plugins <-

    The basic plugin data is scraped from here every 6 hours and put into fancy tables allowing you to sort and move through the list of plugins very quickly. Should help you find new plugins easily.

    Additionally, there is a way to log what plugins were updated (or added) since your last check, it's not a great system but the front page explains it.

    Right now I'm gauging interest in the project. If people really like it and want to see more (feature ideas on front page) I'll put more development into it. If not, I'll just use it myself for my servers.
