    Wulf submitted a new resource:

    SleeperGroup - Puts players in a permissions group on disconnect if sleeping

    SleeperGroup puts players in a permissions group on disconnect if sleeping. The default group is 'sleeper', which is made automatically if it doesn't already exist.

    Configuration

    You can configure the settings in the SleeperGroup.json file under the oxide/config directory.
    Code:
    {% raw %}
    {
  "Permissions Group": "sleeper"
}
    {% endraw %}
