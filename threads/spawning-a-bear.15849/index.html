  4. PaCe

    Yes, but it only works on modded (oxide) server.
    Build for Rust | Oxide

    1) Install

    2) grant permission to yourself/admingroup
    group admin:
    oxide.grant group Admin builder
    player:
    oxide.grant user YourName/Steam64ID builder

    3) type in:
    /spawn bear
    or /buildhelp
