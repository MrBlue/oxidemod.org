    Plugin Request or an idea: Temporary and Permanent Cupboards
    This plugin can have got a timed chat command like Remove Tool has got.
    /cupboard : will activate permanent cupboard, you must use this command while building your base.
    It will be deactivated in 30 seconds.
    So when player places a cupboard around the map while pvp, they will disappear.
    limited permanent cupboards: true
    limited permanentcupboards: 5
    permission: cupboard.limit x: 10 , cupboard.limit y: 15, cupboard.limit z: 20
    This plugin will help modded servers about pvp buildings and external walls + other deployables , so we can remove pvp buildings with auto decay plugins even there is a cupboard and with permissions server owner can benefit about it.

    People placing cupboards and then external walls or foundations+walls around the map while they are fighting and they don't use that walls, foundations etc again. With this plugin, unused - unimportant cupboards will be decayed in X time and garbage things will be decayed too.

    I know there is decay plugins but because of unnecessary - unused cupboards, most of te garbage buildings does not decaying. We need this plugin.
