    MalkoR submitted a new resource:

    EnchantTools - Add enchanted tools to the game that mining melted resources.

    Allows to give enchanted tools for mining, woodcuting and skining to player.
    • If player use enchanted tool for woodcuting, player will recieve charcoal instead wood.
    • If player use enchanted tool for skinig, player will recieve cooked meat instead raw.
    • If player use enchanted tool for minig, player will recieve Metal Fragments instead Metal Ore, Sulfur instead Sulfur Ore and High Quality Metal instead High Quality Metal Ore.
