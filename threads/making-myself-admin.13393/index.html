  2. woop ツ

    add/edit this line to your host.bat or any *.bat which starts up your server

    Code:
    {% raw %}
    addadmin <ADDyourSTEAMidHERE>
    {% endraw %}
    F1 will open the console and u can give urself items with

    Code:
    {% raw %}
    g <ITEMid> <AMOUNT>
    {% endraw %}
    If u wanna spawn a car f.e.

    Code:
    {% raw %}
    spawn roach
    {% endraw %}
    Keep in mind that those cars never spawn useable, you will always need to add something like engine/gearbox/wheels and gasoline
