  2. Brko™

    spawn.min_rate
    spawn.max_rate
    spawn.min_density
    spawn.max_density

    No need plugin for it since rust haves native settings for it (type these commands in rcon console or type find spawn) Just be carefull with the settings, its affected how map is big, also you need to experiment a little to get it work well also care, if you have high pop server with 150k+ ents, be very very careful - it can cause laggs
     
    Brko™, Jan 3, 2017
