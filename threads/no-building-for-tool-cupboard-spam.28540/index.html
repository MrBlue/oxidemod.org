  5. cltwo

    Noticed this as well and was able to solve by removing the problematic cupboards. Not 100% sure of the cause yet, but given the dev blog description, an educated guess is this happens if you have some TCs on the map that don't follow the new building ID association (via CopyPaste plugin, for example).

    You can tell a TC is causing this problem if you "use" it in-game and notice they do not have an inventory like the new ones do, and/or they will say "Your building is unprotected". Those will be console-spamming. Removed those old TCs and the messages stopped.
     
    cltwo, Dec 8, 2017
