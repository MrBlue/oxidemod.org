  2. Wulf Community Admin

    Oxide does not wipe your server when installed. Sometimes hosts do, but most do not. Make sure you are using the same +server.seed that you were using before, else specify one if you aren't, otherwise Rust will create a random map each time you start it.

    Also keep in mind that some Rust updates have forced wipes, which you can't restore a backup or use an existing save with.
     
    Wulf, Aug 16, 2015
