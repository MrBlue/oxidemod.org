    Connection Commands - Runs one or more server command when a player connects/disconnects

    Connection Commands runs one or more server command when a player connects/disconnects. This can be used for granting and revoking permissions, or pretty much anything else.

    You can use {id} and/or {name} in the commands which are placed with the player's Steam ID or username respectively.

    Configuration

    You can configure the settings in the ConnectionCmds.json file under the oxide/config directory.
    Code:
    {% raw %}
    {
  "Commands on connection...
