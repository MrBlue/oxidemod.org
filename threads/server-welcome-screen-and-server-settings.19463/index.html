  2. Wulf Community Admin

    The map only gets wiped if you, your host, or Rust forces a wipe (these generally happen once per month.)

    Updating Oxide is the same as installing it, so however you installed it, just do it again after any game updates.

    There are a few plugins under the Plugins section that offer GUI when you join, Rules GUI, Server Help, and a few others probably.

    The server.writecfg command is not for all settings, it's only for users and bans really. If you want to edit settings, add them to your start .bat or your server/identity/server.cfg file.
