  10. SweetLouHD

    This snippet will print to the console every second the current system time.
    ex 3:14 PM

    you could use the DateTime.Now.ToShortTimeString() to compare to a variable then run something

    Code:
    {% raw %}
    using System.Collections.Generic;
using System;namespace Oxide.Plugins
{
    [Info("Timers Test", "SweetLouHD", 0.1)]
    public class Timers : ReignOfKingsPlugin
    {
        void Loaded()
        {
            timer.Repeat(1, 0, () => Puts(DateTime.Now.ToShortTimeString()));
        }
    }
}
    {% endraw %}
    Additionally
    DateTime.Now.ToString("HHmm");
    will print out military time
    ex. 1339 = 1:39 PM
