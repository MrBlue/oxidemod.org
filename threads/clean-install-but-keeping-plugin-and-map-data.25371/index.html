  2. Wulf Community Admin

    Oxide isn't related to your map save and plugin data is only removed if you remove it, Oxide doesn't. If you have the +oxide.directory set to be under your server/ident path, that's where you'd look, otherwise it'd be under the oxide folder in the server's root (rust/oxide). Plugin data is stored under oxide/data and configs are under oxide/config.
