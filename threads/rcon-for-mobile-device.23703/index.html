  4. Windamyre

    I use RustDroid: Rust Server Admin. It is a a paid app at (I think) US$2.35. It's stable for me and was last updated around Christmas of 2016 (about 2 months). This seems like a long time ago, but it's the most current of the apps I found.

    Tried Rusty Sheriff - Rust RCON tool, but didn't like it as much. Rusty Sheriff seems to have a free and paid version, but neither has been updated in over a year.
    RustyTool looks good, but at US$7.49 I haven't bothered to try it out. It also hasn't been updated since the XP system dropped.
    Rust Mobile RCON has nice, Rusty graphics but also was updated about a year ago.

    There are also a few generic RCON terminal programs for any Steam program that uses RCON, but they seem under-powered compared to the above. (TBH, I didn't know RCON was a multi-game standard until I started writing this reply. Shows how much I have to learn. XD )

    All info was current when I wrote it -- probably changed since then.
