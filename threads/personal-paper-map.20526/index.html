    Personal Paper Map - Personalize the Paper Map, set a default map, let users save there maps

    You may use this plugin different ways:
    1) set an auto paper map so that when players craft a new paper map they will get a default map set by the admin
    2) let players save/load there own paper maps.

    To make this plugin lightweight, it is using PlayerDatabase.

    User Commands:
    /pmap list => to show your current map list
    /pmap save mapname => to save your current map
    /pmap load mapname => to load a specific map saved
    /pmap remove mapname => to remove a specific map.
