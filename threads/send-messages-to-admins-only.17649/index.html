    So I have had a pretty good idea for a plugin, my idea was that players that require admin help could do
    /Admin "Message" and it would send messages to admins only. Reducing the amount of chaos in chat and making it easier for admins...

    How it would work:
    /Admin "Message" - allows a player to send messages to admins
    /ar "Message" - Allows admins to reply to last admin messages (ar standing for Admin Reply)
    /Admin "PlayerName" "Message" - Allows admins to reply to individuals (Useful for multiple messages)

    Here's an example:
    Player: /Admin Please help me
    Admin: /ar ok how can I help?

    or

    Player1: /Admin Can you help me?
    Player2: /admin Can you tp to me? I need help....
    Admin: /Admin "Player2" "How can I help?"
    Admin: /Admin "Player1" "I'll be there in 2Mins"



    Let me know what you think
