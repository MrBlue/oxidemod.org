    I made this plugin on a request and hadn't thought that there would been actually anyone liking it.
    I wouldn't have done it just by myself (I still think it's game breaking, but nah I like it), so you should also partially thank the one who requested something like that.

    I'll think of your suggestions and think they will fit great into this, that means:
    1) I will probably add a shorter chat-command or even remove the existing two then (I get what you mean but please refrain from insulting others indirectly or even directly).
    2.1) The dice is nice ... maybe you got unlucky and get the worst medic who comes to late or just don't manage to get you back up.
    2.2) Well with the one above I will also add that you can only call a medic once per wounded-phase, so players won't start to spam the command so that maybe one out of ten will patch them up.
    3) I will take a look into the pop-up-stuff, but actually I think it won't help that much as it will probably be to late when you're lying on the ground begging for your life and have to read a pop-up first, but then you will at least know for when it happens again ... so I'll go with it.
    Of course I will make this stuff configurable.

    For the timer-thingy, it's already configurable, see "Settings" "WaitTillMedic" (maybe not the best name for that ... but I've gone with medic here and there anyway so I don't see a problem with that ) so if you set that to 0 it will get you unwounded the instant you sent the chat-command, while setting it much higher will take you any chance to get you patched up by this plugin as the game will either send you back to your feet (yeah the game is actually nice ... sometimes, it's pretty rare) or will already have called for the reaper.
