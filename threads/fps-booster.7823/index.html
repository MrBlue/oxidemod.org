  8. Wulf Community Admin

    FPS type plugins also tend to be a gateway to malicious behavior, as they often come from unknown and off untrusted users. I recall one at least one occasion where the author added commands and o give every user C4, and made himself admin. This resulted in numerous servers being destroyed.

    There is nothing wrong with posting one here, just take caution and review/test before using it or any plugin on a live server. We will not be accepting plugins of this type into the Plugins section though.
