    Ryan submitted a new resource:

    ItemTranslations - Provides a data file of item names for translation

    This plugin basically gives you the option to change the name for each item in the game. This is especially useful for foreign servers where they cannot localize the item names to their servers language.

    You are not limited to translating the item names however, you can change the names of the items to whatever you wish. But the original purpose of this plugin was for localization purposes.

    This plugin doesn't change the names of the items in the native Rust UI's such as inventories,...
