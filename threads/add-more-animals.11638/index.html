  10. Deicide666ra

    Leave everything at the default -1 at first and start the plugin. Make sure the server has been running for at least 1 hour since restart and run /cdump and note the animal counts. These are the normal amount of animals for your map. Now go back to the config and set the minimum for each animal type to double the value seen in cdump.

    You can arbitrarily change the values there. For instance on my server I run every animal type double except wolves and bears which are at 350 and 225 respectively (on a 5k map). Keeps the nakeds on their toes
