  2. Wulf Community Admin

    Generally there's no reason for the fps.limit to be increased. Many servers run at 60fps, Rust's server is capped at 256fps by default. If you experience no issues at a lower FPS, there shouldn't be any reason to not use that to lower CPU usage.
