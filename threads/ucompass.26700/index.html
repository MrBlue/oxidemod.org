    UCompass - Useful GUI Compass

    UCompass is a useful GUI compass which display directions of Earth around you

    Commands:
    • /compass - toggle compass GUI (Chat)
    • ucompass.toggle - toggle compass GUI (Console)

    Use command to set bind: bind u ucompass.toggle

    Permissions:
    • ucompass.allow - allows player use commands (chat & console)

    Configuration Options    :
    • "MajorDirectionColor": Text color for major direction (North, East, South,...
