    Respawn Messages - Shows custom messages to players on respawn

    Respawn Messages allows players to view messages on respawn. Made with the help of DylanSMR & K1llY0u

    Default Config File:
    Code:
    {% raw %}
    {
  "ChatEnabled": true,
  "PopupEnabled": false,
  "Prefix": "[ <color=#cd422b>RespawnNotes</color> ]",
  "PrefixEnabled": true,
  "RespawnMessageChat": "Hey, <color=#cd422b>try not to die</color> this time!",
  "RespawnMessagePopup": "<size=20>Hey, <color=#cd422b>try not to die</color> this time!</size>"
}
    {% endraw %}
