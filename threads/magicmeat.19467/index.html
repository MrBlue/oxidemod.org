    MagicMeat - Automatically converts raw meat to cooked

    MagicMeat is a plugin which removes the requirement to cook meat. Depending on configuration, any raw meat which is picked up or placed into a storage container will be converted to its cooked counterpart.

    Usage (console):
    magicmeat.set [option] [value] - Set value of specified option
    magicmeat.get [option] - Get value of specified option
    magicmeat.def - Restore default configuration
    magicmeat.version - Display version...
