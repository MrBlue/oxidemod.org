    Not sure if there is a solution, never seen this, but for the pc crash, in RUST settings (ingame at the main menu) go to options, performans and look for the slider called MAX GIBS, turn it to 0 or lower it. The effect of this settings is the part, that makes those brak things spawn arround you making so much leak to your pc that you crash. Also helps alot if you are using removertool with the /remove all function. - removing with this big buildings is much easier
