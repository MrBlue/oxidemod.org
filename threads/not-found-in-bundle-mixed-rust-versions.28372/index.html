  2. Wulf Community Admin

    Assets are from Rust, not Oxide. If you see missing assets it is possible that you have mixed Rust versions, which could be from installing a version of Oxide (it's a mod, so it includes some patched Rust files) that is for a different version of Rust than what you have installed.

    Make sure your server is fully updated and that you're installing a compatible build of Oxide.
