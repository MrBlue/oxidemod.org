  3. Ankawi

    To change the map size, go to your .bat file where the server starts up and edit and add -server.worldsize 4000 or what size you want, 4000 is the default.
    For you to wipe the map, go to your folder my identity and go to the folder saves, and delete the map. As the post above said you can either randomly spawn new maps by going to playrust HQ and put the seed also in the .bat file.
