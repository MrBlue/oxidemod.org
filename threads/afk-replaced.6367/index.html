    AFK Kick - Kicks players that are AFK (away from keyboard) for set amount of seconds

    AFK Kick is a plugin that automatically kicks players that are AFK (away from keyboard) for set amount of seconds. The default AFK time limit is set to 300 seconds (5 minutes).

    Configuration
    You can configure the chat name, messages, and other settings in the cfg_afk.txt file under the save/oxide/data directory.
