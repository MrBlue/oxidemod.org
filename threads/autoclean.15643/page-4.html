  5. CaptainNebulous

    FiD said: ↑
    How do I see that the plugin is working? I've entered /clean in the chat and got no error message. Does it mean, that from now on the cleaning interval is activated?
    Click to expand...
    If you look in your oxide/data folder there will be a folder in there called "autoclean" and within that is a file containing the location of each detected structure/item with no stake. You will see how many times that item has been scanned.

    Now whichever comes first..... the item gets removed, protected by stake, or it reaches the maximum # of checks that you have set in the config file.... it will stop counting the scans for it and upon the next server restart the plugin will automatically clean it out of the data file. If the item has reached the maximum # of checks it will wipe that item off the map on the next scan and item will be cleared out of the file at next restart.

    But yeah /clean has no message, and there is no real need to use it anymore since the plugin has gotten patched. All it does is force a scan, any "cleaning" will only happen if items have reached the # of scans provided in the config file. ALSO /clean will wipe chests out of shacks for some reason. Also spamming /clean multiple times in short spans of time will lead to a 20 or more second lag spike and possible brief server crash.... as it takes a lot of resources, believe it or not, to remove a lot of objects at once.

    Hope this helped
