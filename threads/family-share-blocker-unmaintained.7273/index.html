  1. Mughisi

    Mughisi submitted a new resource:

    Family Share Blocker - Block family shared accounts from joining your server.

    This plugin will kick players that don't own the game and are playing Rust as a family shared game.

    If you would like certain people with a family shared account to connect you can add them to the whitelist in the configuration file.

    Keep in mind that this plugin uses a webrequest to use the Steam API to check if a player is playing on a shared account, and so for this to work a Steam API key is required. You can obtain your own API key here: http://steamcommunity.com/dev/apikey...
