  2. Wulf Community Admin

    If you are hosting behind a router, you cannot bind to your router's IP. You need to either not set the +server.ip, or set it to 0.0.0.0 or your LAN IP of that machine, not an IP address it doesn't have assigned to it. When forwarding ports on your router, make sure the LAN IP matches the machine hosting the server, and that you are forwarding UDP traffic, not TCP.
