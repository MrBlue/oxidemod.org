  2. Wulf Community Admin

    More plugins would mean more memory, yes; but there isn't really a limit to how many you can add other than your resources. Some plugins can be more resource intensive, but many are pretty small and insignificant in terms of resource usage and impact on the server.

    Most plugins will use very little memory, so if you are noticing excessive memory usage, then you'd want to track down which plugin may be contributing to that through the process of elimination; starting with the more complex plugins such as map ones or CUI.
