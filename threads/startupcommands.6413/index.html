    Wulf submitted a new resource:

    AutoCommands - Automatically executes configured commands on server startup

    AutoCommands is a simple plugin that automatically executes configured commands on server startup. This is useful if you do not have access to the startup command-line parameters for your Rust server, or your host resets it with every restart. Just add the commands you want to the config file, and done! Next time your server starts, all the commands will be run...
