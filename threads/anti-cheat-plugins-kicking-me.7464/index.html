  2. Wulf Community Admin

    VAC bans are based on client modifications being detected, which you should only get if you use client mods/hacks, not Oxide which is a server-side only mod. Most likely one of the plugins you have installed for anti-cheat are kicking you, which would only affect your own server.

    It sounds more like you never properly added yourself as admin and actually saved it. Rust's server doesn't save the users to the users.cfg after using the commands unless you properly "quit" or use "server.writecfg" after you add yourself as admin/moderator. This could be because of a plugin removing them, or your server simply crashing as the result of a poorly written plugin.

    I would suggest going easy on the "anti-cheat" plugins, using them only if you fully understand the benefits, usefulness, and consequences.
