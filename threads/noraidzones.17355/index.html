    Lizzaran submitted a new resource:

    No Raid Zones - Create easily zones which blocks the usage of C4

    No Raid Zones allows you to define zones where players can't use C4.

    How exactly does it work / do you use it?
    Simply stand on the cell (place which is surrounded by the blue dots) you want to protect and type /nrz add Example. No Raid Zones will automatically create a cluster of all connected cells which are owned by you and protect them all.

    Commands
    /nrz add <zone>
    Adds a new zone.

    /nrz remove <zone>
    Removes the specific zone.

    /nrz update...
