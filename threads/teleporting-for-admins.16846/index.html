  5. Freaky

    Hi! Only the admins can use : /tp player (teleport to the exact player location), /tpn player (teleport near the player, default 30m), /tpsave name (save a location, for teleporting) and /tpl name (teleport to the location name)...all these teleportings are instant. You can set to true for moderators: "UseableByModerators":true!
