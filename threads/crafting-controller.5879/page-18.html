  16. Mughisi

    Update + Fix

    Changed the command /craft to /rate
    Fixed the issues with the console command crafting.unblock
    Added the chat command /itemrate <item> <rate> and console command crafting.rate <item> <rate> to set the crafting rate of a single item.

    Example: instant crafting except slower craft on C4:
    /rate 0
    /itemrate Timed Explosive Charge 150
