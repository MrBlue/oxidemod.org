    AutomatedEvents - A small plugin that will run various events (when using a night skip plugin)

    This is a simple plugin that allows you to run various events on timers to counteract them not being run when using a skip night plugin.

    Event Types - CargoPlane, Helicopter and XMas event

    The time between each event is randomized using a minimum and maximum time value in the config and any of the events can be toggled off

    Config
    Code:
    {% raw %}
    {
  "Events": {
    "CargoPlane": {
      "Enabled": true, // Toggles the automation for this event on/off
      "MaximumTimeBetween": 45, //...
    {% endraw %}
