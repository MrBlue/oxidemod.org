    redBDGR submitted a new resource:

    Snapper - Adds a few new features to the snapping of beartraps

    This plugin adds a few features to the snapping of bear traps, to increase usability

    • Traps placed by someone with the permission will automatically reset after a set amount of time
    • Traps will not go off if their owner walks over them (configurable)
    Permissions:
    • snapper.auto - Players with this permission will get their bear traps automatically reset
    Configuration File:
    Code:
    {% raw %}
    {
  "Settings": {
    "Reset Time": 5.0,...
    {% endraw %}
