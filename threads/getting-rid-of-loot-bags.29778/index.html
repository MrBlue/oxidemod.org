    Hello I saw some people were trying to get rid of lootbags
    I figured it out for my Duels server Frenchies duels

    Just download the TimedExecute plugin and copy and paste this into your config file

    "TimerOnce": {
    "killbyblueprint 212 10000": 0.1,
    "killbyblueprint 212 10000'": 0.2,
    "killbyblueprint 212 10000": 0.4,
    "reset.timeronce": 0.5
    },
    "TimerRepeat": {
    "killbyblueprint 212 10000": 1,
    "killbyblueprint 212 10000'": 2
    }
    }

    It destroys lootbags within milliseconds of them being created.

    Also if you want to destory player sleepers
    it's
    killbyblueprint 247 10000
