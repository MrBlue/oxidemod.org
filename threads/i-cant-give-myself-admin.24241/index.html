  3. GetMiked

    Hi,

    In order to define yourself as an admin on your server, follow these steps:

    1. Find your 64 bit steam ID, you can search it in here: home - STEAMID I/O


    2. Open up your server configuration file and add the following command anywhere in the file:
    Code:
    {% raw %}
    addadmin Your64bitSteamID;
    {% endraw %}
    Mine would be:
    Code:
    {% raw %}
    addadmin 76561198045365357;
    {% endraw %}
