  2. Vlad-00003

    Hi there.
    As far as I can tell you are reading file called "MyDataFile" and writing file called "playersWithKit"
    Try to change in Loaded() this:

    Code:
    {% raw %}
                try
            {
                _flies = Interface.Oxide.DataFileSystem.ReadObject<dataFiles>("MyDataFile");
            }
    {% endraw %}
    to this:
    Code:
    {% raw %}
                try
            {
                _flies = Interface.Oxide.DataFileSystem.ReadObject<dataFiles>("playersWithKit");
            }
    {% endraw %}
    Also - in your case you don't have to create your own class
    You could use this:
    Read:
    Code:
    {% raw %}
    Dictionary<string, ulong> _playerIdRegistered;
try
{
      _playerIdRegistered = Interface.Oxide.DataFileSystem.ReadObject<Dictionary<string, ulong>>("playersWithKit");
}catch
{
       _flies = new Dictionary<string, ulong>();
}
    {% endraw %}
    Write:
    Code:
    {% raw %}
    Interface.Oxide.DataFileSystem.WriteObject("playersWithKit", _playerIdRegistered);
    {% endraw %}

    You need to wait until the player has finished loading in and stopped sleeping before giving them anything. Something like this;
    Code:
    {% raw %}
    void MethodName(BasePlayer player)
{
   if (player.IsSleeping() || player.HasPlayerFlag(BasePlayer.PlayerFlags.ReceivingSnapshot))
   {
      timer.In(1, () => MethodName(player));
         return;
   }
   // Do Stuff
}
    {% endraw %}
