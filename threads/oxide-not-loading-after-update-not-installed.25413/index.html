    Here's what i do in order to make sure the server is running with mods.

    -Stop the server
    -Open SteamCMD (paste each line below into the cmd)
    login anonymous
    force_install_dir "C:\YOURLOCATIONPATH\"
    app_update 258550
    -Download latest version of oxide
    -Drag and drop ONLY the Rustdedicated folder (into root of server)
    or you can drag the whole folder, i just like my .batch file and dont want it overwrited with a 'default' one
    -then start up the server after dragging updated oxide inside and you're good to go

    hope i didn't miss a step
