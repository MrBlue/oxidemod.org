    Warlock said: ↑
    Go to your Oxide Config folder.
    Locate the QuickSmelt.json file.
    Right click it. (you may need to select a program to edit it, just choose notepad)
    Select Edit.
    Scroll down to the "OvenMultipliers":
    Find for example "Furnace"
    Change 1.0 to 2.0 (each increase by 1.0 reduces the time by .25 seconds)
    Save
    You must ensure that the user has smelting permission.

    Hope this helps.
