  4. Mughisi

    WhompMaster said: ↑
    Is there a way to do a quick multiply? Like gather.rate all 15
    Click to expand...
    you still need to specify the type of gathering but you can use * (asterisk) instead of a the name of an item to modify all items with that multiplier.

    Taken from the overview:
    It is also possible to specify * as the item to increase the rate on all the items by that value, except those that were added manually, example: to have everything from dispensers x10 and wood x2 you could use 'gather.rate dispenser wood 2' and 'gather.rate dispenser * 10'.
