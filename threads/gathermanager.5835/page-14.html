  7. Mughisi

    Mughisi updated Gather Manager with a new update entry:

    Minor fix and command additions

    Fixed the typo regarding the available dispensers for the dispenser.scale command;
    Added "Metal Fragments" to the available resources;
    Added the option to specify * instead of a resource to have all resources from the selected dispenser multiplied excluding the ones mentioned separately:
    Example: gather.rate dispenser * 10
    gather.rate dispenser Wood 5
    -> Gathering Wood will be x5 and everything else x10
    Added the remove flag which takes place instead of the modifier to remove a single value...
