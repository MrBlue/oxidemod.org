    Every time there is an update that is a bit more than just a hotfix the old loot and spawn config files become incompatible. That's not a bug, just the way it works. All you need to do is go into the .old files it makes, change the version one number up (in this case, from 49 to 50)... save then rename to the normal .json it was before.... then restart server. (Note this method will not work if something has been changed in the game's loot/spawn tables in a patch) Also the axe waving thing was a game bug but it looks like it's since been fixed.
