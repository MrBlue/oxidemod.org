    Мизантроп submitted a new resource:

    Running Man - Plugin Runnung Man

    The plugin Running Man is porting my plugin with mod RustExtended(js)

    The meaning of the plugin: randomly selects a player, his task is to survive for a certain time. The task remaining is to kill him. The winner gets a reward.

    Commands:
    Code:
    {% raw %}
    /event - start the running man (required rights owner)
/run - show the status of the event
    {% endraw %}
