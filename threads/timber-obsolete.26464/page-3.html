    You can disable flyhack protection (hackers are still kicked) from the startup batch script:
    Code:
    {% raw %}
    +antihack.flyhack_protection 0 +antihack.flyhack_protection: "0"
    {% endraw %}
    Or by using this config to make Timber just a visual effect:
    Code:
    {% raw %}
    {
  "DespawnLength": 10.0,
  "HarvestFallen": 0.0,
  "HarvestStanding": 1.0,
  "IncludeCacti": true,
  "ScreamPercent": 0.0
}
    {% endraw %}

    Click to expand...
    In the config file, "DespawnLength" determines how long the fallen tree will sit on the ground before despawning, the default is 30.0 seconds. So the player has set time to cut the tree when its down, or if they finish the tree on the ground before this time is up it will despawn. This can be used with the harvest ratios to make the plugin work just as a effect so no farming mechanics are changed.
