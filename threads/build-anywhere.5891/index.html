  10. Deicide666ra

    Well there's a way of toggling the building blocked flag on the bottom and it affects whether or not you can build but I doubt it would work in rad zones. It certainly works for places where you have no cupboard access, but I'm pretty sure these are two separate mechanics.

    Here are commands to toggle building priv on and off:

    Code:
    {% raw %}
            [ChatCommand("on")]
        void cmdOn(BasePlayer player, string cmd, string[] args)
        {
            player.SetPlayerFlag(BasePlayer.PlayerFlags.HasBuildingPrivilege, true);
        }
        [ChatCommand("off")]
        void cmdOff(BasePlayer player, string cmd, string[] args)
        {
            player.SetPlayerFlag(BasePlayer.PlayerFlags.HasBuildingPrivilege, false);
        }
    {% endraw %}
