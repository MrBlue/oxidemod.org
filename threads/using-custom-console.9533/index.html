    For anyone wondering how to do it:
    Make a new .txt file
    Open it
    Paste this code :
    Code:
    {% raw %}
    @ROK -batchmode -nographics -silentcrash -logFile "logs/unity_%DATE:~-7,2%-%DATE:~-10,2%-%DATE:~-4,4%.txt"
    {% endraw %}
    Then (Save as) .bat file
    And Start your server from the .bat that you've saved
