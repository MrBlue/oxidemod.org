  2. AlienX

    You probs already have done this, but it's worth mentioning.
    Every time you update the Rust Server steamcmd will do a sanity check of its files.

    As OxideMod changes some vital files, steamcmd will "uninstall" oxidemod, you will need to copy the oxide files back into the servers directory every time the Rust Dedicated Server is updated.


    Note:
    Take "uninstall" with a pinch of salt, steamcmd simply replaces the file that oxidemod uses to load their own custom code to it's original state.
