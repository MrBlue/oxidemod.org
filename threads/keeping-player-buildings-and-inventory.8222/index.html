  2. Wulf Community Admin

    Moved to own thread. In the future, please do not post in a thread unrelated to your question.

    To answer your question, saves are always loaded unless a new map was generated due to a Rust save version / protocol change. This applies to inventory as well. There's nothing you can do to sanely keep an old map if a wipe is required. You can however keep a player's inventory by using a plugin such as "Inventory Guardian", which can be found under the Plugins section.
