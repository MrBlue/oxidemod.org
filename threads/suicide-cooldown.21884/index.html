    Suicide Cooldown - Permission based cooldown timer and aliases for the kill command

    This is a simple plugin that allows you to modify the cooldown that is triggered by the console command `kill`.

    This plugin also provides a console command alias `suicide` to the existing `kill` command and includes both `suicide` and `kill` as chat commands as well.

    Setting up the plugin
    When installing this plugin for the first time the default config file will be generated adding two permissions with both their own cooldown:

    There is no "all" command unless a plugin provides it. If you want to grant ALL (I don't see why...), you'd use the the wildcard method: "grant ground default suicidecooldowns.*"

    I don't think you really want to grant all though, the reason for multiple permissions is to allow for a different setup per group. If you want to only have one, just remove one of the entries and keep the other.
    Code:
    {% raw %}
    {
  "Permissions": [
    {
      "Permission": "none",
      "Cooldown": 0,
      "Priority": 2
    }
  ]
}
    {% endraw %}
    Then grant the command: grant group default suicidecooldowns.none
