  2. sqroot

    The Unload hook is for resource cleanup (to prevent memory leaks, dangling file pointers, etc.).
    When the server is being shut down, there is no point in cleaning up resources, as the server shuts down anyways.
    What plugins you're using are doing *essential* work on Unload?
     
    sqroot, Aug 14, 2016

  4. sqroot

    My server build is older than 3 weeks and Unload isn't called on quit.

    Flush your log cache more often, not just when the server restarts.
