  3. Meridian

    In console:

    spawn.min_rate #
    spawn.max_rate #
    spawn.min_density #
    spawn.max_density #

    Make sure to type server.writecfg if you're not starting from your batch file.

    These variables adjust the rate of anything that autospawns (trees, ores, barrels, and animals). Maybe one day we'll get some variables to separate them, but for now if you want to increase the rates without using a mod to do it manually, that's your option.
