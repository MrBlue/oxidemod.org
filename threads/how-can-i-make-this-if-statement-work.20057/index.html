    You probably want the if block to be run either when args[0] is any of these characters or if args[0] isn't any of these characters.
    In Python you can write this as
    Code:
    {% raw %}
    if args[0] in "/.\#,|":
    {% endraw %}
    or
    Code:
    {% raw %}
    if args[0] not in "/.\#,|":
    {% endraw %}
    accordingly, as long as you don't need to check for many characters (in that case you should use {"/", ".", "\#", ",", "|"} instead of "/.\#,|").
