    I would create a function in the LUA plugin that returns the array and then call the LUA plugin from your C# plugin.

    Code:
    {% raw %}
    var PluginName = plugins.Find("PluginName");
var output = PluginName?.Call("FunctionName","OptionalParameter1", "OptionalParameter2");
    {% endraw %}
