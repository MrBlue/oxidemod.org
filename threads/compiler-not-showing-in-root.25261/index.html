  5. bitman

    As far as I know based on my own experience is that you can not download the compiler separately ( I looked but could not find). The compiler automatically downloads itself into the appropriate location when you launch your server with Oxide present. If you're missing the compiler, then chances are that it just hasn't installed yet. In my case (for some odd reason) it took five server restarts for the compiler to be downloaded. Just keep restarting your server until it downloads. You'll notice when it happens.

    EDIT: I stated that it's not possible to download the compiler separately, I was wrong. You can get it here. However, for the sake of avoiding issues in the future, you should definitely either figure out what's blocking the download like Wulf said, or keep trying the server restart like I mentioned.
