  5. Wulf Community Admin

    When you see those messages, it means that it is saving fine. If it fails to save, you'll get an error. Oxide filters a couple messages from the console to prevent it from getting too cluttered, but there is no actual "Server saved" message, just that it is saving and then written to disk.
