    k1lly0u submitted a new resource:

    ZoneDomes - Assign shaded sphere entities to physically see the border of zones (ZoneManager)

    Create a shaded sphere entity to show the perimeter of zones from Zone Manager.

    By using a zone's ID this plugin will gather the required information from ZoneManager and create a new sphere. On restart/reload the plugin will verify that the zone still exists, if it doesn't it will automatically remove the sphere

    Chat Commands
    /zd add <zoneid> - Add a sphere to a zone
    /zd remove <zoneid> - Remove a sphere from a zone
    /zd list - Lists the zone ID, radius and...
