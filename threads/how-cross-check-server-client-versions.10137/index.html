  2. Wulf Community Admin

    The server and client do not have matching "versions", and aren't really versioned aside from the Steam buildid. Oxide builds for Rust are dated with the release date of the associated Rust server build, so generally if you are using the correct build from that date of the release, you should have no issues. It sounds like you're using an outdated Oxide build though.
