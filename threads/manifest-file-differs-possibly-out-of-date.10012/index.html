  4. Deicide666ra

    I see this a lot when an "optional" patch shows up for a server. The clients will pretty much all auto update through steam but if you don't update your server, the older and newer clients will be able to connect to it. Those running the newer version of the client will trigger this message. It's just the server saying "Hey, someone with a version newer than myself connected, this might be an issue". I haven't see any issues with it.

    Also, it is possible for clients to update their version to the dev branch which still allows you to connect to normal servers. I don't know if this triggers the message but it very well might, since technically dev builds are more recent than official builds.

    Update your server, you're probably just off 1 patch... They often release 1-2 optional patches after the main one to fix some small things.
