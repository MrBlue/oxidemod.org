  1. WolfsDarker

    WolfsDarker submitted a new resource:

    Block Remover - Removes on player or console command any building blocks without a cupboard.

    Well this is a spinoff from the twig remover i got from this forums. Basically it will remove any building block without a cupboard at least 60m away from it. The only building it wont remove is armored ( for obvious reasons ), but metal, stone, wood etc will be removed.

    Chat Commands:
    /blockcount => Shows the amount of blocks outside a cupboard area.
    /blockremove => Remove all blocks without a cupboard near them.

    Console Commands:
    blocks.count => Same as /blockcount...

  16. WolfsDarker

    WolfsDarker updated Block Remover with a new update entry:

    Barricades support!

    Well it is not just barricades, there is also a change in the commands, instead of /blockremover, /blockcount, /barricaderemove, etc it is now:

    Chat:
    /block barricade count
    /block building remove
    /block barricade remove
    /block building count

    Console:
    block barricade count

    Now you can change the distance for building and barricades from a cupboard, default is 60m for buildings and 100m for barricades.

    Enjoy!
