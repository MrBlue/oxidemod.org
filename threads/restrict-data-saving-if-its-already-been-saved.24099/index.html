  2. ignignokt84

    Yes, technically. Even if all of the fields in PlayerInfo are identical, it doesn't necessarily mean two objects are equal. The storedData.Players.Contains call here will only return true if the PlayerInfo passed in is a reference to an existing entry, but it never will be because it's explicitly instantiated above. (You can do a Google search for "c# object equality" or something similar and see how object equality is determined)
