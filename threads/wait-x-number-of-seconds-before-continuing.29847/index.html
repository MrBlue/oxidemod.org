  3. Steenamaroo

    +1.
    Just be aware that the rest of your code doesn't wait.

    Code:
    {% raw %}
    Puts("First Line");
timer.Once(2, () => Puts("Third Line"));
Puts("Second Line");
    {% endraw %}
    You can do multiple lines with
    Code:
    {% raw %}
    timer.Once(2, () => {
Puts("Fourth Line");
Puts("Fifth Line");
});
    {% endraw %}
