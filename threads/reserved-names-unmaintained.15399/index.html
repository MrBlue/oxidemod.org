  1. vitamin.z

    vitamin.z submitted a new resource:

    Reserved Names - Reserve a nickname X by SteamID Y for Z amount of time.

    Reserve users name for given time. First-come, first-served rule applies for reserving names. Every time he logs back he extents his timer for given time (default 1 week).

    Default Config
    Code:
    {% raw %}
    {
  "Settings": {
    "DeletionTime": "604800"
  }
}
    {% endraw %}
    Example Data
    Code:
    {% raw %}
    {
  "UserNames": {
    "Feramor": "76561197965300264"
  },
  "Users": {
    "76561197965300264": {
      "USERNAME": "Feramor",
      "TIMER": "1423605168"
    }
  }
}
    {% endraw %}
