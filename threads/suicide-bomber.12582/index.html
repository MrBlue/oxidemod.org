  2. Calytic Community Admin Community Mod

    Threw this together for you, the following link will work if/when the plugin is approved..

    http://oxidemod.org/plugins/suicidebomber.1425/

    Description

    SuicideBomber will allow players to use a Targeting Computer and Timed Explosives to explode.

    How To
    1. Move a targeting computer to quickbar
    2. Select the computer from your quickbar
    3. Press E
    4. Count down briefly (6s)
    5. Explode
    Configuration
    • damage (default 1200)
      The amount of damage done to objects closest to you
    • radius (default 12)
      The maximum radius of the explosive damage
    • explosives (default 5)
      The number of Timed Explosives required to use the Targeting Computer
    • scream (default true)
      Whether or not the bomber screams before exploding
