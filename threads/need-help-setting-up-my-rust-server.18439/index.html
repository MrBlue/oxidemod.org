  2. Resistance

    in rcon type
    ownerid steamid64 "player name" "reason"
    then
    server.writecfg
    Then add yourself to the admin group
    usergroup add USERNAME admin
    if in server relog as to permissions for plugins you add them to the admin group check each plugin overview for info not all plugins need you to add permissions
